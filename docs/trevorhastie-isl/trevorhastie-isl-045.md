## 拟合岭回归模型

![](img/6537424cd6dc8319098030c36780daa2_23.png)

![](img/6537424cd6dc8319098030c36780daa2_24.png)

![](img/6537424cd6dc8319098030c36780daa2_25.png)

![](img/6537424cd6dc8319098030c36780daa2_26.png)

岭回归通过在损失函数中添加一个与系数平方和成正比的惩罚项来收缩系数。其优化目标为：

![](img/6537424cd6dc8319098030c36780daa2_28.png)

![](img/6537424cd6dc8319098030c36780daa2_29.png)

**公式：** 最小化 `RSS + λ * Σ(βj²)`

![](img/6537424cd6dc8319098030c36780daa2_31.png)

其中，`RSS`是残差平方和，`λ`是控制惩罚力度的调节参数。当`λ`很大时，系数会被强烈收缩趋向于0。

![](img/6537424cd6dc8319098030c36780daa2_33.png)

![](img/6537424cd6dc8319098030c36780daa2_34.png)

在`glmnet`中，通过设置参数`alpha = 0`来指定岭回归。

![](img/6537424cd6dc8319098030c36780daa2_36.png)

```r
ridge_mod <- glmnet(x, y, alpha = 0)
```

模型拟合速度非常快。我们可以使用`plot`函数来可视化系数如何随`log(λ)`的变化而变化。

```r
plot(ridge_mod, xvar = "lambda", label = TRUE)
```

生成的图显示了系数路径：当`log(λ)`很大时，所有系数都接近零；随着`λ`减小，系数逐渐远离零，最终趋近于普通最小二乘估计值。

---

## 使用交叉验证选择岭回归的λ

我们需要一种方法来选择最优的`λ`值。`glmnet`提供了内置的交叉验证函数`cv.glmnet`。

![](img/6537424cd6dc8319098030c36780daa2_38.png)

```r
cv_ridge <- cv.glmnet(x, y, alpha = 0)
plot(cv_ridge)
```

该图展示了交叉验证均方误差随`log(λ)`的变化。图中有两条垂直线：
*   **左边一条**：对应使交叉验证误差最小的`λ`值。
*   **右边一条**：对应“最小误差一个标准差以内”的、更简洁的模型所对应的`λ`值。我们通常倾向于选择这个更简洁的模型。

## 拟合Lasso回归模型

上一节我们介绍了岭回归，本节我们来看看Lasso回归。Lasso与岭回归类似，但其惩罚项基于系数的绝对值之和，而非平方和。其优化目标为：

**公式：** 最小化 `RSS + λ * Σ|βj|`

这个细微的变化使得Lasso能够将某些系数**精确地收缩至零**，从而同时实现变量选择和系数收缩。在`glmnet`中，`alpha`参数的默认值就是1，即对应Lasso回归。

```r
lasso_mod <- glmnet(x, y, alpha = 1) # alpha = 1 是默认值，可省略
plot(lasso_mod, xvar = "lambda", label = TRUE)
```

在系数路径图上，你可以看到系数是“跳入”模型的：它们从零开始，随着`λ`减小，某些系数会突然变为非零。图顶部的数字显示了模型中非零系数的数量。

我们还可以换一种视角，绘制模型解释的偏差百分比（类似于R²）随系数路径的变化。

```r
plot(lasso_mod, xvar = "dev", label = TRUE)
```

![](img/6537424cd6dc8319098030c36780daa2_40.png)

![](img/6537424cd6dc8319098030c36780daa2_42.png)

这张图显示，模型的大部分解释力（R²）在系数被较大程度收缩时就已经获得，而在路径末端，系数大幅增长却只带来R²的微小提升，这可能意味着过拟合。

## 使用交叉验证选择Lasso的λ

![](img/6537424cd6dc8319098030c36780daa2_49.png)

![](img/6537424cd6dc8319098030c36780daa2_51.png)

![](img/6537424cd6dc8319098030c36780daa2_53.png)

同样，我们使用交叉验证为Lasso选择最优的`λ`。

```r
cv_lasso <- cv.glmnet(x, y, alpha = 1)
plot(cv_lasso)
```

图中显示了交叉验证误差以及对应的最佳模型大小（非零系数个数）。我们可以提取对应于“一个标准差”规则的最优模型的系数。

```r
best_lambda <- cv_lasso$lambda.1se
coef(cv_lasso, s = best_lambda)
```

`coef`函数会以稀疏矩阵格式输出结果，只列出非零系数，这使得变量选择的结果一目了然。

![](img/6537424cd6dc8319098030c36780daa2_55.png)

## 在验证集上评估Lasso模型

最后，我们将使用独立的验证集来评估Lasso模型的性能，并选择最优的`λ`。以下是具体步骤：

首先，将数据划分为训练集和验证集。

```r
set.seed(123)
train <- sample(1:nrow(x), nrow(x) / 2)
x_train <- x[train, ]
y_train <- y[train]
x_val <- x[-train, ]
y_val <- y[-train]
```

然后，在训练集上拟合Lasso路径。

![](img/6537424cd6dc8319098030c36780daa2_57.png)

```r
lasso_train <- glmnet(x_train, y_train, alpha = 1)
```

接着，在验证集上进行预测，并计算每个`λ`值对应的均方根误差（RMSE）。

```r
pred <- predict(lasso_train, s = lasso_train$lambda, newx = x_val)
# 计算RMSE：利用R的循环规则，向量与矩阵相减
rmse <- sqrt(apply((y_val - pred)^2, 2, mean))
```

![](img/6537424cd6dc8319098030c36780daa2_59.png)

![](img/6537424cd6dc8319098030c36780daa2_61.png)

```r
plot(log(lasso_train$lambda), rmse, type = "l", xlab = "log(Lambda)", ylab = "RMSE on Validation Set")
```

## 总结与课程回顾

在本节课中，我们一起学习了模型选择的最后一部分内容——使用`glmnet`包进行岭回归和Lasso回归。

我们涵盖了以下核心内容：
1.  **岭回归**：通过`alpha = 0`拟合，使用L2惩罚（系数平方和）收缩所有系数。
2.  **Lasso回归**：通过`alpha = 1`拟合，使用L1惩罚（系数绝对值之和）进行变量选择和系数收缩。
3.  **模型路径**：使用`plot`函数可视化系数如何随正则化强度`λ`变化。
4.  **参数调优**：使用`cv.glmnet`进行K折交叉验证，自动选择最优的`λ`值，并理解“一个标准差”规则。
5.  **独立验证**：将数据分为训练集和验证集，在训练集上拟合模型，在验证集上计算RMSE以选择最终模型。

此外，我们还回顾了在整个模型选择章节中使用过的多种方法，包括最优子集选择、逐步回归、使用验证集、Cp统计量以及交叉验证。我们也看到了使用R Markdown来整合代码、结果和图表，生成可重复、可分享的分析报告的巨大优势。

通过本课程的学习，你已经掌握了使用正则化方法构建更稳健、更可解释的线性模型的关键技能。
