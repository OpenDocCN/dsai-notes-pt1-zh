# 71：K均值聚类算法 🧩

在本节课中，我们将学习一种非常实用的无监督学习方法——K均值聚类。这是一种能在任意维度数据中寻找簇（或分组）的强大工具。为了直观演示，我们将在二维数据上进行操作，以便观察聚类效果。

## 概述

K均值聚类的目标是将数据点划分为K个簇，使得每个点都属于离其最近的簇中心（质心）所代表的簇。算法通过迭代优化来最小化每个簇内数据点到其质心的距离平方和。

## 生成演示数据

为了演示算法，我们首先需要创建一些包含已知簇结构的模拟数据。以下是生成数据的步骤：

首先，我们设定随机数种子以确保结果可重现。

```r
set.seed(101)
```

接着，我们生成一个包含100个观测值和2个变量的初始数据矩阵，数据点来自标准正态分布。

```r
x <- matrix(rnorm(100*2), ncol=2)
```

此时，`x`只是一个高斯分布的散点云。为了创建清晰的簇结构，我们需要为四个不同的簇生成各自的中心点，并将原始数据点向这些中心点偏移。

```r
# 生成四个簇的中心点，每个中心点有两个坐标
xmean <- matrix(rnorm(8, sd=4), ncol=2)

# 随机为100个数据点分配所属的簇（1到4）
which_cluster <- sample(1:4, 100, replace=TRUE)

# 将每个数据点加上其对应簇的中心点，从而形成四个分离的簇
x <- x + xmean[which_cluster, ]
```

最后，我们可以绘制原始数据，用不同颜色代表我们预先设定的真实簇标签。

```r
plot(x, col=which_cluster, pch=19)
```

## 应用K均值聚类

现在，我们将生成的数据（不提供真实标签）输入K均值算法，让它自动发现数据中的结构。我们指定寻找4个簇，并使用15次随机初始值以避免局部最优解。

```r
km_out <- kmeans(x, 4, nstart=15)
```

运行算法后，我们可以查看结果。`km_out`对象包含多项信息：

*   `cluster`: 一个向量，包含每个数据点被分配到的簇标签（1到4）。
*   `centers`: 一个矩阵，显示算法找到的四个簇的质心坐标。
*   `withinss`: 每个簇内部的离差平方和。
*   `tot.withinss`: 所有簇的离差平方和总和。
*   `betweenss`: 簇间离差平方和。
*   一个类似于R²的指标，表示簇中心所能解释的方差百分比，值越高说明聚类效果越好。

```r
print(km_out)
```

## 可视化与结果评估

为了评估K均值算法的表现，我们可以将算法找到的簇与真实的簇进行比较。由于算法分配的簇编号顺序是任意的，直接比较颜色可能不匹配。

首先，我们绘制算法找到的簇，用较大的空心圆圈表示。

```r
plot(x, col=km_out$cluster, cex=2, pch=1, lwd=2)
```

然后，在同一个图上，用较小的实心点叠加显示真实的簇标签。

```r
points(x, col=which_cluster, pch=19)
```

通过这个对比图，我们可以清晰地看到算法分配（外圈）与真实标签（内点）的对应关系。大部分点内外颜色一致，表明算法成功识别了簇结构。少数颜色不匹配的点（例如，一个真实属于某个簇的点被算法分配到了邻近的另一个簇），通常位于簇的边界区域，这在实际聚类分析中是正常现象。

## 总结

本节课我们一起学习了K均值聚类算法。我们从生成具有已知结构的模拟二维数据开始，然后应用K均值算法来发现其中的簇，并通过可视化对比了算法结果与真实情况。K均值是一种高效、应用广泛的聚类方法，其核心思想是迭代优化以使簇内方差最小化。理解其原理并掌握评估结果的方法是使用该工具的关键。
