# 39：岭回归与Lasso中的调参与交叉验证 📊

在本节课中，我们将学习如何为岭回归（Ridge Regression）和Lasso回归选择关键的调优参数——λ。我们将重点探讨为何传统的模型选择准则（如Cp、AIC、BIC）在此不适用，并详细介绍如何使用交叉验证（Cross-Validation）这一强大且实用的方法来选择最优的λ值。

## λ参数的重要性

上一节我们介绍了岭回归和Lasso的基本概念，本节中我们来看看如何为它们选择合适的λ值。

![](img/2f5d99af1e693c4aed839b590e8d05aa_1.png)

![](img/2f5d99af1e693c4aed839b590e8d05aa_3.png)

λ是一个调节惩罚项强度的参数，它强烈地影响着模型在广泛范围内的解。

![](img/2f5d99af1e693c4aed839b590e8d05aa_5.png)

![](img/2f5d99af1e693c4aed839b590e8d05aa_7.png)

*   当 **λ = 0** 时，惩罚项失效，模型退化为普通最小二乘法（OLS），没有正则化。
*   当 **λ → ∞** 时，惩罚项无限大，所有系数都会被压缩至0。

![](img/2f5d99af1e693c4aed839b590e8d05aa_9.png)

![](img/2f5d99af1e693c4aed839b590e8d05aa_10.png)

因此，选择合适的λ至关重要。

## 为何使用交叉验证？

![](img/2f5d99af1e693c4aed839b590e8d05aa_12.png)

![](img/2f5d99af1e693c4aed839b590e8d05aa_13.png)

![](img/2f5d99af1e693c4aed839b590e8d05aa_14.png)

选择λ时，交叉验证是一种很好的技术。我们无法使用其他方法（如Cp、AIC、BIC），因为这些方法都需要知道模型的参数数量 **d**。

![](img/2f5d99af1e693c4aed839b590e8d05aa_16.png)

对于岭回归和Lasso，参数数量 **d** 的定义并不明确。这其实是一个值得思考的有趣问题。

![](img/2f5d99af1e693c4aed839b590e8d05aa_18.png)

![](img/2f5d99af1e693c4aed839b590e8d05aa_19.png)

假设我们从一个包含45个变量的数据集（如信用数据）开始，使用某个特定的λ进行岭回归。即使所有45个系数都不为零（即 **d** 在形式上仍等于变量总数 **p=45**），但由于系数被压缩，模型的“有效自由度”或“有效参数数量”实际上并没有那么多。

这里有一个微妙之处：参数数量不仅仅是我们使用了多少个参数，还包括我们如何拟合它们。对于岭回归和Lasso，收缩（Shrinkage）过程影响了“参数数量”这一概念本身。

因此，对于岭回归和Lasso的调参，使用一种不需要预先知道 **d** 值的方法非常重要，因为 **d** 很难确定。而交叉验证完美地符合这一要求。

## 交叉验证的实施步骤

以下是交叉验证的实施流程，其概念与我们之前应用于子集选择等其他方法时完全相同。

![](img/2f5d99af1e693c4aed839b590e8d05aa_21.png)

![](img/2f5d99af1e693c4aed839b590e8d05aa_23.png)

1.  **数据分割**：将数据随机划分为K个部分（通常K=10）。
2.  **循环训练与验证**：对于每一个部分 `i`（`i` 从1到K）：
    *   将第 `i` 部分作为验证集，其余K-1部分作为训练集。
    *   在训练集上，针对一系列不同的λ值拟合岭回归（或Lasso）模型。
    *   使用拟合好的模型，计算在第 `i` 部分（验证集）上的预测误差。
3.  **汇总误差**：对每个λ值，将其在所有K轮验证中产生的误差相加，得到该λ对应的总交叉验证误差。
4.  **选择最优λ**：绘制交叉验证误差随λ变化的曲线，并选择使误差最小的λ值。

*   在最优λ处（图中虚线位置），许多系数被压缩到接近零（但非精确为零，这是岭回归的特点），而少数几个活跃变量的系数则被保留。

![](img/2f5d99af1e693c4aed839b590e8d05aa_56.png)

对于Lasso，交叉验证的实施方式相同。下图展示了Lasso的交叉验证误差曲线，其中横轴是Lasso解的L1范数与全最小二乘解的L1范数之比。这种缩放使得横轴范围在0到1之间，便于解读。

![](img/2f5d99af1e693c4aed839b590e8d05aa_60.png)

![](img/2f5d99af1e693c4aed839b590e8d05aa_52.png)

![](img/2f5d99af1e693c4aed839b590e8d05aa_62.png)

![](img/2f5d99af1e693c4aed839b590e8d05aa_64.png)

*   **比值 = 1**：对应最小二乘解（无收缩）。
*   **比值 = 0**：对应所有系数为零的解。
*   **中间值**：对应不同程度的Lasso解。
*   **最优值**：交叉验证曲线呈U形，其最小值出现在比值约0.1处，这代表了较强的收缩。

在这个模拟数据示例中（n=50），真实的模型只有两到三个非零系数。Lasso在最优λ处恰好选择了两个非零系数（图中红色和绿色线），而将其余所有系数精确地设为0。这表明，在这个例子中，交叉验证帮助Lasso做出了完全正确的选择。

## 总结

![](img/2f5d99af1e693c4aed839b590e8d05aa_71.png)

![](img/2f5d99af1e693c4aed839b590e8d05aa_73.png)

本节课中我们一起学习了如何为岭回归和Lasso回归选择调优参数λ。我们了解到，由于正则化模型的“有效参数数量”难以界定，传统的基于参数数量的模型选择准则（Cp、AIC、BIC）不再适用。因此，我们引入了交叉验证这一不依赖于参数数量 **d** 的实用方法。通过将数据分割、循环训练验证并汇总误差，我们可以绘制出交叉验证误差曲线，并选择使误差最小的λ值。通过实例分析，我们看到了交叉验证如何帮助岭回归和Lasso在偏差与方差之间取得平衡，并可能实现精确的变量选择（对于Lasso）。
