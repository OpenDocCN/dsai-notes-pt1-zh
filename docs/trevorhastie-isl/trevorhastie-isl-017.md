## 病例对照抽样简介

![](img/4ad34bb77d34130fb154b9a2099c9acb_11.png)

![](img/4ad34bb77d34130fb154b9a2099c9acb_13.png)

上一节我们介绍了逻辑回归模型的基本原理。本节中，我们来看看一种特殊的数据收集方法——病例对照抽样。

![](img/4ad34bb77d34130fb154b9a2099c9acb_15.png)

在讨论的南非心脏病数据中，目标人群的心脏病患病风险约为5%。然而，在我们的样本中，有160个病例（患病）和302个对照（未患病）。因此，样本中的患病风险为 160 / (160 + 302) ≈ 0.35。

![](img/4ad34bb77d34130fb154b9a2099c9acb_17.png)

![](img/4ad34bb77d34130fb154b9a2099c9acb_18.png)

**公式**：样本患病风险 = 病例数 / (病例数 + 对照数) = 160 / 462 ≈ 0.35

![](img/4ad34bb77d34130fb154b9a2099c9acb_20.png)

![](img/4ad34bb77d34130fb154b9a2099c9acb_22.png)

这似乎意味着模型会出现偏差，因为它会高估患病概率。

## 病例对照抽样的原理与优势

![](img/4ad34bb77d34130fb154b9a2099c9acb_31.png)

![](img/4ad34bb77d34130fb154b9a2099c9acb_33.png)

病例对照抽样是流行病学中，尤其是研究罕见疾病时，最常用的工具之一。其操作方法是：尽可能找到所有病例，然后从对照群体中进行抽样。

![](img/4ad34bb77d34130fb154b9a2099c9acb_35.png)

以下是采用病例对照抽样的主要原因：

![](img/4ad34bb77d34130fb154b9a2099c9acb_36.png)

*   **效率高**：最直接的研究方法是进行前瞻性队列研究，即招募大量人群（例如1000人），追踪他们20年，记录风险因素，然后观察谁患病。这种方法虽然有效，但耗时漫长（20年），且需要大量样本才能获得足够的病例数（1000人中可能只有约50个病例）。
*   **成本低、速度快**：病例对照抽样是回顾性的。我们直接寻找已知患病（病例）和未患病（对照）的人，然后记录他们的风险因素。这样可以在短时间内获得大量病例，无需等待多年。

因此，病例对照抽样因其高效和便捷而在流行病学中非常流行。

## 逻辑回归在病例对照抽样下的参数估计

![](img/4ad34bb77d34130fb154b9a2099c9acb_43.png)

![](img/4ad34bb77d34130fb154b9a2099c9acb_44.png)

一个关键问题是：使用这种有偏的样本，我们还能正确估计模型参数吗？

![](img/4ad34bb77d34130fb154b9a2099c9acb_45.png)

对于逻辑回归模型，有一个重要的结论：即使样本是通过病例对照抽样获得的，只要模型设定正确，我们仍然可以无偏地估计出我们关心的**回归参数**（即自变量X的系数）。

**公式**：逻辑回归模型为 log( p/(1-p) ) = β₀ + β₁X₁ + ... + βₖXₖ

然而，模型的**常数项（截距β₀）** 的估计将是错误的，因为它反映了样本中的患病比例，而非总体中的真实比例。

![](img/4ad34bb77d34130fb154b9a2099c9acb_47.png)

![](img/4ad34bb77d34130fb154b9a2099c9acb_48.png)

![](img/4ad34bb77d34130fb154b9a2099c9acb_49.png)

幸运的是，我们可以通过一个简单的变换来校正这个常数项。

## 截距项的校正公式

![](img/4ad34bb77d34130fb154b9a2099c9acb_57.png)

假设：
*   π_true = 总体中的真实患病风险（例如，0.05）。
*   π_sample = 样本中呈现的患病风险（例如，0.35）。

![](img/4ad34bb77d34130fb154b9a2099c9acb_59.png)

![](img/4ad34bb77d34130fb154b9a2099c9acb_60.png)

校正方法如下：我们从模型估计出的截距 β₀_estimated 开始，然后进行如下调整：

![](img/4ad34bb77d34130fb154b9a2099c9acb_62.png)

![](img/4ad34bb77d34130fb154b9a2099c9acb_63.png)

**公式**：校正后的截距 β₀_corrected = β₀_estimated + log( π_true / (1 - π_true) ) - log( π_sample / (1 - π_sample) )

![](img/4ad34bb77d34130fb154b9a2099c9acb_65.png)

![](img/4ad34bb77d34130fb154b9a2099c9acb_66.png)

其中，log( p/(1-p) ) 称为 logit 变换或对数几率。因此，我们只需在原始截距上加上总体真实风险的对数几率，再减去样本风险的对数几率，即可完成校正。

![](img/4ad34bb77d34130fb154b9a2099c9acb_68.png)

![](img/4ad34bb77d34130fb154b9a2099c9acb_69.png)

## 病例对照抽样在现代大数据中的应用

病例对照抽样的思想在现代不平衡数据集中也非常有用。例如，在网页广告点击率预测中，点击概率可能低于0.1%。这意味着如果随机抽样，你会得到海量的“未点击”（0）数据和极少的“点击”（1）数据。

![](img/4ad34bb77d34130fb154b9a2099c9acb_74.png)

![](img/4ad34bb77d34130fb154b9a2099c9acb_75.png)

那么，是否需要使用所有的“0”数据来拟合模型呢？答案是否定的。根据病例对照抽样的逻辑，我们可以对“对照”（即“未点击”的数据）进行抽样。

![](img/4ad34bb77d34130fb154b9a2099c9acb_76.png)

![](img/4ad34bb77d34130fb154b9a2099c9acb_77.png)

![](img/4ad34bb77d34130fb154b9a2099c9acb_78.png)

下图展示了控制组与病例组比例（对照/病例）与回归系数方差之间的关系：

![](img/4ad34bb77d34130fb154b9a2099c9acb_65.png)

核心观点是：参数估计的方差主要取决于病例（稀有类别）的数量。上图表明，当对照与病例的比例达到约5:1或6:1时，方差的下降趋于平缓。这意味着获取更多的对照数据带来的收益是递减的。

因此，在数据极度稀疏的情况下，为每个病例样本大约5-6个对照数据，就可以在一个更易管理的数据集上得到几乎同样好的模型效果。这对于处理现代超大规模数据集非常方便。

## 从二分类到多分类：多项逻辑回归

现在，我们探讨另一个问题：当响应变量Y的类别超过两个时，逻辑回归是否仍然适用？答案是肯定的，它可以很容易地推广到多类别情况。

![](img/4ad34bb77d34130fb154b9a2099c9acb_83.png)

![](img/4ad34bb77d34130fb154b9a2099c9acb_84.png)

![](img/4ad34bb77d34130fb154b9a2099c9acb_85.png)

一种常用的推广形式，也是R语言`glmnet`包中采用的方法，被称为**多项逻辑回归**或**多类别逻辑回归**。

![](img/4ad34bb77d34130fb154b9a2099c9acb_87.png)

![](img/4ad34bb77d34130fb154b9a2099c9acb_89.png)

**公式**：对于K个类别（K>2），给定自变量X，观测属于第k类（k=1,2,...,K）的概率为：
P(Y = k | X) = exp(βₖ₀ + βₖ₁X₁ + ... + βₖₚXₚ) / [ Σ_{j=1}^{K} exp(βⱼ₀ + βⱼ₁X₁ + ... + βⱼₚXₚ) ]

![](img/4ad34bb77d34130fb154b9a2099c9acb_91.png)

![](img/4ad34bb77d34130fb154b9a2099c9acb_92.png)

在这个模型中：
*   **分子**：每个类别k都有自己的线性模型 exp(βₖᵀX)。
*   **分母**：是所有K个类别的线性模型指数函数之和，确保所有类别的概率之和为1。

![](img/4ad34bb77d34130fb154b9a2099c9acb_94.png)

![](img/4ad34bb77d34130fb154b9a2099c9acb_95.png)

这个函数形式通常被称为 **Softmax函数**。

![](img/4ad34bb77d34130fb154b9a2099c9acb_97.png)

![](img/4ad34bb77d34130fb154b9a2099c9acb_98.png)

有数学基础的同学可能会注意到，在这个比率中可以进行一些消元，实际上只需要K-1个线性函数就足够了（就像二分类逻辑回归只需要一个函数）。但这更多是技术细节。对于实际应用，尤其是像`glmnet`这样的实现，上述对称的表示形式更为方便和实用。

![](img/4ad34bb77d34130fb154b9a2099c9acb_100.png)

![](img/4ad34bb77d34130fb154b9a2099c9acb_102.png)

## 总结

![](img/4ad34bb77d34130fb154b9a2099c9acb_106.png)

![](img/4ad34bb77d34130fb154b9a2099c9acb_107.png)

本节课中我们一起学习了：
1.  **病例对照抽样**：一种回顾性抽样方法，通过收集所有可得病例和部分对照来高效研究疾病风险因素，尤其适用于罕见病。
2.  **参数估计与校正**：在使用病例对照样本拟合逻辑回归时，自变量的系数可以被正确估计，但截距项需要根据总体与样本的患病率差异进行校正。
3.  **在不平衡数据中的应用**：病例对照抽样的思想可以用于处理现代大数据中的类别不平衡问题，通过下采样多数类来构建更高效的数据集。
4.  **多项逻辑回归**：逻辑回归模型可以推广到响应变量为多类别的情况，其核心是使用Softmax函数为每个类别分配一个概率。

![](img/4ad34bb77d34130fb154b9a2099c9acb_108.png)

![](img/4ad34bb77d34130fb154b9a2099c9acb_110.png)

理解这些概念，将帮助你更灵活地在不同数据收集场景和应用需求下使用逻辑回归模型。
