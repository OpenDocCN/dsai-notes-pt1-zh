在本节课中，我们将要学习一种名为“模型辅助标注”的技术。这项技术能显著减少为训练目标检测模型而手动标注大量图像所需的时间和精力。

训练一个准确的目标检测模型可能需要成千上万张图像，每张图像都带有多个目标标签。

![](img/fd1472270f8a3c916ae3d34bc64c35be_1.png)

手动标注所有这些图像是一项艰巨的任务。模型辅助标注是解决此问题的一个有效方法，有时也被称为AI辅助标注或AI辅助注释。

![](img/fd1472270f8a3c916ae3d34bc64c35be_5.png)

## 模型辅助标注的核心思想

![](img/fd1472270f8a3c916ae3d34bc64c35be_7.png)

上一节我们介绍了手动标注的挑战，本节中我们来看看模型辅助标注如何工作。在这种方法中，你无需手动标注所有图像，而是使用一个检测模型来帮助标注。

![](img/fd1472270f8a3c916ae3d34bc64c35be_9.png)

以下是模型辅助标注的几种典型应用场景：

![](img/fd1472270f8a3c916ae3d34bc64c35be_11.png)

*   **处理大量未标注图像**：例如，你可能有数万张未标注的图像。与其手动标注所有图像，你可以只标注其中的一个子集。
    ![](img/fd1472270f8a3c916ae3d34bc64c35be_5.png)
    然后使用这个子集来训练一个初始模型。这个模型可能不太准确，但这没关系。接着，使用这个模型自动标注一个更大的图像子集。
    ![](img/fd1472270f8a3c916ae3d34bc64c35be_7.png)
    最后，根据需要手动调整这些自动生成的标签。更新现有标签比从头开始标注要快得多。这种迭代标注新图像子集、审查和更新标签、再训练新检测器的过程，可以节省大量时间和精力。
    ![](img/fd1472270f8a3c916ae3d34bc64c35be_9.png)

![](img/fd1472270f8a3c916ae3d34bc64c35be_13.png)

*   **改进现有模型**：你可能已经有一个模型，但希望通过在新图像上训练来提升其性能。
    ![](img/fd1472270f8a3c916ae3d34bc64c35be_11.png)
    这时，你可以使用现有模型来标注新数据，而不是手动标注，并遵循与之前相同的迭代流程。
    ![](img/fd1472270f8a3c916ae3d34bc64c35be_13.png)

![](img/fd1472270f8a3c916ae3d34bc64c35be_15.png)

*   **为现有模型添加新类别**：你可能有一个已经能准确检测某些目标的模型（无论是同事开发的还是像YOLO这样的知名检测器）。
    ![](img/fd1472270f8a3c916ae3d34bc64c35be_17.png)
    ![](img/fd1472270f8a3c916ae3d34bc64c35be_18.png)
    这个模型可以用来标注那些已有的目标类别，而你只需手动添加新类别的标签。

![](img/fd1472270f8a3c916ae3d34bc64c35be_17.png)

![](img/fd1472270f8a3c916ae3d34bc64c35be_18.png)

以上是一些模型辅助标注可以节省时间和精力的例子。

## 在MATLAB中实践模型辅助标注

现在，让我们看看如何在MATLAB中实现模型辅助标注。

![](img/fd1472270f8a3c916ae3d34bc64c35be_20.png)

首先，打开图像标注器应用程序，并从课程文件中导入新的紧固件图像。
![](img/fd1472270f8a3c916ae3d34bc64c35be_20.png)

![](img/fd1472270f8a3c916ae3d34bc64c35be_22.png)

![](img/fd1472270f8a3c916ae3d34bc64c35be_24.png)

我们需要标签定义，因此创建或导入现有的标签定义。
![](img/fd1472270f8a3c916ae3d34bc64c35be_22.png)
![](img/fd1472270f8a3c916ae3d34bc64c35be_24.png)
![](img/fd1472270f8a3c916ae3d34bc64c35be_25.png)

![](img/fd1472270f8a3c916ae3d34bc64c35be_25.png)

要使用现有模型标注图像，需要创建一个自定义自动化函数。在图像浏览器窗格中，通过点击第一张图像，按住Shift键，再点击最后一张图像，选中所有图像。然后，进入自动化窗口。
![](img/fd1472270f8a3c916ae3d34bc64c35be_27.png)

选择你的自动化函数。课程文件中提供了一个示例自动化函数。
![](img/fd1472270f8a3c916ae3d34bc64c35be_29.png)

![](img/fd1472270f8a3c916ae3d34bc64c35be_27.png)

![](img/fd1472270f8a3c916ae3d34bc64c35be_29.png)

需要注意的是，自动化函数会为每张图像调用。由于模型可能非常大，每次调用都加载模型在计算上非常昂贵。为了避免这种情况，可以使用**持久变量**从课程文件中加载紧固件检测器。这样，模型只会在函数第一次被调用时加载，并在后续调用中持续存在。

接下来，函数将检测器应用于图像，获取唯一的标签，并创建一个正确大小的输出结构。最后，使用一个循环遍历唯一标签，并存储检测到的对象的名称和边界框。
```matlab
% 示例代码结构示意
persistent detector;
if isempty(detector)
    detector = load(‘fastenerDetector.mat’);
end
[bboxes, scores, labels] = detect(detector, I);
```
![](img/fd1472270f8a3c916ae3d34bc64c35be_31.png)

现在，运行代码来自动标注你的图像。
![](img/fd1472270f8a3c916ae3d34bc64c35be_33.png)

然后在图像标注器应用程序中查看结果。
![](img/fd1472270f8a3c916ae3d34bc64c35be_35.png)

![](img/fd1472270f8a3c916ae3d34bc64c35be_31.png)

许多目标被正确标注，或者只需要微小的调整。浏览结果，根据需要修正标签，然后导出新的真实标注数据。
![](img/fd1472270f8a3c916ae3d34bc64c35be_37.png)

![](img/fd1472270f8a3c916ae3d34bc64c35be_33.png)
