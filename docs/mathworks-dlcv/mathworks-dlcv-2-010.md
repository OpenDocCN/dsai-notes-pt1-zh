# 深度学习在计算机视觉中的应用：9：解释网络行为

![](img/d1295ca8373cafecb0a528ff821c75e8_1.png)

在本节课中，我们将学习如何解释深度学习网络的行为。我们将通过分析预测类别分数和生成遮挡敏感度图这两种技术，来理解网络是如何做出预测决策的，从而让深度学习的过程不再那么“黑箱”。

![](img/d1295ca8373cafecb0a528ff821c75e8_3.png)

## 概述

![](img/d1295ca8373cafecb0a528ff821c75e8_5.png)

你已经训练了多种分类网络，但很难知道网络是如何做出预测的。在本视频中，你将使用**预测类别分数**和**遮挡敏感度图**，使深度学习的过程更加透明，并深入了解模型的预测逻辑。

我们将使用两个六角螺母的图像来探讨这些主题：一个被模型正确分类，另一个被错误分类。

## 访问预测类别分数

![](img/d1295ca8373cafecb0a528ff821c75e8_7.png)

上一节我们介绍了理解网络行为的重要性，本节中我们来看看如何获取和分析网络的预测置信度。

![](img/d1295ca8373cafecb0a528ff821c75e8_9.png)

分类网络会为输入图像分配一个标签以及所有预测的类别分数，这些分数是每个类别的概率。**最大预测类别分数**表明了网络对于输入图像属于该类别的置信度。

![](img/d1295ca8373cafecb0a528ff821c75e8_11.png)

现在，让我们看看如何访问这些预测类别分数。

![](img/d1295ca8373cafecb0a528ff821c75e8_13.png)

我们首先加载之前训练好的网络和两张六角螺母图像。

![](img/d1295ca8373cafecb0a528ff821c75e8_15.png)

![](img/d1295ca8373cafecb0a528ff821c75e8_16.png)

请注意，网络期望图像的尺寸为 224 x 224，因此需要调整图像大小。

然后执行分类以获得预测的类别分数。

![](img/d1295ca8373cafecb0a528ff821c75e8_18.png)

![](img/d1295ca8373cafecb0a528ff821c75e8_20.png)

可视化预测类别分数分布的一个有效方法是使用水平条形图。

以下是生成条形图的步骤：
1.  使用 `classify` 函数获取预测标签和分数。
2.  使用 `barh` 函数创建水平条形图，显示前 N 个类别的分数。

![](img/d1295ca8373cafecb0a528ff821c75e8_22.png)

对于正确分类的图像，网络对其预测具有很高的置信度。

需要注意的是，高分并不总是意味着预测正确。

然而，对于被错误分类的六角螺母图像，模型对其预测的置信度约为 61%，这表明在此情况下模型更加不确定。

值得注意的是，无论模型的准确性如何，它总是会返回一个预测。

![](img/d1295ca8373cafecb0a528ff821c75e8_24.png)

根据你的应用场景，你可以设置一个置信度阈值，只接受置信度足够高的预测。

![](img/d1295ca8373cafecb0a528ff821c75e8_26.png)

例如，如果你为此六角螺母图像设置 0.8 的置信度阈值，鉴于其最高预测类别分数低于此阈值，那么你可以从图像中移除该标签。

![](img/d1295ca8373cafecb0a528ff821c75e8_28.png)

总的来说，预测类别分数通过将输入图像视为一个整体来帮助你解释网络行为。然而，它们无法提供关于图像中哪些特定区域对模型预测贡献最大的见解。

![](img/d1295ca8373cafecb0a528ff821c75e8_29.png)

## 使用遮挡敏感度图

![](img/d1295ca8373cafecb0a528ff821c75e8_31.png)

上一节我们介绍了预测类别分数，本节中我们来看看另一种更直观的技术——遮挡敏感度图。

![](img/d1295ca8373cafecb0a528ff821c75e8_33.png)

你可以使用**遮挡敏感度图**来更深入地探索这一点。

![](img/d1295ca8373cafecb0a528ff821c75e8_35.png)

![](img/d1295ca8373cafecb0a528ff821c75e8_36.png)

遮挡敏感度图是一种可视化技术，用于理解图像的不同部分如何影响网络的预测。

![](img/d1295ca8373cafecb0a528ff821c75e8_38.png)

考虑这张四分之一英寸六角螺母的图像。遮挡映射涉及系统地遮盖或遮挡图像的不同部分，并观察模型的类别分数如何变化。

![](img/d1295ca8373cafecb0a528ff821c75e8_40.png)

在这里，当六角螺母被遮挡时，最高的预测类别分数发生了最显著的变化。

![](img/d1295ca8373cafecb0a528ff821c75e8_41.png)

![](img/d1295ca8373cafecb0a528ff821c75e8_42.png)

相反，遮盖背景对模型的置信度分数影响很小。

![](img/d1295ca8373cafecb0a528ff821c75e8_44.png)

将遮挡敏感度图叠加在输入图像上，可以显示图像的哪些区域对于给定类别最为重要。

![](img/d1295ca8373cafecb0a528ff821c75e8_46.png)

一般来说，具有较高正值的区域为该指定类别提供了支持证据。

![](img/d1295ca8373cafecb0a528ff821c75e8_48.png)

相反，具有较低值的区域对模型的置信度影响有限。

![](img/d1295ca8373cafecb0a528ff821c75e8_50.png)

此外，标记为负值的区域则作为支持其他类别的证据。

现在，让我们回到 MATLAB，为两张六角螺母图像生成遮挡敏感度图。

首先，使用 `occlusionSensitivity` 函数。该函数计算给定类别的预测类别分数变化图。

![](img/d1295ca8373cafecb0a528ff821c75e8_52.png)

要显示叠加了遮挡敏感度图的正确分类图像，请使用 `imshow` 绘制图像，然后使用 `hold on` 命令。接着使用 `imagesc` 函数，并以遮挡敏感度图作为输入。这里，我们将 `AlphaData` 调整为 0.5，使颜色图半透明。

![](img/d1295ca8373cafecb0a528ff821c75e8_54.png)

对于错误分类的图像，此过程类似。

运行此代码后，让我们看一下结果。

![](img/d1295ca8373cafecb0a528ff821c75e8_56.png)

对于正确分类的图像，遮挡图高亮了六角螺母覆盖的区域。这表明网络成功识别了其独特的形状。

![](img/d1295ca8373cafecb0a528ff821c75e8_58.png)

![](img/d1295ca8373cafecb0a528ff821c75e8_59.png)

尽管正确分类和错误分类的六角螺母原始图像在人眼看来相似，但它们的遮挡敏感度图显示出一些差异。

对于错误分类的图像，焦点主要集中在背景和周围环境上，只对六角螺母的边缘给予了一些关注。

![](img/d1295ca8373cafecb0a528ff821c75e8_61.png)

这些结果与错误分类图像的预测类别分数一致，其中两个最大的预测分数彼此接近。在这种情况下，模型能够准确识别紧固件的一般类别，但在识别特定类型的六角螺母时遇到了困难。
