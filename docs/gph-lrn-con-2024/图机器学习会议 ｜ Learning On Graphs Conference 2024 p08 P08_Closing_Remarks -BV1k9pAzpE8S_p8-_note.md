# 图机器学习会议：P08：闭幕致辞与展望 🎉

在本节课中，我们将一起回顾2024年图机器学习会议（LOG）的闭幕环节。本节内容将涵盖会议成果总结、重要奖项公布以及关于未来会议的展望。

## 全球社区活动回顾 🌍

今年，我们在全球范围内组织了16场本地聚会。

部分聚会已经成功举办，另一些将在未来几周内举行。

我们衷心感谢所有本地聚会的组织者，并期待看到你们在全球各地分享的聚会照片。

## 会议发展历程与未来展望 🚀

上一节我们回顾了今年的社区活动，本节中我们来看看会议本身的发展。这是LOG会议的第三届。我们已经以线上形式成功举办了三年，现在非常激动地宣布，明年的LOG会议将首次以线下形式举行，地点在美国洛杉矶的加州大学洛杉矶分校。

LOG会议的成功离不开过去三年每一位参与者和贡献者的支持。我们仍将保留线上参与的选项，并在时机成熟时公布更多细节。我们期待明年在洛杉矶的首届线下LOG会议上与大家相见。

## 最佳论文与贡献者奖项公布 🏆

现在进入激动人心的部分，我将宣布最佳论文、杰出领域主席和杰出审稿人奖项。

今年我们共收到194篇投稿，其中79篇进行了海报展示。在所有出色的论文中，最佳论文奖授予了《将力场分解为基于随机轨迹重建的图上的流》。

该论文因其重要性而被选中，它通过将平稳朗之万过程的动力学建模为相空间图表示上的离散状态马尔可夫过程，开辟了图学习的新应用。作者通过将动力学分解为可逆和不可逆分量来研究生物系统，展示了这一成果。祝贺作者们取得的这一卓越成就。

我们还有一个最佳论文荣誉提名奖，授予了《Unravel：一个用于回答知识图谱中图模式查询的神经符号框架》。

该论文引入了一个框架，用于高效地回答不完整知识图谱上的任意图模式查询，涵盖了树状和循环查询，这是在知识图谱中回答图模式查询的神经符号框架领域的一项关键进展。再次祝贺作者们的出色论文。
