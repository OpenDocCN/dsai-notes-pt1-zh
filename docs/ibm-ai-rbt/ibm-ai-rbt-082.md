## 二维数组的创建与可视化

考虑列表 `A`。该列表包含三个嵌套列表，每个列表大小相等。为了简化，每个列表用不同颜色标记。我们可以将此列表转换为NumPy数组，如下所示：

```python
import numpy as np
A = [[11, 12, 13], [21, 22, 23], [31, 32, 33]]
A = np.array(A)
```

将NumPy数组可视化为矩形阵列很有帮助。每个嵌套列表对应矩阵中的一行。

## 数组属性：维度、形状与大小

我们可以使用属性 `ndim` 来获取数组的轴数或维度数，这被称为秩（rank）。这里的“秩”并非指矩阵中线性无关列的数量，而是指嵌套列表的层数。第一个列表代表第一个维度，它内部包含的另一组列表则代表第二个维度或轴。

与一维数组类似，属性 `shape` 返回一个元组。使用矩形表示法有助于理解：元组的第一个元素对应原始列表中嵌套列表的数量（即矩形表示中的行数），第二个元素对应每个嵌套列表的大小（即矩形阵列中的列数）。例如，对于上述数组 `A`，`A.shape` 返回 `(3, 3)`。

![](img/c1cad779d7c3a68b87a91e85045b6b10_7.png)

我们还可以使用属性 `size` 获取数组的总元素数。行数与列数相乘即可得到总元素数，本例中为 `9`。

建议在实验环节尝试不同形状的数组并查看其他属性。

## 二维数组的索引

![](img/c1cad779d7c3a68b87a91e85045b6b10_12.png)

![](img/c1cad779d7c3a68b87a91e85045b6b10_13.png)

我们可以使用方括号访问数组的不同元素。下图展示了列表式表示法与索引约定之间的关系：

![](img/c1cad779d7c3a68b87a91e85045b6b10_15.png)

第一个括号内的索引对应不同颜色的嵌套列表（即行），第二个括号内的索引对应嵌套列表内特定元素的位置（即列）。

使用矩形表示法时，第一个索引对应行索引，第二个索引对应列索引。

![](img/c1cad779d7c3a68b87a91e85045b6b10_19.png)

![](img/c1cad779d7c3a68b87a91e85045b6b10_21.png)

**示例1：** 语法 `A[1, 2]`。索引 `1` 对应第二行，索引 `2` 对应第三列，其值为 `23`。

![](img/c1cad779d7c3a68b87a91e85045b6b10_23.png)

![](img/c1cad779d7c3a68b87a91e85045b6b10_24.png)

![](img/c1cad779d7c3a68b87a91e85045b6b10_27.png)

![](img/c1cad779d7c3a68b87a91e85045b6b10_28.png)

**示例1：** `A[0, 0:2]`
*   第一个索引 `0` 对应第一行。
*   第二个索引 `0:2` 访问前两列。

**示例2：** `A[0:2, 2]`
*   第一个索引 `0:2` 对应前两行。
*   第二个索引 `2` 访问最后一列。

### 数组加法

数组相加的过程与矩阵加法相同。考虑矩阵 `X` 和矩阵 `Y`，每个元素用不同颜色标记。矩阵相加即是将相同位置（相同颜色框）的元素相加。结果是一个与 `X` 或 `Y` 大小相同的新矩阵，其中每个元素是 `X` 和 `Y` 中对应元素的和。

在NumPy中，我们定义数组 `X` 和 `Y`，然后直接相加：

![](img/c1cad779d7c3a68b87a91e85045b6b10_37.png)

![](img/c1cad779d7c3a68b87a91e85045b6b10_38.png)

```python
X = np.array([[1, 0], [0, 1]])
Y = np.array([[2, 1], [1, 2]])
Z = X + Y  # 结果与矩阵加法相同
```

### 数组与标量的乘法

![](img/c1cad779d7c3a68b87a91e85045b6b10_43.png)

将NumPy数组乘以标量与将矩阵乘以标量相同。考虑矩阵 `Y`，将其乘以标量 `2`，即简单地将矩阵中的每个元素乘以 `2`。结果是一个大小相同的新矩阵，其中每个元素都变为原来的两倍。

在NumPy中：

![](img/c1cad779d7c3a68b87a91e85045b6b10_45.png)

![](img/c1cad779d7c3a68b87a91e85045b6b10_46.png)

```python

Z = 2 * Y  # 或 Y * 2
```

### 数组的逐元素乘法（哈达玛积）

![](img/c1cad779d7c3a68b87a91e85045b6b10_51.png)

两个数组的乘法对应逐元素乘积，即哈达玛积（Hadamard Product）。考虑数组 `X` 和 `Y`，哈达玛积即是将相同位置（相同颜色框）的元素相乘。结果是一个与 `X` 或 `Y` 大小相同的新矩阵，其中每个元素是 `X` 和 `Y` 中对应元素的乘积。

在NumPy中，我们可以用一行代码计算两个数组 `X` 和 `Y` 的乘积：

```python
X = np.array([[1, 0], [0, 1]])
Y = np.array([[2, 1], [1, 2]])
Z = X * Y  # 结果与哈达玛积相同
```

### 矩阵乘法

![](img/c1cad779d7c3a68b87a91e85045b6b10_57.png)

我们也可以使用NumPy数组执行矩阵乘法。矩阵乘法稍微复杂一些，以下是基本概述。

![](img/c1cad779d7c3a68b87a91e85045b6b10_59.png)

![](img/c1cad779d7c3a68b87a91e85045b6b10_60.png)

考虑矩阵 `A`（每行用不同颜色标记）和矩阵 `B`（每列用不同颜色标记）。在线性代数中，在将矩阵 `A` 乘以矩阵 `B` 之前，必须确保矩阵 `A` 的列数（本例中为3）等于矩阵 `B` 的行数（本例中为3）。

为了得到新矩阵的第 `i` 行、第 `j` 列元素，我们取 `A` 的第 `i` 行与 `B` 的第 `j` 列的点积。

**计算过程示例：**
*   对于新矩阵的第一行第一列：取 `A` 的第一行与 `B` 的第一列的点积，结果为 `0`。
*   对于新矩阵的第一行第二列：取 `A` 的第一行与 `B` 的第二列的点积，结果为 `2`。
*   对于新矩阵的第二行第一列：取 `A` 的第二行与 `B` 的第一列的点积，结果为 `0`。
*   对于新矩阵的第二行第二列：取 `A` 的第二行与 `B` 的第二列的点积，结果为 `2`。

![](img/c1cad779d7c3a68b87a91e85045b6b10_62.png)

![](img/c1cad779d7c3a68b87a91e85045b6b10_63.png)

![](img/c1cad779d7c3a68b87a91e85045b6b10_64.png)

在NumPy中，我们可以定义数组 `A` 和 `B`，然后执行矩阵乘法：

![](img/c1cad779d7c3a68b87a91e85045b6b10_65.png)

![](img/c1cad779d7c3a68b87a91e85045b6b10_66.png)

![](img/c1cad779d7c3a68b87a91e85045b6b10_67.png)

![](img/c1cad779d7c3a68b87a91e85045b6b10_68.png)

```python
A = np.array([[0, 1, 1], [1, 0, 1]])
B = np.array([[1, 1], [1, 1], [-1, 1]])
C = np.dot(A, B)  # 或使用 A @ B
# 结果数组 C 对应数组 A 和 B 的矩阵乘法结果
```
