## 图像操作：放大与重新生成

如果您对某张图不满意，可以进行操作。以下是可用的操作：

![](img/6913b63ba87c6db6da9f769124b71dc7_27.png)

![](img/6913b63ba87c6db6da9f769124b71dc7_29.png)

*   **放大图像**：在生成的四宫格图片下，点击 `U1`、`U2`、`U3` 或 `U4` 来放大对应的单张图片。
*   **重新生成**：点击 `🔄`（重新生成）按钮，系统将基于原提示词重新生成一组四张新图像。

![](img/6913b63ba87c6db6da9f769124b71dc7_31.png)

例如，点击重新生成按钮，系统会再次处理相同的提示词。

![](img/6913b63ba87c6db6da9f769124b71dc7_24.png)
![](img/6913b63ba87c6db6da9f769124b71dc7_25.png)
![](img/6913b63ba87c6db6da9f769124b71dc7_27.png)
![](img/6913b63ba87c6db6da9f769124b71dc7_29.png)

新生成的图像会有些许变化。在浏览器中打开对比，虽然差异不巨大，但在植物等细节上可以看到不同的处理方式。

![](img/6913b63ba87c63ba87c6db6da9f769124b71dc7_31.png)

## 默认版本说明

通常，您无需每次手动设置版本。Midjourney默认会使用其最新的可用版本（当前是5.2）来生成图像。只有在需要特定风格或效果时，才需要前往设置中选择旧版本或特殊模型（如Niji）。
