## 提升RAG性能的后续步骤 🔍

上一节我们总结了课程的核心目标，本节中我们来看看如何进一步深化学习与实践。作为下一步，我建议你更深入地理解你的数据管道、检索策略和大语言模型提示，以帮助提升RAG的性能。
