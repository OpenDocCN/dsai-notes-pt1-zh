## 课程整体安排 🗺️

本课程的整体安排是以项目实战为驱动。在讲解几个知识点之后，我们会动手完成一个稍大型的项目。

![](img/73d913d70ea59738b60ece5627ceb411_6.png)

上一节我们介绍了课程的整体安排，本节中我们来看看课程的具体风格与特点。

## 课程风格与特点 ✨

![](img/73d913d70ea59738b60ece5627ceb411_10.png)

本课程的风格是通俗易懂的。课程将采用最接地气的方式，讲解其中比较复杂难懂的知识点。

![](img/73d913d70ea59738b60ece5627ceb411_12.png)

以下是本课程的核心特点：

![](img/73d913d70ea59738b60ece5627ceb411_14.png)

*   **实战驱动**：学习过程由项目实战驱动，帮助巩固知识点。
*   **循序渐进**：课程难度从零基础开始，逐渐进阶。
*   **业务导向**：在项目实战中，不仅练习技术，也熟悉完成一个实际项目的业务流程。

## 课程难度与要求 ⚙️

![](img/73d913d70ea59738b60ece5627ceb411_16.png)

课程难度相当于从零基础到进阶。课程开始时，会讲解计算机视觉中最基本的操作，以及如何在OpenCV中使用函数构建小案例。后续我们会将这些知识融入更大型的项目中。

![](img/73d913d70ea59738b60ece5627ceb411_18.png)

课程难度会逐渐提升，并且后续部分包含了非常丰富的项目实战内容，这些需要大家投入时间和精力来熟悉。

![](img/73d913d70ea59738b60ece5627ceb411_20.png)

学习本课程有一个基本前提：需要掌握Python语言。

![](img/73d913d70ea59738b60ece5627ceb411_22.png)

以下是具体的要求说明：

![](img/73d913d70ea59738b60ece5627ceb411_24.png)

*   **Python基础**：所有实战案例、大型项目以及OpenCV函数的使用均基于Python。不要求精通，但需要熟悉并能看懂代码。
*   **环境准备**：关于如何安装OpenCV、配置Notebook及相关IDE，将在下一节课中详细说明。

![](img/73d913d70ea59738b60ece5627ceb411_26.png)

> 提示：如果你对Python完全不熟悉，可以参考讲师提供的Python快速入门教程。

![](img/73d913d70ea59738b60ece5627ceb411_28.png)
