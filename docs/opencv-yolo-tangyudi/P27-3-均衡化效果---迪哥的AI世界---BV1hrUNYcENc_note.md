## 全局直方图均衡化

![](img/cb257ce30b52464643f3a56037460a1d_10.png)

上一节我们介绍了直方图的概念，本节中我们来看看如何使用OpenCV进行均衡化。OpenCV提供了`cv2.equalizeHist()`函数来实现全局直方图均衡化。

![](img/cb257ce30b52464643f3a56037460a1d_12.png)

![](img/cb257ce30b52464643f3a56037460a1d_14.png)

以下是进行全局均衡化的步骤：

![](img/cb257ce30b52464643f3a56037460a1d_16.png)

1.  调用`cv2.equalizeHist()`函数。
2.  将原始图像作为参数传入。
3.  函数会返回均衡化后的图像。

```python
# 进行全局直方图均衡化
equ = cv2.equalizeHist(img)
```

![](img/cb257ce30b52464643f3a56037460a1d_18.png)

![](img/cb257ce30b52464643f3a56037460a1d_20.png)

均衡化完成后，我们可以绘制并对比原始图像与均衡化后图像的直方图。原始图像的直方图分布集中（较“尖”），而均衡化后的直方图分布变得更为平缓和分散（变“胖”），这表明像素值的分布变得更加均匀。

![](img/cb257ce30b52464643f3a56037460a1d_22.png)

## 全局均衡化的局限性

然而，全局均衡化并非总是最佳选择。当我们处理一些包含丰富细节的图像时，全局操作可能会带来问题。

例如，在某张测试图像中，均衡化后整体亮度虽然得到提升，但人脸部分的细节却丢失了。这是因为全局均衡化是对整幅图像的像素值进行重新分配，可能会将某些局部区域的特征“平均”掉。

## 自适应直方图均衡化

为了解决全局均衡化可能丢失细节的问题，我们可以采用分块处理的思路，即**自适应直方图均衡化**。其核心思想是将图像划分为多个小格子（例如8x8的块），每个格子独立进行自身的均衡化。

![](img/cb257ce30b52464643f3a56037460a1d_32.png)

OpenCV提供了`cv2.createCLAHE()`方法来创建自适应均衡化器。

![](img/cb257ce30b52464643f3a56037460a1d_34.png)

以下是使用自适应均衡化的步骤：

![](img/cb257ce30b52464643f3a56037460a1d_36.png)

1.  创建CLAHE对象，可设置格子大小（`tileGridSize`）和对比度限制（`clipLimit`）。
2.  应用该对象到输入图像上。

```python
# 创建自适应直方图均衡化器
clahe = cv2.createCLAHE(clipLimit=2.0, tileGridSize=(8,8))
# 应用自适应均衡化
cl1 = clahe.apply(img)
```

![](img/cb257ce30b52464643f3a56037460a1d_38.png)

> 注：为了避免分块处理在格子边界处产生不自然的痕迹，OpenCV在内部还进行了插值等额外处理。

![](img/cb257ce30b52464643f3a56037460a1d_40.png)

## 综合效果对比

最后，我们将原始图像（A）、全局均衡化结果（B）和自适应均衡化结果（C）放在一起进行对比。

通过观察可以发现：
*   **图像C（自适应）** 在提升整体亮度的同时，很好地保留了图像中的纹理和细节，效果优于图像B。
*   **图像B（全局）** 虽然提升了对比度，但损失了部分细节。
*   **图像A（原始）** 对比度最低。

![](img/cb257ce30b52464643f3a56037460a1d_42.png)

这个对比说明，当需要保留图像具体细节时，自适应均衡化通常是比全局均衡化更好的选择。当然，局部处理也可能受到噪声的影响，在实际应用中需要根据具体情况权衡。

## 课程总结

![](img/cb257ce30b52464643f3a56037460a1d_48.png)

本节课中我们一起学习了：
1.  **直方图的含义**：图像中像素值分布的统计图表。
2.  **直方图均衡化的操作**：使用`cv2.equalizeHist()`对图像进行全局对比度增强。
3.  **均衡化的效果与局限**：能提升整体对比度，但可能导致局部细节丢失。
4.  **自适应均衡化**：使用`cv2.createCLAHE()`进行分块均衡化，能在增强对比度的同时更好地保留细节。
