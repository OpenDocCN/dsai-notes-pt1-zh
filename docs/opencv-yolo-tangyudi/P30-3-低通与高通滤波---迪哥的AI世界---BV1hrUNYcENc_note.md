在本节课中，我们将学习如何使用傅里叶变换在频域中实现图像的低通滤波和高通滤波。我们将通过构建掩码来保留或过滤特定的频率成分，从而实现对图像不同特征的提取。

---

## 概述

上一节我们介绍了傅里叶变换的基本概念。本节中，我们来看看如何利用傅里叶变换的结果，通过构建掩码来分离图像中的低频和高频信息，从而实现低通与高通滤波。

![](img/06fccdc033b140acd3c21d2e86db173a_4.png)

## 计算图像中心与尺寸

![](img/06fccdc033b140acd3c21d2e86db173a_6.png)

首先，我们需要读取图像并进行傅里叶变换。为了后续构建掩码，我们需要计算图像的中心点坐标。

![](img/06fccdc033b140acd3c21d2e86db173a_8.png)

以下是计算图像尺寸和中心点的代码：

![](img/06fccdc033b140acd3c21d2e86db173a_10.png)

```python
# 假设 `dft_shift` 是经过傅里叶变换并中心化后的结果
rows, cols = dft_shift.shape[:2]
crow, ccol = int(rows/2), int(cols/2)  # 计算中心点坐标
```

![](img/06fccdc033b140acd3c21d2e86db173a_12.png)

![](img/06fccdc033b140acd3c21d2e86db173a_14.png)

计算中心点的目的是为了在频域中定位低频区域，因为经过 `fftshift` 操作后，低频成分集中在频谱图的中心。

![](img/06fccdc033b140acd3c21d2e86db173a_16.png)

## 构建低通滤波器 🎯

![](img/06fccdc033b140acd3c21d2e86db173a_18.png)

低通滤波器的目的是保留图像中的低频信息（如平滑区域和整体轮廓），过滤掉高频信息（如边缘和噪声）。在频域中，低频位于中心区域。

以下是构建低通滤波器掩码的步骤：

1.  创建一个与图像尺寸相同的全零矩阵。
2.  以频谱中心点为原点，指定一个矩形区域（例如上下左右各30像素）。
3.  将该矩形区域内的值设为1（白色），其余区域保持为0（黑色）。这个矩阵就是我们的低通掩码。

![](img/06fccdc033b140acd3c21d2e86db173a_20.png)

```python
# 创建低通滤波器掩码
mask_low = np.zeros((rows, cols, 2), np.uint8)
mask_low[crow-30:crow+30, ccol-30:ccol+30] = 1
```

这个掩码中，值为1的区域（中心白色矩形）将被保留，值为0的区域（周围黑色部分）将被过滤掉。

![](img/06fccdc033b140acd3c21d2e86db173a_22.png)

![](img/06fccdc033b140acd3c21d2e86db173a_24.png)

## 应用掩码与逆变换

构建好掩码后，我们需要将其应用到频域数据上，然后进行逆傅里叶变换以恢复图像。

![](img/06fccdc033b140acd3c21d2e86db173a_26.png)

以下是应用低通滤波的完整流程：

![](img/06fccdc033b140acd3c21d2e86db173a_28.png)

1.  **结合掩码**：将掩码与傅里叶变换后的频谱图进行逐元素相乘。这样，只有掩码中为1的频率成分被保留。
    ```python
    fshift_low = dft_shift * mask_low
    ```
2.  **频率中心还原**：在进行逆变换之前，需要将频谱中心移回左上角，即执行 `ifftshift` 操作。
    ```python
    f_ishift_low = np.fft.ifftshift(fshift_low)
    ```
3.  **逆傅里叶变换**：执行逆离散傅里叶变换（IDFT），将处理后的频域数据转换回空间域。
    ```python
    img_back_low = cv2.idft(f_ishift_low)
    ```
4.  **计算幅度谱**：逆变换的结果包含实部和虚部，我们需要计算其幅度来得到可显示的图像。
    ```python
    img_back_low = cv2.magnitude(img_back_low[:,:,0], img_back_low[:,:,1])
    ```

![](img/06fccdc033b140acd3c21d2e86db173a_30.png)

应用低通滤波后，得到的图像会变得模糊，因为只保留了低频信息，丢失了构成清晰边缘的高频细节。

![](img/06fccdc033b140acd3c21d2e86db173a_32.png)

## 构建高通滤波器 🔍

![](img/06fccdc033b140acd3c21d2e86db173a_34.png)

高通滤波器的目的与低通相反，它保留图像中的高频信息（如边缘和纹理），过滤掉低频信息（如平滑区域）。

![](img/06fccdc033b140acd3c21d2e86db173a_36.png)

构建高通滤波器掩码的逻辑与低通正好相反：

![](img/06fccdc033b140acd3c21d2e86db173a_38.png)

1.  创建一个全一矩阵（表示最初保留所有频率）。
2.  将中心矩形区域的值设为0（表示过滤掉低频）。

![](img/06fccdc033b140acd3c21d2e86db173a_40.png)

```python
# 创建高通滤波器掩码
mask_high = np.ones((rows, cols, 2), np.uint8)
mask_high[crow-30:crow+30, ccol-30:ccol+30] = 0
```

![](img/06fccdc033b140acd3c21d2e86db173a_42.png)

这个掩码中，中心区域为0（黑色）被过滤，周围区域为1（白色）被保留。

![](img/06fccdc033b140acd3c21d2e86db173a_44.png)

应用高通滤波的后续步骤（结合掩码、逆变换）与低通滤波完全相同。得到的结果图像将主要包含物体的轮廓和边界，而内部细节区域会变暗或消失。

![](img/06fccdc033b140acd3c21d2e86db173a_46.png)

## 频域处理的优势

![](img/06fccdc033b140acd3c21d2e86db173a_48.png)

你可能会问，为什么要在频域进行这些处理？

![](img/06fccdc033b140acd3c21d2e86db173a_50.png)

![](img/06fccdc033b140acd3c21d2e86db173a_52.png)

在空间域（原始图像）中直接区分或操作低频与高频信息非常困难。而在频域中，不同频率的成分被分离开来，低频集中在中心，高频分布在四周。这种结构化的分布使得我们能够通过简单的掩码操作（如画矩形）来精确地选择或过滤特定频率，操作变得直观且高效。因此，许多图像处理任务会先将图像转换到频域进行处理，以获得更好的效果和更高的效率。

## 总结

![](img/06fccdc033b140acd3c21d2e86db173a_54.png)

本节课中我们一起学习了傅里叶变换在图像滤波中的应用。我们掌握了如何通过构建掩码来实现低通滤波（保留低频使图像模糊）和高通滤波（保留高频突出边缘）。关键在于理解频域中频率的分布规律，并利用 `fftshift`、掩码乘法以及 `ifftshift` 和 `idft` 这一套流程在频域与空间域之间进行转换。这为我们后续进行更复杂的图像处理奠定了基础。
