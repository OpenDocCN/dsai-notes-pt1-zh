# 015：电子表格中的字符串操作

![](img/71ec6d906123ee3335b42ffd287ffabb_0.png)

在本节课中，我们将学习如何在电子表格中使用字符串函数。字符串是数据分析中常见的数据类型，掌握处理字符串的技巧对于数据清洗和提取关键信息至关重要。

## 字符串函数回顾

上一节我们介绍了SQL中的字符串函数，本节中我们来看看这些函数在电子表格中的具体应用。

字符串函数在SQL和电子表格中通常具有相似的功能。本视频将重点回顾**LEN**、**LEFT**、**RIGHT**和**FIND**函数。

## 实际案例：共享单车数据集

回到我们的共享单车数据集，让我们查看其中一个电子表格。

这是行程数据表格之一，其中“开始时间”和“结束时间”列包含表示每次骑行日期和时间的字符串。这些数据很有用，但通常我们只需要这些字符串的一部分来构建公式或回答问题。

例如，这些字符串包含日期和时间等多个数据点。但如果我们要计算开始时间的平均值，就不需要日期部分。我们可以使用**LEFT**、**RIGHT**和**FIND**函数将时间戳拆分到单独的列中。

## 拆分日期时间字符串

让我们构建一个简单的公式来分离这些字符串中的日期部分。

我们知道**LEN**函数可以告诉我们字符串的长度。首先检查这些日期时间字符串的长度。

以下是公式的第一部分：

```excel
=LEN(A2)
```

然后我们选择一个包含日期时间字符串的单元格。这些字符串的长度为19个字符。

## 使用FIND函数定位字符

我们可以使用**FIND**函数定位字符串中的特定字符。请注意，这个函数区分大小写。因此，如果使用**FIND**提取子字符串，请确保正确输入子字符串。

我们注意到所有日期时间字符串都有一个空格分隔日期和时间戳。因此，我们可以使用**FIND**来确定日期结束的位置。

```excel
=FIND(" ", A2)
```

空格是这个字符串中的第11个字符。因此，时间戳子字符串将从第12个字符开始。

## 提取字符串的特定部分

我们可以使用**LEFT**和**RIGHT**函数选择要在新列中隔离的字符串部分。

我们将对其中一个单元格使用**RIGHT**函数，表示要获取右侧部分：

```excel
=RIGHT(A2, LEN(A2)-FIND(" ", A2))
```

**LEFT**函数的工作方式完全相同：

```excel
=LEFT(A2, FIND(" ", A2)-1)
```

现在我们可以将此公式应用于C列的其余部分，以提取这些时间戳。

## 字符串处理的重要性

作为数据分析师，能够处理字符串是一项关键技能，尤其是在处理来自外部来源的数据时。

希望您现在对在SQL和电子表格中应用**LEN**、**RIGHT**、**LEFT**和**FIND**函数更加熟悉。稍后，我们将使用这些函数执行更复杂的公式。您可以尝试在自己的数据上使用这些函数，甚至可以尝试一些我们今天使用的开放数据。
