## 研究兴趣：蛋白质进化

在 Harms 实验室，我们研究蛋白质进化。生命进化出了执行所有生命必需功能的、种类繁多的蛋白质，这是一个惊人的过程。蛋白质是由氨基酸亚基组成的线性链，折叠成美丽的三维结构，成为可以执行所有生命过程的小型分子机器。

进化在过去数十亿年中不断创新、多样化，创造了这个庞大的阵列以满足生命的所有需求。为了探究这个问题，我们首先要问：它们是如何进化出简单的功能变化并进行微小的创新的？这就是我今天问题的起点。

![](img/8f08c26f76ecd14129d7e0ab7949ecb3_11.png)

## 案例研究：类固醇激素受体

![](img/8f08c26f76ecd14129d7e0ab7949ecb3_13.png)

我将以类固醇激素受体为例。这不是我的数据，甚至不是我的工作，我只是想展示我们可以重现文献中的结果。我对这种蛋白质并非专家，但会做一个简要介绍。

这些受体对于调节 DNA 上的各种转录很重要。它们的工作原理是等待激素分子到来并与之结合，这会诱导蛋白质构象变化，使其能够结合到 DNA 上，从而启动下游 DNA 的转录。这样就可以靶向许多不同的基因。

这类受体主要有两类：雌激素受体和非芳香族类固醇受体。有趣的是，它们识别不同的激素分子来开启转录功能，但它们在结构和序列上非常相似。这意味着它们有一个共同的祖先，后来才分化成这两类。这正是引文右下角那些研究者探讨的问题。

![](img/8f08c26f76ecd14129d7e0ab7949ecb3_15.png)

## 系统发育学与祖先序列重建

为了研究这个问题，他们使用了系统发育学。我们从各种生物体中获取这些蛋白质的序列，这得益于我们能够对许多生物体基因组进行测序的先进技术。我们可以从数据库中提取这些蛋白质的序列。

对于雌激素受体和类固醇受体，提取它们的序列并进行比对。每一行代表不同物种中的不同雌激素受体蛋白。根据序列信息的相似性，可以构建一个显示蛋白质之间关系的层次拓扑树。

![](img/8f08c26f76ecd14129d7e0ab7949ecb3_17.png)

通过这样做，可以找到并推断出发生分化的节点，这很可能是这两类受体分化的时间点。然后，可以利用这些序列信息来推断该节点的序列，这种技术被称为**祖先序列重建**。它让时间倒流，查看沿途每个节点的序列，从而推断出这个祖先。

接着，可以从这些程序中获取该序列，并通过我们称之为**蛋白质复活**的过程合成它。请花点时间欣赏这一点，因为这些蛋白质距今已有2亿多年，它们只存在于最早的脊椎动物中。你在实验室中复活了自然界今天已不存在的蛋白质。

然后，你可以测量它们的功能，看看该蛋白质在那个时间点会做什么。接着，你可以开始描述沿着这个节点向这些分化分支发生的变化，以及导致这些新功能和创新的突变是什么。

## 当前分析流程与挑战

![](img/8f08c26f76ecd14129d7e0ab7949ecb3_19.png)

祖先序列重建和系统发育学通常通过以下四个步骤完成：
1.  从数据库收集序列。
2.  使用序列比对工具比对序列。
3.  构建系统发育树，推断比对序列之间的关系。
4.  基于该树拓扑结构重建祖先。

听起来很简单，只有四步。但每一步都有多个程序可以完成，而且这些程序是用各种不同的低级语言编写的，并不总是在某些机器上编译，也不能跨平台工作。不同版本可能得到不同结果，输入输出格式也可能因版本而异。

因此，你面临多重工具选择的问题，并且需要知道如何将数据放入其格式中。此外，作为生物信息学家，你必须处理右侧显示的所有文件格式。顶部的部分是所有测序格式（实际上更多），然后是所有不同的树格式，甚至在单一树格式内，也可能需要节点标签与分支长度等不同要求。

由此产生了三类问题，构成了进入系统发育学和进化生物学领域的巨大障碍：
1.  需要在不同机器上编译程序。
2.  需要处理多种格式，大量时间浪费在管理这些格式并将数据转换为必要格式上。
3.  这些程序通常只提供命令行界面，用 C++ 或 Java 等语言编写，难以轻松安装、管理并以可重复且易于在多台机器上安装的方式进行交互。至少10年前，还没有 Python 工具。

## 引入 PhyloPandas

![](img/8f08c26f76ecd14129d7e0ab7949ecb3_31.png)

![](img/8f08c26f76ecd14129d7e0ab7949ecb3_33.png)

为此，我将介绍我的第一个软件，它处于 Beta 版（不是 Alpha 版）。这个工具叫做 **PhyloPandas**，它被定义为用于系统发育学数据的 Pandas DataFrame。

![](img/8f08c26f76ecd14129d7e0ab7949ecb3_35.png)

![](img/8f08c26f76ecd14129d7e0ab7949ecb3_37.png)

最好的展示方式就是实时演示。就像 Pandas 一样，它只是定义了一些额外的读取调用。如果你了解 Pandas，Pandas 有各种不同格式的读取调用，这里也一样。我添加了许多特定格式的读取功能，用于之前幻灯片中提到的那些工具。FASTA 是非常流行的一种，还有 Nexus、Phylip、FastQ 等等。

我可以演示读取不同格式的相同数据，你会得到一个漂亮的 DataFrame。它定义了一些列，如描述、标签，特别是序列列，这非常重要。它还会创建一个唯一的 ID，但这是为了内部使用。它可以处理各种不同格式，如 Clustal 和 Phylip，根据读取的数据，你基本上会得到相同的 DataFrame。

当然，接下来是写入功能。它利用 Pandas 最近引入的 Accessor Extension API 来实现。如果你不熟悉，我可以稍后解释。这允许你将一个具有特定用例额外方法的对象附加并注册到 DataFrame 上。我在这里创建的访问器关键字是 `.phylo`。通过调用 `dataframe.phylo.to_...`，你可以写入各种不同格式。

![](img/8f08c26f76ecd14129d7e0ab7949ecb3_39.png)

这使得格式转换变得非常容易。我可以读入一个 Phylip 文件，然后输出一个 FASTA 文件，非常方便。

![](img/8f08c26f76ecd14129d7e0ab7949ecb3_41.png)

PhyloPandas 不仅能读取序列数据，还能读取树数据。这是独特的，因为很多地方没有这种功能，它可以将这种非人类可读格式的树数据扁平化。你可以看到现在我们有了描述父节点、节点类型（叶节点还是内部节点）的列。

![](img/8f08c26f76ecd14129d7e0ab7949ecb3_43.png)

![](img/8f08c26f76ecd14129d7e0ab7949ecb3_45.png)

![](img/8f08c26f76ecd14129d7e0ab7949ecb3_46.png)

因为它是 Pandas DataFrame，你可以使用所有很酷的切片操作。例如，我调用了 `.leaves`，使用标准的 Pandas 语法，突然之间我就可以切片出我关心的元素。

![](img/8f08c26f76ecd14129d7e0ab7949ecb3_48.png)

![](img/8f08c26f76ecd14129d7e0ab7949ecb3_50.png)

更进一步，真正的魔力在于，你不必将一种格式读入单个 DataFrame。你可以将多个信息片段读入单个 DataFrame，并且它知道如何处理。具体来说，想象你读取了序列数据，然后有描述这些序列如何组合的树数据。它们会一起读入。

![](img/8f08c26f76ecd14129d7e0ab7949ecb3_52.png)

![](img/8f08c26f76ecd14129d7e0ab7949ecb3_54.png)

你会看到树的叶节点现在都有了序列，而内部节点还没有，因为树数据还没有这些信息。这样，我们就将这两个复杂的数据集压缩到了一个易于映射的单一 DataFrame 中。这就是 PhyloPandas 的介绍。

![](img/8f08c26f76ecd14129d7e0ab7949ecb3_56.png)

![](img/8f08c26f76ecd14129d7e0ab7949ecb3_58.png)

## 重现类固醇受体进化研究

整个承诺是，我将通过类固醇受体的祖先序列重建来讲述 SR 进化。我本想实时演示，但计算大约需要40分钟，所以就不让大家等待了。

在这个代码仓库中，我将导入我的系统发育学项目。这是实际重复它的代码，只需大约6行代码就能完成完整的祖先序列重建。我将直接加载那个已经保存的项目，里面有我完全重建的祖先。

你可以在这里的列中看到序列，我可以将其可视化，树会很难看（因为是预 Alpha 版）。但如果你看得足够久（虽然我时间不够了），这就是他们的树的样子，看起来一模一样。我们重现了他们的确切结果。

这是他们论文中的树。这里有一个小事实：这是他们最感兴趣的祖先，它告诉你雌激素受体和类固醇受体从这里分化。这是一个他们通过这个过程在实验室重建的3亿年前的蛋白质。

为了简化那棵树，使用第一张幻灯片的颜色，基本上可以这样总结：我们整个雌激素受体类在顶部，通过这种重建，他们测量了整个树最远的最后共同祖先。这个非常非常古老的蛋白质，他们发现它看起来像一个雌激素受体。

在大约1亿年的时间里，它进化了，或者说实际上发生了基因复制，这意味着它复制了一个基因副本，其中一个被限制保持相同的雌激素受体功能，另一个则被允许分化，学习接收其他激素分子的新方式。这就是你在绿色部分看到的。

他们实际上描述了这两个节点之间发生的突变，并确定了导致这种功能创新的关键因素。看起来最古老的祖先是雌激素受体。我鼓励你去阅读那些论文，内容非常有趣。
