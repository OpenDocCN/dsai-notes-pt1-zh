## IPyvolume 的诞生与发展

在 IPyvolume 之前，我曾尝试在网页上使用 WebGL 和 jQuery 滑块创建一个概念验证。虽然效果不错，但方法笨拙，需要向笔记本中注入 JavaScript 和数据，这并非良策。

因此，我决定基于 IPython widgets 构建 IPyvolume，这被证明是其成功的关键。它填补了当时 Jupyter Notebook 缺乏交互式 3D 库的空白。如今，IPyvolume 在其细分领域已相当流行，并且安装极其简单，只需 `pip install ipyvolume` 即可。

## 为什么选择 IPyvolume？

以下是选择 IPyvolume 的几个关键原因：
*   **高度交互**：可以流畅旋转视图，响应迅速。
*   **双向通信**：内核可以获取前端（如相机状态）的信息并据此执行操作。
*   **可移植性**：可视化结果可以保存为 HTML 文件，在浏览器或平板电脑上独立查看。
*   **高性能**：能够处理多达百万级的数据点，所有动画均在着色器中完成，效率很高。
*   **云端友好**：Jupyter 生态非常适合“将代码带到数据旁”的工作流。在浏览器客户端进行可视化，无需复杂的远程 OpenGL 设置。

## 核心 API 与基础绘图

![](img/f6efcd6e1f4edb53ad9377f7e60054ac_30.png)

对于笔记本环境，一个简单直观的 API 至关重要。IPyvolume 的设计遵循了这一原则，只需几行代码即可创建图表。

首先进行导入并创建数据，然后调用绘图函数。

```python
import ipyvolume as ipv
import numpy as np
# 创建示例数据
x, y, z = np.random.random((3, 1000))
# 创建图形、绘制散点图并显示
ipv.figure()
ipv.scatter(x, y, z)
ipv.show()
```

执行上述代码后，你将获得一个 3D 散点图。由于它是一个 Jupyter 小部件，你可以动态修改其属性。例如，将标记形状从球体改为立方体，并且这个变化过程是平滑动画的，这对于观察模型参数如何影响数据呈现非常有用。

## 与 IPyWidgets 集成构建 GUI

IPyvolume 构建于 IPyWidgets 之上，因此可以轻松创建图形用户界面来控制可视化。

我们可以再次使用之前的图形，但这次不是直接修改代码，而是创建一个下拉菜单让用户选择标记形状（球体、立方体或菱形）。同样，也可以创建一个滑块来动态调整散点的大小。这使得在探索数据时，能够快速构建出交互式控件。

## 支持的可视化类型

接下来，我们简要了解一下 IPyvolume 支持的各种 3D 绘图类型。

**矢量图**：类似于散点图，但用箭头表示，具有方向信息。颜色和方向的变化都支持平滑动画。

**网格与表面**：可以绘制由三角形定义的 3D 网格。对于简单的表面，无需手动定义三角形，直接使用 `plot_surface` 函数即可。网格还支持纹理映射，例如，可以将大脑的不同剖面图或动态活动视频作为纹理贴到 3D 模型上。

**线图**：绘制 3D 空间中的线条。

**体绘制**：这是 IPyvolume 的核心功能之一，用于渲染 3D 体数据（如一个数据立方体）。用户可以交互式地调整传输函数等参数来探索数据内部结构。

![](img/f6efcd6e1f4edb53ad9377f7e60054ac_32.png)

![](img/f6efcd6e1f4edb53ad9377f7e60054ac_34.png)

**等值面**：从 3D 数据中提取并渲染出特定值的等值面。计算在内核端完成，当调整等值水平时，三角面片数量会动态变化。

## 性能与大数据处理

IPyvolume 性能出色，可以流畅交互地处理百万级数据点。对于更大的数据集，例如拥有 330 万行的暗物质模拟数据，直接绘制散点图不可行。解决方案是将其转换为 3D 直方图（一个低分辨率立方体）进行渲染。

通过双向通信，当用户缩放关注某个区域时，前端会通知内核，内核则计算该区域的高分辨率数据并发送给前端更新显示。这类似于 2D 的 DataShader 思想在 3D 中的应用。

对于非表格型的大数据（如天文数据立方体），同样可以采用这种“按需加载”的策略，在网络条件有限时，先显示降采样的整体视图，在用户缩放时再传输局部的高清数据。

## 高级功能：选择与动画

![](img/f6efcd6e1f4edb53ad9377f7e60054ac_40.png)

![](img/f6efcd6e1f4edb53ad9377f7e60054ac_42.png)

![](img/f6efcd6e1f4edb53ad9377f7e60054ac_44.png)

**数据选择**：IPyvolume 支持基础的选择功能（如套索选择）。当前端进行选择时，相关信息会发送回内核，内核可以据此计算被选数据的统计信息（如平均坐标），实现了真正的双向交互。

**时间序列动画**：如果数据包含时间维度（例如，粒子的轨迹数据），IPyvolume 会自动将第一维识别为时间。通过控制“序列索引”，可以在不同时间步之间切换，查看动态演变过程。结合 IPyWidgets 的播放控件，可以创建平滑的动画。

![](img/f6efcd6e1f4edb53ad9377f7e60054ac_46.png)

**虚拟现实与特殊投影**：可视化结果可以导出为 HTML，并在手机结合 Google Cardboard 等设备上实现 VR 体验。此外，还支持穹顶投影等特殊显示方式，用于天文馆演示或制作 3D 电影。

利用 `ipyvolume.pylab` 子包，可以访问底层 Three.js 的相机、场景和动画支持。用户可以添加关键帧来定义相机路径，创建复杂的漫游动画。

更进一步，可以使用 `ipywebrtc` 包来录制这些动画，生成视频文件用于分享或演示。这展示了将不同小部件（如视频、摄像头）作为纹理融入 IPyvolume 场景的强大能力，虽然这更多是技术演示，但体现了 Jupyter 小部件框架的灵活性。

![](img/f6efcd6e1f4edb53ad9377f7e60054ac_49.png)

## 未来展望

IPyvolume 的未来版本将更深度地集成 Three.js，让用户能完全控制相机、灯光、材质等。同时，正在开发多体积渲染和顺序无关透明等高级功能。项目也致力于更好地与其他可视化库（如 Napari）协作，并探索在 JVM 语言乃至 C++ 中的使用支持。
