## 概述 📋

Dask 是一个灵活的并行计算库，它能够与现有的 Python 生态系统（如 NumPy、Pandas）无缝集成。本教程将从基础概念开始，逐步介绍 Dask 的核心组件，包括 `delayed`、`array`、`dataframe` 以及分布式调度器。我们将通过实际代码示例，演示如何将串行代码轻松转换为并行代码。

像这样的延迟对象足够聪明，如果您执行 `delayed` 对象加一（在这种情况下，它们都被称为 `y`），如果您执行第一个 `y` 加第二个 `y`，而它们恰好都是延迟的，那么 Dask 会意识到这一点，并产生第三个延迟对象，即这两个东西相加的结果。
