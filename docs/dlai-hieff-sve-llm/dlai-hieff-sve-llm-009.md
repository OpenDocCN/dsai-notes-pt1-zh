## 从基础到生产系统 🚀

上一节我们介绍了服务栈的核心实现，本节中我们来看看如何将这些技术应用于实际生产环境。

我们在纯PyTorch环境中，仅使用CPU就完成了所有这些工作。然而，在Prebase，我们创建了一个名为**Lorx**的开源语言模型推理系统。

以下是Lorx系统的核心特点：
*   它结合了本课程所学的技术。
*   它集成了更多优化手段。
*   其目标是构建最高效的、用于生产环境部署语言模型的系统。
