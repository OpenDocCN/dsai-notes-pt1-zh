## 什么是提示工程？ 🤔

上一节我们介绍了课程背景，本节中我们来看看专家如何定义提示工程。

提示工程是一种编写优质或有效提示的技术。模型能够利用这些提示，生成一致且出色的输出。

提示工程是随着生成式AI兴起而在近两年出现的一个术语。它指的是理解如何向大型语言模型发出提示，以获得期望的回应。

## 核心提示工程技术 🛠️

理解了基本定义后，本节我们将深入探讨几种核心的提示工程技术。

以下是几种关键的提示方法：

*   **零样本提示**：这是目前人们使用ChatGPT的典型方式，即直接要求模型做某事。“样本”即例子，零样本意味着不提供任何示例，直接发出指令。例如：`“写一封商务邮件。”`
*   **单样本提示**：在指令中提供一个格式示例。例如：`“请为我撰写一份产品描述。以下是一个示例格式：[产品名称] 是一款 [核心功能] 的工具，它能帮助用户 [核心价值]。”` 模型会利用这个格式来组织信息，使输出更符合要求。
*   **少样本提示**：提供两个或更多示例。有趣的是，这些示例本身不一定正确，但格式必须正确。你甚至可以先用生成式AI根据已有格式创建更多示例，然后将它们加入提示中。
*   **思维链**：要求模型分步思考。例如：`“请解决这个数学问题。首先，制定一个计划。然后，一步一步地执行这个计划。”` 这有助于大型语言模型保持逻辑连贯性，尤其适用于复杂任务。
*   **角色扮演**：为模型指定一个角色。例如：`“你是一名JavaScript专家。”` 或 `“你是一家小型企业的CEO。”` 赋予模型越具体的角色，输出的专业性通常越强。

此外，与模型交互时还可以调整一些参数来控制其生成行为：

*   **温度**：控制输出的随机性。`temperature` 值越高，输出越多样、有创意；值越低，输出越确定、保守。
*   **Top-k / Top-p**：这些是采样参数，用于控制生成下一个词时考虑的候选词范围，从而平衡生成结果的多样性与相关性。

掌握了基本技术后，我们来看看专家们推荐使用哪些工具来实践和优化提示。

![](img/e589030875029685aa5a15a7788e8e9d_5.png)

以下是专家提到的一些常用工具和平台：

*   **Watsonx.ai Prompt Engineering Lab**：这是一个非常出色的工具，专门用于针对特定应用练习和开发有效的提示。
*   **OpenAI / Microsoft Copilot 等平台**：这些平台通常提供关于如何编写优质提示的指南和教程。你可以利用这些工具来辅助编写提示，并预览可能得到的回应类型。
*   **模型特定语法**：某些模型（如 **Llama 2**）有自己必须遵循的特定提示语法。遵循正确的语法对于获得理想输出至关重要。

## 实践建议与重要性 ⚡

最后，我们来听听专家关于实践提示工程的建议，并总结其重要性。

专家分享了一个实践案例：当需要向ChatGPT解释一个工作流程时，可以直接询问模型它偏好的输入格式。例如：`“我希望你为我分析这个工作流程。我以什么格式提供信息对你来说最有效？”` 模型会返回一个建议的结构，按照这个结构提供信息后，模型就能更准确地进行关联和分析。

提示生成对于从模型中获取正确输出至关重要。道理很简单：如果你问错了问题，就会得到错误的答案。这对人类如此，对当前的生成式AI模型尤其如此。提示工程的目标是以最少的试错、时间和资源消耗，获得我们想要的输出。

因此，提示工程在当前极具价值，相对容易学习，是一项值得投入时间掌握的关键技能。
