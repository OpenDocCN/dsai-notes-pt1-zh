# 从AI-for-Science到AI-for-Industry-p05-中国互联网协会人工智能团标启动仪式｜中国互联网协会“全国‘人工智能+’应用与技能大赛”启动仪式

在本节课中，我们将学习2025北京智源大会上的两个重要启动仪式：人工智能团体标准研制工作的启动，以及首届“AI领航杯”全国人工智能+应用与技能大赛的启动。我们将了解这些举措的目标、内容和意义。

## 团体标准研制工作启动

上一节我们介绍了会议背景，本节中我们来看看第一个启动仪式。科技创新与产业发展离不开标准的引领和规范，人工智能领域的发展同样如此。

中国互联网协会始终秉承技术向善、标准先行的理念，充分发挥行业组织枢纽作用，持续深化标准的研制与应用推广，为引领行业高质量发展贡献力量。

今天，中国互联网协会携手多家权威机构，正式启动一系列人工智能团体标准的研制工作。

以下是即将研制的核心标准系列：
*   **人工智能智能体互联互通技术要求**：此标准旨在促进智能体、大模型、以及智能体之间的模型、数据和组件系统的互联互通，加速前沿技术从科学研究向产业落地过渡。
*   **人工智能技术企业科技信用评级标准**
*   **智能体应用场景能力要求**

其中，**人工智能智能体互联技术要求**的标准目标，可以用一个核心公式来理解：
`标准目标 = 促进（智能体 + 大模型 + 数据 + 组件系统）的互联互通`
该目标与本次大会“从AI for Science到AI for Industry”的主题高度对齐。

同时，其他标准也将从企业科技信用体系、智能体的可信应用场景评估等领域，迈出标准化建设的关键步伐。

## “AI领航杯”全国人工智能+应用与技能大赛启动

在见证了团体标准启动后，我们进入第二个启动环节。接下来将启动中国互联网协会首届“AI领航杯”全国人工智能+应用与技能大赛。

本次大赛以“智融千行百业，启航无限可能”为主题，主要设立三个目标。

以下是大赛的三个核心目标：
1.  **以赛汇智**：以大赛为桥梁，汇聚企业、科研机构、高校、职业院校及行业组织的力量，构建产、学、研、用联动的生态。
2.  **以赛引才**：通过大赛选拔人工智能+领域的优秀人才与创新成果，挖掘具有潜力的创新项目。
3.  **以赛育产**：以大赛成果为依托，促进供需对接、创业项目与资本对接，助力人工智能技术在多领域应用落地，推动人工智能与传统产业深度融合。

大赛在赛道、赛制、场景、模式等方面进行了精心策划，总结出四大亮点。

以下是本次大赛的四个主要亮点：
1.  **双轮驱动，多维竞技**：设置“应用类”和“技能类”双赛道。每个赛道的项目评价均涵盖场景应用性、开拓创新性、实施可行性及行业影响力等多个维度。
2.  **多元赛制，广泛参与**：赛制包含专题赛、分站赛以及特色技能赛。面向个人、团体、企业、院校等多方主体，邀请广泛参与。
3.  **场景贯通，跨界联动**：大赛已设立近20个赛道，赛项内容丰富多彩，应用场景广泛。
4.  **开门办赛，聚势共赢**：协会将携手科研院所、企业、政府等多方主体协同发力，构建产、权、研、用一体化的赛事生态。

应用类赛道目前涵盖多个“AI+”方向。

以下是应用类赛道的主要方向（包括但不限于）：
*   AI+能源、交通、金融、教育、制造
*   AI+终端算力、具身智能、数字人、医疗健康
*   AI+IT研发运营、工业互联网、个人信息保护、医疗器械
*   AI+工业领域中小企业数字化转型

应用类赛道中还设有特色分站赛，例如“AI+应用创新创业”分站赛（为高校校友提供特别通道）和“AI+县域经济发展”分站赛（为河北省大名县量身定制）。

技能类赛道则面向职业院校、高校学生及企业技术人员，开展个人技能比拼，着力选拔兼具技术创新思维和AI实操能力的复合型人才。赛道方向包括AI+工业机器人制造、AI+工业视觉系统运维操作、AI+金融创新实践。

大赛日程安排如下：6月12日开放报名通道，7月31日截止报名。预计8月完成各赛道初赛与决赛，9月至10月举行全国总决赛。颁奖典礼将于第四季度在协会计划举办的“2025AI中国发展大会”上举行。

大赛的筹备获得了众多合作伙伴的支持，包括中国信通院、交通运输部科学研究院、中国传媒大学数字人研究院等研究机构，以及中国互联网协会人工智能、智慧教育、智慧医疗等工作委员会。

大赛将持续面向业界招募各分赛道的合作单位与赞助单位，诚邀各界积极参与，共同策划，携手共创人工智能领域的行业盛会，助推产业高质量发展。

本节课中我们一起学习了2025北京智源大会上的两项重要发布。我们了解了中国互联网协会联合多家机构启动人工智能团体标准研制工作，旨在通过标准化引领产业互联互通与高质量发展。同时，我们也详细了解了首届“AI领航杯”全国人工智能+应用与技能大赛的目标、亮点、赛道设置与赛程安排，该大赛旨在以竞赛形式汇聚智慧、选拔人才、培育产业，推动人工智能技术的广泛应用与深度融合。
