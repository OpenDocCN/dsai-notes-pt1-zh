# 具身技术与产业应用-p01-多学科交叉赋能机器人创新发展：孙立宁

在本节课中，我们将学习孙立宁教授关于多学科交叉如何推动机器人领域创新发展的宏观见解。报告从国家战略、技术发展脉络、具体应用案例等多个维度，阐述了机器人技术与人工智能、生命科学、纳米技术等学科融合所带来的巨大机遇与挑战。

## 国家战略与产业背景 🌍

上一节我们介绍了课程概述，本节中我们来看看机器人发展的宏观背景。

机器人是制造业皇冠顶端的明珠。当前，机器人产业本身是一个规模达上万亿的大市场，在生产、生活和特种领域有着重要需求。从战略维度看，国际高科技竞争激烈。美国连续10年将机器人、人工智能、传感器、3D打印和纳米材料列为五项颠覆性技术。因此，发展机器人技术非常必要。

国家层面的规划引导着产业发展。近五年，国家规划推动产业向高端产品发展，从制造业扩展到生活、安全等各个领域。在基础零部件方面需要进一步加强。技术创新则越来越向智能化方向发展，例如操作系统、仿生感知、电子皮肤、技能学习、一体化微纳操作等，这些都为今天讨论的具身智能打下了重要基础。

早在2019年，国际学术前沿已预判了机器人的发展趋势，涉及仿生、集群、导航、智能化、脑机接口、社交、医疗伦理安全等主题。机器人发展已成为多学科交叉的重要课题，甚至涉及社会学和伦理学，例如当机器人进入家庭所带来的影响。

## 多学科交叉的创新路径 🔬

上一节我们了解了宏观背景，本节中我们来看看学科交叉如何具体驱动创新。

从机械发展的总体脉络看，目标是实现高性能、高速、高精度、轻量化，并走向智能化和网络化，实现无处不在的应用。与不同学科的交叉融合催生了大量创新。

以下是几个关键的交叉领域及其成果：

*   **与纳米技术结合**：产生了**纳米机器人**。
*   **与生命科学结合**：服务于医疗健康，并利用生命科学原理发展机器人，即**仿生机器人**。

这种大交叉不仅产生了技术创新，也催生了众多新产品。例如，医用机器人是一个重要且持续热门的赛道。国际上已将医疗机器人列为重要的学科和产业方向。

## 典型案例：手术机器人 🤖

上一节我们介绍了交叉创新的方向，本节中我们通过一个具体案例来深入理解。

医用机器人涵盖手术、影像、康复、医院数字化（如配药、导诊护士）等方面，具有巨大的产业前景。

以哈工大的康多机器人为例，通过十年孵化，成功实现了国产替代。其典型技术是一个具有四个自由度的手术钳，完成了超过8000例临床手术，并从大学成果转化为产品和企业，实现了全球销售。

未来手术机器人的优势在于远程手术。例如，北京、上海的医生通过5G网络，为海南医院的病人成功完成了远程手术，这展示了技术创新的价值。

创新的特点是**高端化、高投入、高风险，最终带来高利润**。该企业的估值达到8亿元，这是一个典型的成功案例。

## 技术发展脉络与前沿探索 ⚙️

上一节我们看到了成功的应用，本节中我们探讨技术本身的发展与前沿探索。

技术创新需要持续深化。例如，达芬奇手术机器人最初没有力觉反馈，医生操作时因缺乏手感而受到安全限制。2018年，团队将尺寸为2mm x 2mm的MEMS力传感器集成到3mm x 4mm的微小夹钳区域，实现了多维力感知，走在了国际前列，使未来的手术机器人具备感觉。

第二个方向是**自主化与智能化**。具身智能的核心在于将人类经验数字化，这类似于大模型的概念。例如，将老医生数十年的手术经验数字化并迁移到机器人中，让机器人具备人类的智能和技能。通过AI方法，可以用资深医生的经验指导年轻医生或机器人进行手术。其路径是：**采集医生手术经验大数据 -> 将经验转化为机器人可执行的语言 -> 通过迭代学习优化手术方案**。目前已在立体缝合等任务上进行了初步研究。

## 脑机接口与仿生感知 🧠

上一节我们讨论了机器人的自主智能，本节中我们看看更前沿的人机交互与仿生技术。

脑机接口是一个不亚于人形机器人的重要领域。其目标是从大脑中提取意识信息，用于控制机器人或分析情绪、情感等。

团队开发了一项创新技术：利用近红外光谱信号照射脑部血管，通过血氧含量的变化来反映脑区的意识活动。结合人工智能算法，能够解读意识意图。这项技术实现了在操作指令产生前约0.5秒的预测，平均识别率达到96%。此外，在步态识别等任务上，与标准步态的吻合率达到了84%，为脑机接口提供了新的模式。

仿生机器人是具身智能的重要体现。其发展可分为几个维度：
1.  **结构仿生**：如早期的大狗机器人。
2.  **功能仿生**：如机械假手，完成特定功能。
3.  **机理仿生**：源于生物界原理，如鲨鱼皮的特殊结构。

当前人工智能多基于数据、数学模型和人工网络。而感知仿生应更多源于人类真实的生理学原理和脑科学机制（人脑拥有约39万亿个细胞），这为学术创新提供了广阔空间。

## 核心技术与感知创新 ✨

上一节我们探讨了仿生理念，本节中我们聚焦实现这些理念所需的核心技术与感知创新。

实现先进的机器人系统需要五大关键技术：**仿生轻量化设计、高能量密度关节、全身动态运动控制、环境感知、以及大模型驱动的操作**。

感知变得尤为重要。人类拥有“五感”，机器人要完成复杂动作，也需要先进的感知能力并与AI结合。

团队自2018年开始研究柔性电子皮肤。利用纳米摩擦发电材料，在运动时通过摩擦产生电信号，使机器人具备防撞和类人触觉。目前触觉感知已发展到十指。

机器人的手部触觉是关键。团队利用微柱阵列绑定在皮肤上，当微柱变形时产生电信号，从而实现电子皮肤、滑觉、触觉感知。未来，灵巧手将能像人手一样感受外部环境。此外，还开发了U型传感器，可测量手部姿态和交互力。这些都需要与MEMS技术、电子技术紧密结合。

有了传感器，才能开发训练设备和数据集，用于下一代操作技能的学习，例如如何测量和适应装配力，这就需要多维传感器。

## 从仿生原理到控制范式革新 🔄

上一节我们介绍了感知技术，本节中我们看看如何从生物原理中革新控制范式。

将人类行为数字化并训练大模型是当前主流路径：**先采集行为大数据 -> 学习大模型 -> 部署到机器人**。

团队思考能否颠覆这一范式，从人类和生物界的本源寻找答案。人类智能的基础是神经元细胞网络。其生理基础是：神经元细胞膜上的离子通道阀门周期性开合，导致离子流动，产生脉动的电信号，这符合波动理论。团队用机械理论解释了这一信号传输模型，并将其固化为芯片。

验证表明，用这种基于真实神经生理的编码模型连接断肢信号，吻合率达到98%。这不同于训练出来的模型。基于此，再进行训练和迭代，神经网络就有可能像人一样自发涌现智能。其应用包括神经修复和开发新一代仿生控制器（将此类芯片作为控制核心）。

后续研究旨在将AI技术与这种神经元技术结合，构建类似大脑海马体的系统，实现**仿生感知、快速学习和仿生控制**。

## 创新应用与未来展望 🌟

上一节我们探讨了原理性的革新，本节中我们看看这些创新带来的具体应用并展望未来。

基于上述原理，团队实现了网络的自发演化与知识迁移。例如，训练机器狗在复杂地形（如坡道、障碍物）中自动生成安全的脚部放置角度和最优行进路径。在轻量化神经网络构建地图、快速学习不同路面（雪地、砂石、楼梯）方面也表现出色。这种网络源于生物学原理，而非传统的深度学习。

另一个有趣的应用是异构机器人协作。在模拟核电站的真实场景中，大小不同的机器人（轮式、足式）协作完成任务：小机器人先开门，大机器人进入；下一个场景则反之。这解决了当前机器人协同作业的问题。系统还能融合听觉与视觉，根据声音方向定位故障点。这为具身智能的发展开辟了新思路。

纳米机器人是另一个重要方向。精准医疗需要从宏观病例观察到细胞乃至基因层面，机器人是实现这一目标的手段。团队利用原子力显微镜（分辨率0.1纳米）和14自由度的机械手，实现了对碳纳米管、石墨烯等纳米材料的原位三维操作，这是全球精度最高的机器人系统之一。

展望未来，纳米机器人可进入血管工作。团队探索用液态金属（常温下为液态）在外部磁场控制下作为驱动。进一步模拟“机器人进入血管”的场景，目标是用类似蝌蚪原理的微型机器人，在图像导航下，定点清除血栓或释放药物。

另一个设想是针对昂贵的CAR-T细胞免疫疗法（一次约8万美元）。方案是：用液态金属微球包裹患者自身的红细胞膜（解决生物兼容性），内部装载扩增后的T细胞。这些“微型机器人”进入血管后，可定向靶向肿瘤，并通过血管通道钻入肿瘤内部进行治疗。这项技术涉及纳米制备、生物组织成像、微血管动力学、导航与控制，如果突破，将是爆发性的创新。
