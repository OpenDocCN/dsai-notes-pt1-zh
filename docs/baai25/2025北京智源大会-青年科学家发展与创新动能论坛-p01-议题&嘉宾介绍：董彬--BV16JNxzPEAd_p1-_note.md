# 青年科学家发展与创新动能论坛-p01-议题&嘉宾介绍：董彬

在本节课中，我们将要学习2025年北京智源大会“青年科学家发展与创新动能论坛”的议题设置与嘉宾阵容。我们将了解论坛的背景、核心议程以及参与分享的杰出青年科学家们。

在人工智能浪潮奔涌向前的时代，青年科学家已然扛起了创新大旗，以蓬勃的创造力不断拓展技术边界与应用版图。

今天我们齐聚2025智源大会“青年科学家发展与创新动能”这一特色论坛。这里星光璀璨，汇聚了9位来自学界和业界的精英力量。

嘉宾阵容中既有曾任职于DeepMind、OpenAI的资深研究员，也有来自国内顶尖学府以及像绅世科技、幻新琼等企业的青年才俊。他们将带来精彩的主旨演讲。

嘉宾们将分享自己的学术成长经历。最后我们还有一个专题讨论环节，将围绕“跨越边界，构建创新生态”展开深度探讨，共同为青年学者成长与科研生态建设建言献策。
