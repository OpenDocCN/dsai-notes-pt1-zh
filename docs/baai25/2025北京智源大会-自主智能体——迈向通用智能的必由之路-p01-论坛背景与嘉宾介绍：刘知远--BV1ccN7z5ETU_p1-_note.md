# 自主智能体——迈向通用智能的必由之路-p01-论坛背景与嘉宾介绍：刘知远

在本节课中，我们将了解2025年北京智源大会“自主智能体”专题论坛的背景信息，并认识本次论坛的主持人及首位演讲嘉宾。

尊敬的各位观众，大家下午好。我是来自清华大学计算机系的老师刘知远。非常荣幸由我来主持今天关于自主智能体的专题论坛。

我本身曾担任智源研究院的青年科学家，并且几乎每年都会参与智源大会。今年智源大会已举办至第七届，令人感慨。在这一轮大模型浪潮到来之前，智源大会就已经成为国内乃至国际人工智能领域的一次技术峰会。每年它都能吸引众多观众和讲者，共同分享人工智能最前沿的技术。

到今年，很多人称2025年为“智能体元年”。我们今天的论坛，正是希望与大家一同分享大模型驱动的自主智能体最前沿的相关技术。

我们今天的讲者，均来自国内外在最前沿开展智能体研究的青年学者。相关话题基本覆盖了自主智能体非常前沿的各个方向。

以下是本次论坛将涵盖的核心议题：

*   **领域知识增强**：如何让智能体具备相关领域的实施性知识（RAG技术）。
*   **长序列处理**：如何处理近乎无限长的序列的技术。
*   **智能与自主性提升**：如何让智能体变得更加智能和自主。
*   **群体智能**：如何让多个智能体进行更好的协作。
*   **前沿应用**：我们也邀请了讲者分享自主智能体在教育等领域的最前沿应用。

以上是论坛内容的简要介绍。接下来，我将时间交给今天的几位讲者。

我们今天的第一位讲者，在之前的宣传中是一位神秘嘉宾。现在我们来揭晓他的身份。

他是来自普林斯顿大学的教授，著名的人工智能与大模型领域专家——**王梦迪**老师。他将为大家分享他及其团队在智能体方向上的最新探索与思考。

接下来，我们将连线王老师，请他为大家进行最新成果的分享。大家欢迎。

本节课中，我们一起了解了本次自主智能体论坛召开的背景、意义及其涵盖的前沿技术方向，并认识了主持人刘知远老师与首位演讲嘉宾王梦迪教授。
