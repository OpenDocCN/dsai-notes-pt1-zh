# 具身智能与人形机器人-p01-论坛背景与嘉宾介绍：王鹤

在本节课中，我们将要学习2025年北京智源大会“具身智能与人形机器人”分论坛的背景信息、发展历程以及本次论坛的嘉宾阵容与议程安排。

## 论坛背景与发展历程

上一节我们介绍了课程主题，本节中我们来看看本次论坛的背景与演变过程。

我是本次论坛的主席王鹤。我是北京大学的助理教授、智源学者，同时也是北京银河通用机器人的创始人和CTO。

今年是第七次举办智源大会，也是我们第三届具身智能论坛。具身智能论坛从2022年第一届开始，在2023年是一个半天的论坛。到去年，论坛升级为一个全天的论坛。

再到今年，我们将人形机器人加入了主题当中，形成了“具身智能与人形机器人”这样一个分论坛。这体现了过去三年时间里，具身智能和人形机器人领域的快速进展和变化。

## 论坛议程安排

上一节我们介绍了与会嘉宾，本节中我们来看看论坛的具体流程安排。

今天我们的论坛将在上午以5个报告开始。中午12点钟到下午1点半为午休时间。

下午我们会继续有5个精彩的报告，然后进行一个圆桌讨论环节。

最后，我们的论坛结束也将标志着此次智源大会的结束。智源大会理事长黄铁军教授将来到论坛进行大会闭幕致辞。

这就是我们今天论坛的整体流程。
