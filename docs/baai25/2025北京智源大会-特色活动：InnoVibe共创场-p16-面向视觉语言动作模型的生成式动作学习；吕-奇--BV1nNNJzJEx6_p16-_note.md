## 1. 引言：VLA模型的兴起与核心问题 🤖

近年来，具身智能与视觉语言模型成为研究热点。从2024年至今，相关技术发展迅速。例如，Figure公司发布的Figure 02模型已能完成多种复杂任务。

作为研究者，我们需要思考VLA模型究竟是什么，以及如何训练一个好的VLA模型。纵观现有模型，其本质仍是基于序列建模的方法，学习从视觉和语言到动作的映射过程。

对于文本和图像，已有成熟的表征方案（如词嵌入、图像块嵌入）。然而，对于动作的表征，目前尚无明确方案。因此，我们首先需要思考：动作应该采用什么样的表征？

## 2. 动作的表征探索 🔍

![](img/c0c4b383dc3703c0769c033c8b635eeb_6.png)

上一节我们提出了动作表征的问题，本节中我们来看看动作本身具有哪些属性，以及如何借鉴其他模态的表征方法。

![](img/c0c4b383dc3703c0769c033c8b635eeb_8.png)

![](img/c0c4b383dc3703c0769c033c8b635eeb_9.png)

文本和图像各有其属性。文本是离散的、抽象的，具有因果连续关系。图像是连续的表征，更为具体。那么动作呢？我们总结出动作包含以下关键属性：
*   **连续性**：动作本身是连续信号。
*   **因果性**：完成上一个动作是执行下一个动作的前提。
*   **异构性**：不同机器人的动作空间各不相同。

动作的“异构性”与语言中的“多语言性”高度相似。因此，我们的思路是：从文本和图像模态中，借鉴它们对上述属性的建模方式，并融合其优势，为动作设计一个好的表征。

![](img/c0c4b383dc3703c0769c033c8b635eeb_11.png)

![](img/c0c4b383dc3703c0769c033c8b635eeb_12.png)

## 3. 现有方法：两阶段训练框架及其挑战 ⚙️

![](img/c0c4b383dc3703c0769c033c8b635eeb_16.png)

基于上述思路，我们探索了从离散化表征出发的路径。这包含两个步骤，构成了一个两阶段训练框架：

![](img/c0c4b383dc3703c0769c033c8b635eeb_17.png)

![](img/c0c4b383dc3703c0769c033c8b635eeb_18.png)

![](img/c0c4b383dc3703c0769c033c8b635eeb_19.png)

1.  **动作分词化**：类似于大模型中对语言和图像进行分词，将连续动作转换为离散的标记序列。
2.  **训练生成模型**：训练一个模型来预测分词化后的标记索引。

![](img/c0c4b383dc3703c0769c033c8b635eeb_20.png)

![](img/c0c4b383dc3703c0769c033c8b635eeb_21.png)

![](img/c0c4b383dc3703c0769c033c8b635eeb_22.png)

以下是该框架的示意图：

![](img/c0c4b383dc3703c0769c033c8b635eeb_23.png)

![](img/c0c4b383dc3703c0769c033c8b635eeb_24.png)

```
[连续动作] -> [编码器] -> [量化器/码本] -> [标记索引] -> [解码器] -> [重建动作]
         (阶段一：动作分词化)                              (阶段二：生成模型预测)
```

然而，现有方法存在显著问题：
*   **阶段一（分词化）**：采用类似VQ-VAE的量化方法，存在严重的**码本坍缩**问题，即模型只频繁使用码本中的少数向量，导致表征能力受限。
*   **阶段二（生成模型）**：现有方法忽略了动作间的**因果连续关系**。同时，模型预测的标记索引与真实动作间存在偏差，一旦遇到分布外情况，机器人可能无法做出有效动作。

![](img/c0c4b383dc3703c0769c033c8b635eeb_26.png)

![](img/c0c4b383dc3703c0769c033c8b635eeb_27.png)

## 4. 我们的改进方案 🛠️

针对上述挑战，我们提出了相应的改进措施。

**针对阶段一的码本坍缩问题**：我们改进了训练目标，引入了**旋转向量**等技术，以缩小原始向量与量化后向量之间的误差，从而缓解码本利用不均衡的问题。

**针对阶段二的建模与泛化问题**：
1.  我们将语言模型的**因果自回归建模能力**引入生成模型，以更好地捕捉动作序列的连续因果关系。
2.  对于分布外泛化问题，我们增加了**动作细化**步骤，对预测出的动作进行优化调整，提升其准确性和鲁棒性。

![](img/c0c4b383dc3703c0769c033c8b635eeb_29.png)

![](img/c0c4b383dc3703c0769c033c8b635eeb_30.png)

这些改进在下游任务中取得了显著效果。例如，在多项任务性能上，我们的方法超越了当时的先进技术。在码本利用率上，基线方法（深色线）的码本使用严重不均，而我们的方法（浅色线）能更均匀、充分地利用整个码本，从而让生成模型能更好地表征多样化的动作。

## 5. VLA模型的演进与未来方向 🚀

在探索了动作表征之后，我们来看看VLA模型本身的训练范式演进。我们当前的工作旨在回应“下一代VLA模型应该是什么样子”这一问题。

现有的方法大致可分为三类：
1.  **传统策略模型**：以当前观测图像为输入，直接输出动作。它缺乏对语言指令的理解能力，控制能力有限。
2.  **VLM扩展模型**：以视觉语言模型为基础，额外训练一个动作头。它引入了指令理解能力，并通过多任务学习提升了在具体任务上的成功率，但模型参数量大。
3.  **端到端视觉预测策略**：这是我们团队此前提出的新范式。其核心思想是**预测逆动力学模型**——让模型先预测执行动作后的未来状态，再根据这个预测的未来状态来指导当前应执行的动作。

![](img/c0c4b383dc3703c0769c033c8b635eeb_38.png)

**PIDM范式**让模型在长周期任务和鲁棒性方面取得了显著提升。例如，在面对灯光扰动、背景干扰或需完成多种挑战性任务时，模型都表现出了强大的性能。

既然PIDM路线被验证有效，我们自然迈入下一阶段：**为何不将这种能力融入VLA模型的训练中？** 因此，我们当前的工作正在探索第四条路径：构建一个以**理解专家**、**生成专家**和**动作专家**为核心的认知模型，来学习更强大的VLA模型。这项工作仍在进行中，预计不久后将发布。

## 6. 总结 📝

本节课中我们一起学习了面向视觉语言动作模型的生成式动作学习。
*   我们首先探讨了**动作表征**的独特性，并提出了一个改进的两阶段离散化表征框架，以解决码本坍缩和因果建模问题。
*   接着，我们回顾了VLA模型的几种训练范式，并重点介绍了一种基于**未来状态预测**的新思路，它通过让模型“想象”动作后果来提升决策质量。
*   最后，我们展望了未来方向，即融合理解、生成与动作专家，构建更强大的认知模型。

## 附录：问答精选 💬

**问：基于2D图像的未来预测方法，在更换物体或场景时是否会有局限性？**

答：瓶颈不在于2D或3D，而在于模型中的理解与生成专家能否真正理解物体在真实世界中的运动规律。任何物体都有其运动规律（如球滚动、方块滑动）。如果基础模型能建模这种固有规律，则2D/3D不是主要限制。3D的优势在于能提供深度等空间信息，但并非泛化能力的根本限制。

**问：与现有VLA框架相比，这种“预测未来”的方法在提升泛化性方面的优势从何而来？**

答：现有VLA模型要想泛化性好，需要海量的真实机器人数据或高保真仿真数据，成本极高。我们的思路是将泛化能力的负担从**动作专家**转移到**理解与生成专家**上。后者可以利用已有的、基于海量数据训练的基础模型（如视频生成模型），它们已具备强大的泛化能力。动作专家只需根据生成准确的未来图像去执行动作，从而实现有目的的、泛化性更强的控制。

**问：动作分词化的具体做法是什么？**

答：类似于图像生成中VQ-VAE的做法，对动作进行量化压缩，将其映射到一个码本中的离散标记。我们尝试过VQ-VAE、LFQ、FSQ等多种量化方法。
