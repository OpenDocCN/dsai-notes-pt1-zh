# AI+理工&医学-p15-圆桌讨论：AI如何赋能理工&医：张恒贵、吴立波、李建平、张路霞、王宽全、倪海波、欧阳德方

在本节课中，我们将一起探讨人工智能（AI）与理工、医学领域协同创新所面临的关键瓶颈与未来突破方向。本次内容整理自一场专家圆桌讨论，我们将聚焦于智慧医学的实现、数据与技术壁垒、跨学科人才培养等核心议题。

## 1：临床实践中的AI价值与落地瓶颈

上一节我们介绍了本次讨论的背景，本节中我们来看看临床专家们对AI工具在实际应用中的真实感受与遇到的挑战。

**问题：** 在日常临床实践中，是否真实感受到AI工具的协助价值？有哪些限制使其难以落地？

以下是三位临床专家的观点。

**李建平教授（心血管临床）：**
*   **临床需求巨大：** 目前临床存在大量未解决的“真问题”，例如对急性心肌梗死缺乏有效的预测方法。
*   **核心瓶颈与AI机遇：** 关键在于如何整合多模态、多尺度的临床数据（如影像、心电图、穿戴设备数据等），并通过AI算法建立它们之间的映射关系，最终实现疾病预测。公式可表示为：**疾病风险预测 = f(多模态临床数据映射)**。
*   **更广阔的应用场景：** AI在医疗器械评测（如心脏瓣膜在真实人体环境中的力学模拟）、新药研发（利用数字孪生人体或器官芯片替代部分临床试验）、以及应对临床试验中的“职业受试者”问题等方面，都有迫切需求和巨大潜力。
*   **人才培养是关键：** 需要真正专业的、从本科早期阶段入手的“AI医学家”交叉人才培养，而非简单的工具使用者。

**何以华教授（研发者视角）：**
*   **研发初衷与价值：** 研发AI的初衷在于实现医疗服务的“均质化增效”，即将专家知识下沉至基层，提升整体诊疗能力。
*   **科研驱动：** AI有助于从多维度、多组学数据中挖掘更接近疾病本质的规律，解决行业痛点。

**张路霞教授（肾脏内科/医学教育）：**
*   **克服学科割裂感：** 跨学科研究存在强烈的“割裂感”，推动融合需要双方共同努力。
*   **人才培养实践：** 关键在于系统性的人才培养。北京大学医学部已采取两项措施：
    1.  为医学生开设重构的、贴合医学场景的**AI通识必修课**。
    2.  为AI科学家开设**医学通识课**，促进相互理解。
*   **核心观点：** 临床医生需将问题凝练为科学问题并学会与算法专家沟通；算法专家需努力理解医学问题与应用场景。双方“各迈一步”，才能在中间相遇，推动领域良性发展。

## 2：AI与医疗结合的数据与技术壁垒

上一节我们聆听了临床专家对AI落地难点的分析，本节中我们转向技术专家，看看从研究和算法角度，最大的障碍在哪里。

**问题：** 从科学研究角度，AI与医疗结合最困难的数据技术壁垒是什么？

**王宽全教授（人工智能/算法）：**
*   **数据获取难题：** 医院数据涉及病人隐私，且医院出于利益考量，缺乏共享数据的动力和机制。部分数据受设备厂商限制，无法导出。
*   **数据标注成本高：** 高质量的监督学习依赖医生标注，但医生时间宝贵，标注成本极高。自动标注的质量则难以保证。
*   **数据异构性：** 医疗数据多模态、非结构化，且散落在不同科室，整合难度大。
*   **未来方向：** 除了大数据驱动，从医生经验、医患交互中学习的“经验学习”可能是补充途径。机构间的战略合作（如高校与医院）是解决这些问题的良好开端。

## 3：生命系统建模中AI与传统方法的融合

在讨论了通用数据挑战后，本节我们聚焦于一个具体领域：生命系统建模。我们来看看AI与传统建模方法在此处的接口现状与障碍。

**问题：** 在生命系统建模中，AI与传统方法的接口是否成熟？最大障碍在哪？

以下是两位生物医学工程专家的见解。

**欧阳德方教授（计算药学）：**
*   **接口尚未成熟：** 生命系统从分子、细胞、组织到人体，跨尺度建模方法各异，AI与这些传统方法的融合刚刚开始，远未成熟。
*   **制药数据孤岛：** 药企数据同样分散且涉及商业利益，难以共享，单一药厂的数据不足以构建优质模型。
*   **人才培养是核心关键：** 传统药学教育以实验为主，缺乏计算训练；计算机学生则不懂药学。差距巨大，必须从本科课程体系改革入手，培养复合型人才。

**倪海波教授（心脏数字孪生）：**
*   **结合已有探索但远未成熟：** 在构建数字孪生（如心脏）的流程中，AI（如分割算法）已用于辅助生成解剖结构，形成了初步的研究管线，但成熟度不足。
*   **新范式带来新机遇：** 在传统深度学习不适用的动态系统（如心脏动力学）中，大语言模型等新AI范式与传统基于物理的模型相结合，是一个崭新且有益的方向。
*   **打破专业标签：** 在交叉学科研究中，最应忘记“专业”这个自我设置的标签。应鼓励学生和研究者突破学科边界，这种精神的传递至关重要。

## 4：对年轻科研工作者的寄语与总结

最后，我们聆听资深专家对年轻一代科研工作者的寄语，并对本次讨论的核心观点进行总结。

**李建平教授对年轻科研工作者的寄语：**
*   **聚焦真问题：** 坚持“研究真问题，真研究问题，真解决问题”。
*   **深入临床需求：** AI与医疗的结合，需要双方坐在一起，共同梳理出临床最急需、威胁人民健康的重大疾病领域的卡脖子问题。未来在此大有作为。

**总结：**
本节课中我们一起学习了AI赋能理工与医学领域所面临的多维度挑战：
1.  **临床落地瓶颈** 在于需求转化、数据映射及复杂场景应用。
2.  **数据与技术壁垒** 体现在获取难、标注贵、整合复杂。
3.  **跨尺度建模融合** 仍处于早期，需方法学创新。
4.  **最核心的突破点** 在于**跨学科人才培养**与**合作模式创新**。双方需要打破专业壁垒，相互拥抱，临床专家向前凝练问题，技术专家向前理解场景，通过机制化合作，共同攻克关乎生命健康的真问题。
