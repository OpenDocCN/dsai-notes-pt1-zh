# AI安全-p01-开幕致辞：张宏江

在本节课中，我们将学习张宏江博士在2025北京智源大会AI安全论坛上的开幕致辞。我们将了解AI安全为何成为人工智能发展的核心议题，以及国际社会在此领域的合作与共识。

---

非常欢迎大家再次来到智源大会，尤其是欢迎大家来到AI安全论坛。我很高兴能够再次担任本次安全论坛的主席。如果大家关注我过去两年的研究，可以看到自GPT发布以来，我参与最多、讨论时间最长的学术活动就是AI安全。

自从2023年10月参加了由刘晶在剑桥主持的AI安全闭门讨论会后，我们看到了一个清晰的趋势：随着大模型能力的快速发展，以及人们对AI应用的日益深入，AI安全问题变得至关重要。因此，我非常高兴并欢迎大家参加今天的讨论。

在昨天的大会开幕式上，班九教授做了开幕演讲。他着重强调了避免来自失控AI智能体的灾难性风险，并指出了AI安全在AGI发展过程中的重要性。

事实上，当我们越来越深入地审视AI安全问题时，就会越来越深刻地认识到，它实际上是AI未来发展和应用的前提。如果我们无法妥善解决这个问题，它可能为人类带来灾难性后果。这种后果远超AI 1.0时代我们所认知的深度伪造、偏见或虚假信息风险。

作为智源的创始人及前理事长，智源自成立以来就一直高度关注人工智能的伦理，从最初的伦理探讨到治理，如今我们进一步强调AI安全技术的发展。在这方面，我们也积极推动国际合作。

在2023年的智源大会上，我们邀请到了杰弗里·辛顿和萨姆·奥尔特曼，就AI对齐与安全进行了深入讨论。我印象非常深刻的一点是，辛顿教授当时提到，当硅基智能的学习能力超越碳基智能时，我们应如何判断未来。他提出了一系列见解。奥尔特曼也在会上就大模型的安全问题发表了他的观点。

在一年多前，即2024年3月，我们召集了当初在英国牛津参加闭门会议的所有专家，在北京进行了为期两天的会议。在那次会议上，我们达成了良好的《北京AI安全国际共识》，首次由这批专家共同划定了AI安全的66条红线。

这些红线对于我们未来的研究，尤其是AI技术研究至关重要。过去一谈到AI安全，我们常将其视为政策性问题。但今天我们看到，这不仅是政策层面的研究，更重要的是技术层面的研究。例如，大模型的对齐问题、越狱问题，以及欺骗人类的问题。今天，我们已经在推理模型中观察到其试图避开人类控制的偶发现象。幸运的是，目前这仍是偶发现象。但当这种现象演变为系统性危险时，人类将面临巨大风险。

因此，我非常高兴我们今天能继续在AI安全论坛上，就刚才提到的三个方面进行更深入的讨论。我更希望，通过这次讨论，我们能推动学术界——我强调的是学术界，而不仅仅是政策层面——真正开始深入研究这些问题。我认为班九教授本人树立了一个极好的榜样。

作为一名在人工智能领域耕耘40多年的学者，他是深度学习的早期提出者、倡导者，也是最早开源深度学习算法的学者之一。在ChatGPT发布后，他看到了大模型能力带来的潜在风险——这种风险远超我们过去在AI 1.0时代所面临的风险。因此，他放下了原有的研究，从技术角度探讨AI模型可能产生何种风险、我们如何发现这些风险，以及如何监督AI系统，以避免出现最可怕的系统性风险。这种风险可能超越人类的可控能力。我认为，班九教授对此的认识非常深刻。

正是基于这种信念以及对技术发展的认识，在过去两年的一系列国际会议、论坛，尤其是闭门技术讨论会（包括我们在牛津、北京、威尼斯以及最近在新加坡的会议）中，我们都能看到他深刻的思考。昨天在开幕大会上，他也与大家分享了一些见解。

在此，我想呼吁各位来宾，这是一个很好的研究课题，也是一个很好的技术研究课题。希望通过今天的讨论，我们能够认识到这一点，并找到一些好的研究题目，从而推动国内在AI安全技术研究方面的投入，获得更多公众和政策决策者的关注，将这方面的研究进一步向前推进。

同时，基于我们自己的研究、对问题的理解以及对解决技术的认知与改进，我们能够推动国际合作。在当今全球地缘政治如此复杂的世界中，我们更需要找到人类面临的共同危机。我想，人工智能或通用人工智能可能带来的未来生存危机，恰恰是我们大家可以共同努力、进行合作的为数不多的领域之一。

我也希望通过今天的讨论，能让更多同事、更多学术界的研究人员，以及更多国际学术界的同事，共同投入到这一领域。在此，我也想感谢天天和班九等人在幕后付出的努力，让我们今天得以举办这场AI安全论坛。接下来，我们还将有非常精彩的发言，并期待非常热烈的讨论。

谢谢大家，再次感谢各位的参与。

本节课中，我们一起学习了张宏江博士关于AI安全重要性的阐述。我们了解到，随着大模型能力的飞速发展，**AI安全已成为AI未来应用和发展的前提**，其风险远超AI 1.0时代的认知。国际社会已通过《北京AI安全国际共识》划定了66条红线，强调了从**技术层面**深入研究大模型对齐、越狱等问题的紧迫性。最后，课程呼吁学术界加强投入与国际合作，共同应对这一人类可能面临的共同危机。
