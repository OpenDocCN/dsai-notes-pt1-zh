# 类脑大模型-p01-主持人论坛背景介绍：马征宇

在本节课中，我们将学习类脑大模型论坛的背景、当前大模型面临的挑战，以及借鉴大脑机制发展类脑智能的潜在路径。

随着大模型的迅速发展，它不仅受到了学术界和工业界的广泛关注，也深刻影响着我们的日常生活。

然而，当前的大模型规模已达到**万亿参数**级别。其惊人的计算成本与数据需求，构成了进一步发展的主要门槛。

上一节我们提到了传统大模型的局限性，本节中我们来看看另一个参照系——人脑。人脑包含近千亿个神经元，其连接与参数的数量级更为庞大。它能实现复杂的认知、推理与控制等一系列能力，但功耗却极低，仅约**20瓦**。

因此，借鉴大脑的动力学原理、结构组织等机制来发展类脑大模型，被认为是实现高级人工智能（AGI）的一条潜在合理路径。

那么，究竟该如何实现类脑智能呢？这涉及一个高度交叉的学科领域。

以下是实现类脑智能所涉及的主要交叉学科与问题领域：
*   **机器学习**
*   **神经科学**
*   **数学与物理**
*   同时，它也涵盖从**算法**、**软件**到**硬件**与**计算架构**等一系列系统工程问题。

鉴于问题的复杂性，本次论坛虽无法给出确切答案，但旨在汇聚前沿思想进行深入探讨。

我们非常有幸邀请到五位在国内外从事前沿研究的专家进行主题报告，并在之后设置圆桌讨论环节，以共同探讨核心问题与发展方向。

首先，第一位报告嘉宾是来自中国科学院自动化研究所的李国奇研究员。

以下是李国奇研究员的主要简介：
*   **职务**：中国科学院自动化所研究员；脑认知与类脑智能全国重点实验室副主任；通用类脑智能大模型北京市重点实验室主任。
*   **荣誉**：国家杰出青年基金获得者；入选北京市杰青、北京智能青年科学家、中国科学院百人计划等。
*   **成果**：在《Nature》子刊等顶级期刊及人工智能顶会上发表论文200余篇；主持科研项目30余项。

本节课中，我们一起学习了举办类脑大模型论坛的背景与动因。我们认识到，在传统大模型面临算力与能效挑战的当下，借鉴高效、智能的人脑机制，为发展下一代人工智能提供了新的思路。这是一个需要多学科深度交叉融合的广阔领域。接下来，李国奇研究员将为我们带来关于“通用类脑智能大模型”的精彩报告。
