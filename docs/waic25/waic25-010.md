在本节课中，我们将要学习2025年世界人工智能大会主论坛上关于“AI重塑企业”的专题讨论。本次论坛汇集了来自政府、全球领先咨询公司及多家行业标杆企业的领导者，共同探讨人工智能如何驱动企业智能化转型，分享实践经验与前沿洞察。



## AI重塑企业：2：开幕致辞与政策引领

上一节我们介绍了论坛的概况，本节中我们来看看开幕致辞如何为AI重塑企业定下基调。

上海市人民政府副秘书长、市经济和信息化委员会主任张英女士为论坛致开幕辞。

她指出，以大模型为代表的生成式人工智能正加速迭代，推动AI从效率工具向经济发展的核心引擎跃升。据预测，未来10年AI将对全球GDP的贡献超过10%。

上海将人工智能作为重点发展的先导产业之一，并围绕其加速创新发展：
*   **持续筑牢智能底座**：布局“1+3”开源和基础大模型体系，做强智能算力统筹调度服务平台。
*   **持续打造智能方案**：加快培育数据服务商、模型服务商及智能系统方案解决商，打造具身智能、自动驾驶等数字孪生实训场。
*   **持续优化智能生态**：打造高能级产业载体，设立产业转型升级基金，依托创智学院加快培育复合型AI人才。

面向未来，上海将深入实施“AI+”行动，通过算力券、模型券、语料券等支持企业加大智能化转型力度，并支持像埃森哲这样的全球专业服务公司深度参与上海人工智能高地建设。



## AI重塑企业：3：企业视角下的AI转型战略

从张秘书长的致辞中，我们可以看到政策与产业协同的重要性。而要实现企业智能化转型的目标，拥有全球化的创新视野和实践同样必不可少。这正是埃森哲所专注的领域。

埃森哲大中华区主席朱虹女士在致辞中分享了企业的核心观察。她指出，在充满变数的市场环境中，AI应成为企业应对挑战、驱动重塑增长的关键武器。

埃森哲始终相信技术的最终价值是赋能于人类。过去两年，埃森哲通过30亿美元专项基金，在全球打造了6000多个AI转型项目，并培养了75000多名数据和AI专业人才。在中国，也已投资开发了70多个针对本土场景的资产。

本次论坛的一个重要环节是发布**埃森哲第八次中国企业数字化转型指数报告**。该报告长期系统性跟踪中国企业的数字化转型进程，近年来的报告再次验证，**AI已成为企业重塑的最关键因素之一**。



## AI重塑企业：4：规模化AI转型的四大要务

上一节我们了解了企业拥抱AI的战略视角，本节中我们来看看如何将AI潜力转化为规模化价值。埃森哲亚太区联合首席执行官关户亮司先生分享了来自亚太地区的洞察与实践。

他指出，AI正从试点项目转变为新的增长操作系统。一个典型案例是日本**明治安田生命保险公司**。面对老龄化社会带来的业务萎缩和劳动力减少，该公司启动了为期五年的企业级AI转型：
*   **CEO亲自赞助**，并任命了单一的**AI转型官**负全责。
*   为超过3万名销售人员部署**个人数字助理**，将生产力提升了2到4倍。
*   重新构想劳动力，将员工重新培训并部署到新的数字化岗位或增长型业务中。
*   同步进行**数字化核心系统现代化**，将主机系统迁移至云端，为AI提供灵活的数据和功能基础。

基于此类案例，埃森哲总结了规模化AI影响的四大要务：

以下是实现规模化AI转型必须关注的四个关键领域：

1.  **重新构想流程**：AI引领的转型不是自动化现有流程，而是**从零开始设计全新的、自适应的流程**，让人类与AI智能体协作以实现特定业务成果。公式可表示为：`新流程 = 人类 + AI智能体协作 → 业务成果`。
2.  **重塑人才与组织**：劳动力转型不仅是利用AI替代现有岗位，更是要**设计新的人才模型和工作方式**，释放人类潜能。可以利用AI模拟未来所需的人才结构。
3.  **转向AI优先的运营模式**：需要从分散的业务单元试点，转向建立**强有力的中央治理架构**（如由AI转型官领导），统一战略、流程、数据和技术，以加速企业范围的转型。
4.  **进化数字化核心**：传统的ERP等核心系统是为高效、准确地处理业务而设计。现在需要将其进化为**企业的“社区大脑”**，成为AI转型的基础。这要求核心系统提供民主化的数据基础、可组合的架构以及支持广泛员工开发和使用AI智能体的平台。



## AI重塑企业：5：中国企业数字化转型指数洞察

在了解了规模化转型的路径后，我们来看看中国企业的具体实践处于何种阶段。埃森哲全球副总裁、大中华区技术服务部总裁俞毅先生分享了《2025年中国企业数字化转型指数报告》的核心发现。

该报告基于对754家大中型企业（包括中国企业和在华跨国公司）的调研，从6大维度、61项细分指标进行评估。2025年中国企业数字化转型的平均得分是**49分（满分100）**，表明价值创造仍有巨大空间。

报告揭示了三大关键趋势：

以下是当前中国企业数字化转型呈现出的三个主要特点：

1.  **坚持创新与全球化**：创新成为中国企业全球化的第一驱动力。数据显示，中国研发投入已与北美、欧洲相当，且上市企业研发投入增速（9%）远高于营收增速（5%）。例如，无锡先导智能通过持续创新，已成为全球锂电设备行业第一，其海外营收占比超过24%。
2.  **AI从工具走向战略**：AI应用正从早期的客服、营销等环节，向供应链、人力资源、财务等企业全方位渗透。然而，实现显著价值的企业仅占9%。主要的短板在于**数据治理**，仅有29%的企业对自身数据质量有信心。差异化的关键在于将企业私有数据与模型、业务深度结合。
3.  **韧性升级对抗脆弱**：在不确定的环境中，人才韧性和运营韧性变得更为重要。尽管许多企业为员工提供了生成式AI工具，但**系统性地为员工提供全方位AI培训和赋能的比例仅为47%**。在运营方面，如何从中国的卓越运营扩展到全球化、动态化的韧性运营是一大挑战。

基于这些趋势，报告提出四大行动建议：以创新实现再突破、开拓AI新商业、打造AI驱动的数字核心、再造人才与组织。



## AI重塑企业：6：数字化底座——AI规模化应用的基石

从指数报告中我们看到，数据治理和数字化基础是AI价值实现的短板。本节中，我们将深入探讨为何坚实的数字化底座如此关键。美的集团首席信息安全官、软件工程院院长刘向阳先生分享了美的的实践。

刘院长指出，AI要大规模应用并产生巨大业务价值，其基本功是**数字化**。如果数字化没有搞好，数据无法打通和治理，AI应用便是空中楼阁。**数字化底座相当于数字化大厦的地基**。

企业建设数字化底座有两种选择：自建数据中心或使用公有云。美的的方案是建设**全球统一的一朵云**，实现“云上云下统一”和“多云统一”。具体做法是：
*   在自建数据中心，将虚拟化技术升级为与公有云同代的技术体系。
*   在公有云上，利用其基础硬件和网络，部署美的自研的同一套云软件。
*   从而实现业务在数据中心与公有云之间、不同公有云之间的**无缝迁移与统一管理**，无需业务改造。

美的的数字化底座是一个完整的自研技术栈，包括云计算平台、AI算力平台、大数据平台、数据库平台、安全平台及研发效能平台等，并将大模型能力内嵌到各个系统中。美的每年在数字化上投入约40多亿元，所有应用均为自研，确保了技术的自主可控与快速迭代。



## AI重塑企业：7：数据重塑——AI驱动企业的燃料

上一节我们强调了数字化底座的重要性，而数据正是这个底座上最关键的资产。本节中，埃森哲全球副总裁、全球数据能力主管董潇山女士将探讨如何打造面向AI的数据基础。

她指出，许多公司长期投资数据，但仍面临挑战：数据过多却难以找到所需、数据质量低下、数据孤岛林立。在生成式AI时代，创建概念验证很容易，但难在**获取高质量数据来揭示真实的业务规律**。

为解锁AI价值，领先企业正在从三方面重塑数据基础：

以下是构建面向AI的数据能力需要关注的三个核心方面：

1.  **业务认知比技术认知更重要**：企业需要由业务部门主导，明确哪些数据最具战略价值，并以此规划投资。董女士以一家英国大型银行为例，说明其CEO亲自领导了一个五年数据计划，从简化技术架构、优化职能部门，到最终利用数据基础彻底重塑业务流程和创造新商业模式。
2.  **利用广泛的技术选择实现跨越式发展**：企业过去擅长处理**结构化数据**（表格数据），这是业务计算的精确基础。但现在需要同等重视**非结构化数据**（文本、图像、视频等）和**合成数据**。新的技术基础应包括：
    *   **数据产品**：像在商店选购商品一样，让业务人员能轻松找到可信的、已治理好的数据。
    *   **语义层**：为数据添加业务上下文和含义，说明数据是什么及如何使用。
    *   **安全检索增强生成**：确保企业私有数据能安全地与生成式AI模型结合使用。
3.  **以严谨的方法和组织纪律掌控数据**：数据治理不再是仅属于风险或IT团队的功能。**每个员工都需要成为数据工作者**，具备数据素养。例如，企业中存在大量无人查看的BI报告，可以利用生成式AI将其转变为对话式查询，让业务人员直接与数据交互，同时将原有报表制作人员解放出来，从事语义层构建、非结构化数据治理等更高价值的工作。



## AI重塑企业：8：圆桌对话（一）——技术领袖的AI实践与思考

在了解了战略、数据与基础架构后，我们进入实践对话环节。本节汇集了联想集团与美的集团两位技术负责人的真知灼见，探讨AI在大型科技与制造企业中的落地。

**联想集团**副总裁、首席数字化转型官李欣分享了联想的AI战略。联想提出“**更聪明的AI服务一切**”，并认为AI是未来十年的核心战略。具体举措包括：
*   **对内**：计划将集团40%的端到端流程内嵌AI，以提升运营效率。
*   **对外**：推出AIPC并已成为市场第一；同时将内部能力提炼，帮助多个城市构建城市智能体。

**美的集团**刘向阳院长阐述了数字化与AI的关系。他认为企业经营的核心是**能力（护城河）**和**效率**。数字化主要提升效率，而AI是强大的赋能工具。美的的AI应用已全面嵌入系统和底座，其价值衡量严格以**财经认可的降本增效指标**为准（例如，用AI翻译和生成营销内容所节省的预算）。

关于**组织与人才**，两位嘉宾观点略有不同：
*   **李欣**认为，若将AI定位为战略变革，必然会涉及流程重组、组织架构和人才模型的根本性变化。联想已设立十类新的AI相关岗位。
*   **刘向阳**则认为，AI更像编程技术，是赋能各团队的工具，不一定需要调整整体组织架构。美的通过**业务方担任赞助人、数字化团队提供技术支持**的联合项目组形式推动AI落地。

关于**数字员工（智能体）与人类员工的协同**：
*   美的目前尚未出现取代人的“数字员工”，AI主要用于完成传统数字化无法完成的任务。
*   联想已发布数字员工“联想乐享”，并正在探索“硅基人才”与“碳基人才”的协同，让资深员工转向知识管理，训练智能体变得更聪明。



## AI重塑企业：9：圆桌对话（二）——构建负责任的AI与信任

技术落地离不开信任的基石。本节对话聚焦“负责任的人工智能”，探讨如何在AI应用中构建可靠性、安全性与透明度。

**欧莱雅北亚及中国首席信息官赵峰**分享了举办“美妆科技黑客马拉松”并将“可信AI”设为独立赛道的初衷。她指出，AI的可持续发展依赖于用户和社会的信任。对于欧莱雅这样的企业，与消费者的互动必须是**安全、可信、透明且可解释的**。欧莱雅中国早在四年前就启动了数据转型计划，构建了湖仓一体的数据平台，为负责任地应用AI打下了坚实基础。

**艺康集团大中华区首席信息官欧庆国**介绍了艺康的AI治理模式。公司通过**数字化（后升级为AI）委员会**来制定红线与规则。他强调，AI治理需要**业务、技术、数据等多部门协同**。例如，财务部门需要帮助量化AI项目的价值（节省多少成本或增加多少营收），这能极大激发企业应用AI的活力。

关于**跨国公司的全球与区域协同**，两位嘉宾分享了相似的观点：
*   **赵峰**：在欧盟AI法案等法规框架上，需与全球总部对齐；在具体技术落地和应用场景设计上，则要充分尊重中国独特的市场生态和技术环境，并与本地团队协同。
*   **欧庆国**：遵循“**求同存异**”原则。在法律法规（如网络安全）上必须完全本地合规；在方法论上（如从业务痛点出发的“B-A-D-T”框架）可与全球达成共鸣，共同解决业务问题。

最后，对话展望未来，当多智能体架构成熟后，**透明度（Transparency）** 将成为解锁AI企业价值的关键要素。构建负责任的AI生态，需要内部各层级、各部门以及外部合作伙伴的共同努力与共创。



## AI重塑企业：10：圆桌对话（三）——AI如何重塑品牌与消费者连接

AI不仅重塑内部运营，也深刻改变着品牌与消费者的互动方式。本节对话探讨AI在时尚零售行业如何深化品牌体验。

**泰佩思琦（Tapestry）集团大中华区首席数字与信息官蔡超**分享了旗下Coach等品牌的AI实践。公司对AI的应用有着清晰的战略思考，并设立了**AI卓越中心**和**AI治理委员会**两个相互制衡又协同推进的组织。

在具体应用上，AI贯穿品牌端到端价值链：
*   **产品创新**：利用大数据和NLP理解年轻消费者偏好，并与Adobe合作，将AI生成的3D设计快速转化为样品，将新品开发周期从数月缩短至一周。
*   **供应链与运营**：通过AI优化库存管理，降低库存周期30%-40%，并实现动态定价。
*   **营销与内容**：AI不取代品牌灵魂，而是辅助创意总监快速迭代，并大规模生成产品展示内容，满足年轻消费者对“质价比”和深度了解产品的需求。
*   **消费者体验**：将客服项目升级为**陪伴式情感化的消费者体验枢纽**，拟人化地帮助用户选择产品。

关于**AI是否会降低品牌质感**，蔡超认为，**品牌是灵魂，AI是工具**。AI的作用是帮助品牌认识到自身的多面性，并将其规模化地呈现，而非取代品牌的核心价值。

展望未来，蔡超看好**具身智能与AI的结合**，认为这将带来陪伴式的情感体验，成为人们生活中无处不在的智能体。



## AI重塑企业：11：圆桌对话（四）——AI重塑制造：超越效率，拥抱超自动化

最后，我们将视角转向制造业。本节探讨AI如何推动工业领域超越简单的效率提升，走向“超自动化”。

**霍尼韦尔中国首席技术官施伟**分享了这家工业巨头的AI投资逻辑。霍尼韦尔将AI投资分为三类：
1.  **平台基础设施投资**：如全球性的“Forge”工业互联网平台，此类投资不计较短期ROI，是赋能业务的基础。
2.  **具体应用投资**：任何针对垂直场景的解决方案都必须考核明确的投资回报率。
3.  **前沿孵化投资**：预留预算用于探索性研究和概念验证，待成熟后再转入正式产品开发。

在霍尼韦尔，**生成式AI与传统AI/机器学习是相辅相成的关系**：
*   **传统AI/小模型**：更适用于对实时性、可靠性要求极高的**边缘控制场景**（如生产线参数优化）。
*   **生成式AI/大模型**：擅长沉淀和调用企业级知识，赋能上层应用（如智能楼宇的能耗优化、供应链排产）。

关于**AI决策的责任归属**，施伟指出，当前实践中，AI更多是提供参考建议，**最终决策权交给客户**。企业高层对AI的信任度，决定了其走向自主决策的速度。
