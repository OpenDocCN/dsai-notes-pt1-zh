# 035：数据可视化工具观点 🛠️

![](img/b7bd262e6e2ca72287cb305e5b345ced_1.png)

在本节课中，我们将了解几位数据专业人士在日常工作中最依赖的可视化工具及其原因。通过他们的分享，你可以初步认识不同工具的特点和适用场景。



上一节我们探讨了数据分析的基本流程，本节中我们来看看几位从业者如何利用工具将数据转化为直观的图表。



一位专家表示，他最常使用的工具是 **Cognos Analytics**。他列举了几个原因：该工具能让他快速导入电子表格或连接数据库，并对数据进行可视化。无论是自主探索数据、驱动字段，还是借助AI助手发现新数据集中的有趣信息，它都能胜任。此外，他还能利用其报告工具进行更复杂的分析，并构建和安排报告自动发送。例如，可以设置每周一早上自动向销售团队发送管道报告。他特别提到，该工具能整合多个数据源，系统可协助完成数据连接，并将所有信息呈现在一个高度交互的单一仪表板上，支持动态筛选、排序，并能轻松与整个组织共享。



另一位专家则主要依赖 **Looker**。这是一个位于公司内部数据库之上的数据可视化工具，与 **Tableau** 类似。她认为这类工具（如Looker和Tableau）的优点在于，能让组织中的每个人——无论是否是数据专业人士——都能轻松查看数据，并进行基本的聚合或排序操作。



对于探索性数据分析，有专家非常依赖 **R语言**。他近年来深刻体会到使用R进行基础数据分析和可视化的高效性，尤其是利用 **tidyverse** 这一软件包集合。tidyverse能帮助用户轻松加载数据、在不同层级进行聚合，并快速实现可视化。



还有观点认为，**Tableau** 和 **Power BI** 是显而易见的选择，因为它们易于上手且非常实用。随着越来越多的公司和人员开始使用，其内置模板和资源库也日益丰富。



![](img/b7bd262e6e2ca72287cb305e5b345ced_3.png)

最后，一位专家提到，基础的 **Excel** 以及整个 **Microsoft** 套件（如利用透视表和条件格式等功能）在数据准备阶段至关重要。在深入分析之前，先用这些工具确保数据是干净、合理且经过适当处理的。



本节课中，我们一起学习了不同数据专业人士对多种可视化工具（如Cognos Analytics, Looker, Tableau, R, Power BI, Excel）的评价和应用场景。理解这些工具的差异能帮助你在未来的数据分析工作中，根据具体任务选择最合适的助手。
