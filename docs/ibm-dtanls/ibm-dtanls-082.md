# 082：数据专业人员的职业选择 👩‍💻👨‍💻

![](img/9d11870509eb4eff45172f2b9b0455bd_1.png)

在本节课中，我们将聆听从业的数据专业人士谈论该领域内可用的各种职业选择。数据领域提供了多样化的职业路径，适合不同兴趣和技能背景的人士。



![](img/9d11870509eb4eff45172f2b9b0455bd_1.png)

如今，整个与数据相关的职业领域也变得非常丰富多彩且充满活力。它一直在不断发展，并为任何希望进入数据专业领域的人提供了广泛的选择范围。

以下是数据领域内一些主要的职业路径：

*   **数据分析师**：这是许多人的起点。
*   **数据科学家**：可以通过提升技能，从数据分析师成长为数据科学家。
*   **统计学家**：这是另一种基础角色。
*   **数据工程师**：可以进一步在数据处理的特定方向上专精，成为数据工程师。
*   **商业智能分析师/专家**：也可以从商业智能分析师开始，然后转向数据工程师。

换句话说，你可以选择**数据分析师 -> 数据科学家**的路径，或者**商业智能分析师 -> 数据工程师**的路径。这些是数据专业领域内两条并行的职业轨道。

此外，你还可以走向另一个方向，成为**机器学习工程师**或**人工智能工程师**等。因此，对于对数据领域感兴趣的人来说，存在许多这样的角色。

上一节我们概述了数据领域的整体职业图景，本节中我们来看看数据分析师常见的几种具体发展方向。

以下是数据分析师可选择的几种常见职业路径：

![](img/9d11870509eb4eff45172f2b9b0455bd_3.png)

*   **转向数据科学或机器学习工程**：深入钻研机器学习和工程领域，成为专注于机器学习建模的**数据科学家**或**机器学习工程师**。
*   **深入业务与战略**：更深入地了解所在业务，为高层公司战略提供信息支持。这个角色非常重要且有趣，近年来也发生了很大演变。
*   **成为管理者**：开始担任**人员管理者**，管理其他数据分析师，并负责筛选和确定工作的优先级。因为在组织中，能用数据回答的问题总是比能回答问题的人多。数据经理的角色在确保最重要的工作得以完成方面非常关键且有趣。



![](img/9d11870509eb4eff45172f2b9b0455bd_3.png)

除了上述技术或管理路径，数据分析技能在许多传统行业中也有广泛应用。

以下是一些同样需要数据分析能力的广泛职业领域：

*   簿记员
*   会计师
*   注册会计师
*   股票经纪人或政府/大公司的财务分析师
*   房地产经纪人

许多人都是优秀的数据分析师。但要从事这类工作，你必须真正喜欢数字，并且非常注重细节。如果这不是你的特点，或者数字不能让你感到兴奋，那么数据分析师可能并不适合你。

本节课中，我们一起学习了数据专业领域的多样化职业选择。我们从宏观的职业图景开始，了解了从数据分析师、数据科学家到数据工程师和商业智能专家的不同路径。接着，我们探讨了数据分析师常见的几种发展方向，包括深入技术、聚焦业务战略或转向管理。最后，我们也看到数据分析技能在会计、金融、房地产等众多传统行业中也极具价值。关键在于找到与个人兴趣和技能相匹配的路径。
