# 030：实现路径规划器 🧭

在本节课中，我们将学习如何将A*算法转化为计算机代码。上一节我们介绍了A*算法的核心思想，本节中我们来看看如何具体实现它。

## 概述

A*算法由Peter Hart、Nils Nilsson和Bertram Raphael等人在斯坦福大学开发。它是一种高效的最短路径搜索算法。现在，我们的挑战是将这个算法编写成可运行的代码。

## 算法回顾与实现思路

A*算法的核心是评估函数 **f(n) = g(n) + h(n)**，其中：
*   **g(n)** 是从起点到节点 `n` 的实际代价。
*   **h(n)** 是从节点 `n` 到目标的预估代价（启发函数）。

我们的目标是编写代码，在给定的路网图中找到从起点到终点的最优路径。

1.  **初始化数据结构**：我们需要一个开放列表（优先队列）来存储待探索的节点，一个关闭列表来记录已探索的节点，以及一个字典来记录每个节点的父节点和累积代价。
2.  **将起点加入开放列表**：起点的 `g` 值为0，`f` 值为 `h(start)`。
3.  **进入主循环**：只要开放列表不为空，就继续执行。
4.  **选择当前节点**：从开放列表中取出 `f` 值最小的节点作为当前节点。
5.  **检查是否到达目标**：如果是，则回溯构建路径并返回。
6.  **处理相邻节点**：对于当前节点的每一个邻居，计算其临时的 `g` 值和 `f` 值。
7.  **更新节点信息**：如果该邻居是首次被发现，或者找到了一条到达它的更优路径（`g` 值更小），则更新其 `g`、`f` 值和父节点，并将其加入开放列表。
8.  **将当前节点移入关闭列表**：表示已探索完毕。
