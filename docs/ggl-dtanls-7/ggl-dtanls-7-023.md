## GGplot2简介

GGplot2是R语言中最流行的可视化包，它是tidyverse的核心组件之一。这个包功能强大且用户友好，能帮助你充分挖掘数据的价值。

使用GGplot2的基本语法结构如下：

```r
ggplot(data = <DATA>) +
  <GEOM_FUNCTION>(mapping = aes(<MAPPINGS>))
```

其中，`<DATA>`是你的数据集，`<GEOM_FUNCTION>`是你选择的几何对象（如点、线、条等），`<MAPPINGS>`则定义了如何将数据变量映射到图形属性上。
