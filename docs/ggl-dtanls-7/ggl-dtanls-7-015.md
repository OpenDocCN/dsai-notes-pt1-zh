## 数据结构与数据框 📊

上一节我们介绍了R语言的基础知识。本节中，我们来看看R中一个核心的数据结构——数据框。

数据框是R中用于存储表格数据的一种结构。它类似于电子表格或SQL表，由行和列组成，每一列可以包含不同类型的数据。

以下是数据框的一些关键特性：

*   数据框的每一列都是一个向量，且所有列的长度必须相同。
*   列内的数据类型必须一致，但不同列的数据类型可以不同。
*   可以使用 `data.frame()` 函数创建数据框。

例如，创建一个简单的数据框：
```r
# 创建数据框
my_data <- data.frame(
  name = c("Alice", "Bob", "Charlie"),
  age = c(25, 30, 35),
  score = c(85.5, 92.0, 78.5)
)
# 查看数据框结构
str(my_data)
```

## 使用Tidyverse处理数据 🧰

了解了数据框的基本概念后，我们来看看如何利用强大的`tidyverse`包集合来操作和转换数据。

`tidyverse`是一系列为数据科学设计的R包，它提供了一套连贯且高效的工具来处理数据。以下是`tidyverse`中几个核心包及其用途：

*   **`dplyr`**：用于数据操作，提供过滤、排序、汇总等功能。
*   **`tidyr`**：用于数据整理，帮助将数据转换为“整洁”格式。
*   **`ggplot2`**：用于数据可视化，基于图形语法创建复杂的图表。
*   **`readr`**：用于快速读取矩形数据（如CSV、TSV文件）。

使用`dplyr`进行数据筛选和汇总的示例：
```r
library(dplyr)
# 筛选年龄大于28的记录
filtered_data <- filter(my_data, age > 28)
# 按姓名分组计算平均分
summary_data <- my_data %>%
  group_by(name) %>%
  summarise(avg_score = mean(score))
```

## 在R中检查数据偏差 ⚖️

上一节我们介绍了如何使用工具高效处理数据。本节中，我们将关注数据分析中的一个关键步骤——检查数据偏差。

数据偏差可能源于数据收集、采样或处理过程，识别偏差对于得出可靠结论至关重要。R社区提供了丰富的经验和代码范例，可以帮助我们学习如何检测类似数据中的偏差问题。

以下是检查数据时可以考虑的几个方面：

*   **样本代表性**：检查数据样本是否能够代表总体。
*   **缺失值模式**：分析缺失数据是否是随机出现的。
*   **分布检查**：使用统计测试和可视化来检查数据分布是否异常。

例如，通过可视化快速检查年龄分布：
```r
library(ggplot2)
ggplot(my_data, aes(x=age)) +
  geom_histogram(binwidth=5, fill="blue", alpha=0.5) +
  labs(title="年龄分布检查", x="年龄", y="计数")
```
