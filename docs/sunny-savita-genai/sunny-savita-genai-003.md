## 语言建模 📝

首先，我们来理解什么是语言建模。语言建模是自然语言处理中的一项核心任务，其目标是让计算机学会预测一段文本序列中下一个词（或字符）出现的概率。简单来说，就是根据已经出现的词，来猜测下一个最可能出现的词是什么。

一个语言模型可以计算一个句子 `S` 由词序列 `(w1, w2, ..., wn)` 组成的概率，公式通常表示为链式法则：
**P(S) = P(w1) * P(w2|w1) * P(w3|w1,w2) * ... * P(wn|w1, w2, ..., wn-1)**

## 从RNN到Transformer的时间线 ⏳

理解了语言建模的基础后，我们来看看实现它的模型是如何演进的。从循环神经网络到Transformer的转变，是生成式AI，特别是大语言模型得以发展的关键。

以下是这一演进过程中的主要模型架构：

*   **循环神经网络**：RNN是最早用于处理序列数据（如文本）的神经网络之一。它的核心特点是具有“记忆”能力，能够将上一步的隐藏状态传递到下一步，从而考虑上下文信息。其基本单元的计算可以简化为：**ht = f(W * xt + U * ht-1 + b)**，其中 `ht` 是当前隐藏状态，`ht-1` 是上一时刻的隐藏状态，`xt` 是当前输入。
*   **长短期记忆网络**：为了解决RNN在处理长序列时容易出现的梯度消失或爆炸问题，LSTM被提出。它通过引入“输入门”、“遗忘门”和“输出门”的机制，更有效地控制信息的保留与遗忘。
*   **门控循环单元**：GRU是LSTM的一个变体，它合并了部分门结构，参数更少，训练速度更快，但在许多任务上仍能保持与LSTM相近的性能。
*   **Transformer**：这是革命性的架构。它完全摒弃了循环结构，转而采用“自注意力机制”，允许模型在处理序列时同时关注所有位置的信息，从而极大地提升了并行计算效率和长距离依赖的建模能力。Transformer成为了后续所有主流大语言模型的基础。
*   **BERT**：基于Transformer编码器部分构建的模型。它采用“掩码语言模型”进行预训练，即随机遮盖句子中的一些词，让模型根据上下文进行预测，从而学习深层的双向语言表征。
*   **GPT**：基于Transformer解码器部分构建的模型。它采用“自回归语言模型”进行预训练，即根据前面的词预测下一个词，非常适合文本生成任务。

## BERT与GPT的核心特点 🔑

上一节我们介绍了从RNN到Transformer的演进，本节中我们来看看基于Transformer的两个最重要变体：BERT和GPT，它们各自有什么特殊之处。

*   **BERT的特点**：BERT的核心是“双向编码”。在预训练时，它通过随机遮盖句子中的词（Masked Language Model, MLM）来学习词语在完整上下文中的含义。这使得BERT在需要理解整个句子语义的任务（如文本分类、问答）上表现优异。其预训练目标可表示为：**预测被[MASK]替换的原始词汇**。
*   **GPT的特点**：GPT的核心是“自回归生成”。它严格地从左到右处理文本，根据已知的前文预测下一个词。这种特性使其天生就是优秀的文本生成器。GPT的预训练过程就是最大化序列的似然概率：**P(文本) = ∏ P(词i | 词1, 词2, ..., 词i-1)**。

## ChatGPT的训练过程 🏋️

了解了BERT和GPT的原理后，我们以GPT系列为例，深入看看像ChatGPT这样强大的对话模型是如何被训练出来的。它的训练通常分为三个关键阶段。

以下是ChatGPT类模型训练的核心步骤：

1.  **预训练**：在海量、无标注的互联网文本数据上训练一个基础语言模型（例如GPT-3）。这个阶段的目标是让模型学会语言的通用模式、语法、事实知识和一定的推理能力。模型通过预测下一个词的任务进行学习。
2.  **有监督微调**：使用高质量的人工标注的对话数据对预训练模型进行微调。数据形式通常是（提示， 理想回复）对。例如，提示是“解释一下牛顿第一定律”，对应的理想回复是一段准确的解释。这个阶段教会模型如何遵循指令并生成符合人类期望的回复。
3.  **基于人类反馈的强化学习**：这是让模型回复更加安全、有用、无害的关键步骤。首先，训练一个奖励模型来学习人类对回复质量的偏好（例如，哪个回复更好）。然后，使用这个奖励模型作为指导，通过强化学习算法（如PPO）进一步微调SFT模型，使其生成的回复能获得更高的奖励分数。

## 迁移学习与微调 🎯

我们刚刚详细了解了ChatGPT的复杂训练流程。对于大多数研究者和开发者来说，从头开始预训练一个大型模型成本极高。因此，迁移学习和微调成为了应用AI模型的关键技术。

迁移学习是指将一个在大型通用数据集上预训练好的模型（已具备广泛的知识），应用于特定的下游任务。在NLP领域，微调是实现迁移学习的主要方式。

以下是NLP微调的常见类型：

*   **特征提取**：将预训练模型（如BERT）作为一个固定的特征提取器，只训练附加在模型顶部的简单分类器或回归层。
*   **全参数微调**：解冻预训练模型的部分或全部参数，与任务特定的层一起，在新的、规模较小的任务数据集上进行训练，使模型适应新任务。
*   **提示微调/前缀微调**：一种参数高效的微调方法。保持预训练模型参数完全冻结，只训练少量附加的“软提示”向量或“前缀”向量，来引导模型适应新任务。

本节课中我们一起学习了生成式AI发展史上的关键概念。我们从语言建模的定义出发，追溯了从RNN、LSTM到Transformer的模型演进之路，剖析了BERT和GPT这两类核心架构的设计思想。接着，我们深入探讨了像ChatGPT这样的先进对话模型是如何通过预训练、有监督微调和基于人类反馈的强化学习三个阶段训练而成的。最后，我们介绍了迁移学习与微调，这些技术使得我们能够高效地将大模型的能力应用到各种具体任务中。理解这段历史和技术脉络，是深入掌握生成式AI的重要基础。
