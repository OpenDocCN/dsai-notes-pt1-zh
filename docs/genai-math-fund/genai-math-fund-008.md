### 构建多层感知机

![](img/72e3983a6fe9bea182104921bb00ff11_36.png)

![](img/72e3983a6fe9bea182104921bb00ff11_38.png)

现在，让我们以MLP为例，看看如何构建一个神经网络。构建MLP需要做出几个关键选择：网络的层数、每层的神经元数量以及使用的激活函数。

![](img/72e3983a6fe9bea182104921bb00ff11_40.png)

以下是构建一个MLP模型的关键代码结构：

```python
import torch
import torch.nn as nn

class NeuralNetwork(nn.Module):
    def __init__(self):
        super().__init__()
        self.flatten = nn.Flatten()
        self.linear_relu_stack = nn.Sequential(
            nn.Linear(28*28, 512), # 输入层到第一个隐藏层
            nn.ReLU(),
            nn.Linear(512, 512),   # 隐藏层到隐藏层
            nn.ReLU(),
            nn.Linear(512, 10),    # 隐藏层到输出层
        )

    def forward(self, x):
        x = self.flatten(x)
        logits = self.linear_relu_stack(x)
        return logits
```

![](img/72e3983a6fe9bea182104921bb00ff11_42.png)

![](img/72e3983a6fe9bea182104921bb00ff11_44.png)

![](img/72e3983a6fe9bea182104921bb00ff11_46.png)

在这个例子中，我们处理的是FashionMNIST数据集，图像大小为28x28。首先，我们使用`nn.Flatten()`将二维图像展平为一个784维的向量。然后，我们通过一个由`nn.Sequential`定义的线性层和ReLU激活函数堆叠而成的管道进行处理。

![](img/72e3983a6fe9bea182104921bb00ff11_48.png)

前向传播过程可以理解为一系列的矩阵乘法。例如，第一层的计算可以表示为：
**`z1 = W1^T * x + b1`**
其中，`x`是784维的输入向量，`W1`是形状为`(512, 784)`的权重矩阵，`b1`是512维的偏置向量。结果`z1`通过ReLU激活函数：**`a1 = ReLU(z1) = max(0, z1)`**。这个过程在后续层中重复。

![](img/72e3983a6fe9bea182104921bb00ff11_50.png)

最后一层输出10个“逻辑值”，对应10个分类。这些逻辑值随后会通过Softmax函数转换为概率分布。

## 模型训练与优化

![](img/72e3983a6fe9bea182104921bb00ff11_52.png)

![](img/72e3983a6fe9bea182104921bb00ff11_54.png)

![](img/72e3983a6fe9bea182104921bb00ff11_56.png)

网络构建完成后，下一步是训练和优化模型参数。

![](img/72e3983a6fe9bea182104921bb00ff11_58.png)

![](img/72e3983a6fe9bea182104921bb00ff11_60.png)

![](img/72e3983a6fe9bea182104921bb00ff11_62.png)

我们将使用梯度下降法来优化模型。其核心更新公式为：
**`W = W - α * ∇L/∂W`**
其中，`α`是学习率，`∇L/∂W`是损失函数关于权重`W`的梯度。PyTorch的自动微分机制会为我们计算这些梯度。

![](img/72e3983a6fe9bea182104921bb00ff11_64.png)

以下是训练循环的核心步骤：
1.  **前向传播**：输入数据通过模型，得到预测值。
2.  **计算损失**：使用损失函数（如交叉熵损失）计算预测值与真实标签之间的误差。
3.  **反向传播**：调用`loss.backward()`计算模型中所有参数的梯度。
4.  **参数更新**：调用`optimizer.step()`根据梯度更新权重。
5.  **梯度清零**：调用`optimizer.zero_grad()`将梯度缓冲区归零，防止梯度累积。

![](img/72e3983a6fe9bea182104921bb00ff11_66.png)

![](img/72e3983a6fe9bea182104921bb00ff11_68.png)

在训练过程中，我们通常会在多个“轮次”上重复这些步骤，并在每个轮次后评估模型在验证集上的性能，以监控训练过程。

## 模型评估与保存

![](img/72e3983a6fe9bea182104921bb00ff11_70.png)

模型训练完成后，我们需要评估其性能并保存下来以备后用。

![](img/72e3983a6fe9bea182104921bb00ff11_72.png)

在评估模式下，我们使用测试数据集来计算模型的损失和准确率。关键步骤是计算预测正确的样本数：我们取模型输出概率最高的类别作为预测结果，并与真实标签进行比较。

![](img/72e3983a6fe9bea182104921bb00ff11_74.png)

以下是计算批量准确率的示例：
```python
pred = output.argmax(1) # 获取预测类别
correct = (pred == y).type(torch.float).sum().item() # 统计正确数
```

![](img/72e3983a6fe9bea182104921bb00ff11_76.png)

训练出满意的模型后，可以将其保存到磁盘。
```python
# 保存模型
torch.save(model.state_dict(), ‘model.pth’)
# 加载模型
model.load_state_dict(torch.load(‘model.pth’))
```
这样，我们就不需要每次使用时都重新训练模型。

## 扩展到卷积神经网络

![](img/72e3983a6fe9bea182104921bb00ff11_82.png)

![](img/72e3983a6fe9bea182104921bb00ff11_84.png)

MLP的原理同样适用于更复杂的网络，如卷积神经网络。

![](img/72e3983a6fe9bea182104921bb00ff11_86.png)

构建CNN时，需要为每个卷积层指定关键参数：输入通道数、输出通道数、卷积核大小、步长和填充。CNN通常由多个“卷积块”（作为特征提取器）和顶部的“全连接层”（作为分类器）组成。在PyTorch中，只需将网络定义部分从MLP替换为CNN结构，而训练、评估和保存的流程完全保持不变。
