# 38：补充学习资源指南 🧭

在本节课中，我们将介绍如何获取和使用额外的学习资源，以弥补课程中因时间限制未能深入讲解的部分，并帮助你更好地准备生成式AI认证考试。

上一节我们介绍了课程的核心内容，本节中我们来看看如何利用外部资源进行补充学习。主讲人Angie Brown指出，由于时间限制，部分主题（如模型微调）未能详细展开。因此，她与freeCodeCamp合作整理了一份资源列表，供学员查漏补缺，确保学习成功。
