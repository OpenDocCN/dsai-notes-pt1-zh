# 16：16. 填充和步幅 🧩

![](img/2d726c6cf38a18b87e60fdc97328f64e_1.png)

在本节课中，我们将学习卷积神经网络中的两个重要概念：**填充**和**步幅**。它们是调整卷积操作、控制输出特征图尺寸以及平衡计算效率的关键技术。

![](img/2d726c6cf38a18b87e60fdc97328f64e_3.png)

上一节我们介绍了基础的卷积操作，本节中我们来看看如何通过调整步幅和填充来优化这个过程。

![](img/2d726c6cf38a18b87e60fdc97328f64e_5.png)

## 回顾卷积操作

![](img/2d726c6cf38a18b87e60fdc97328f64e_7.png)

卷积是一种简单的操作，用于在图像的不同区域检测特征。在之前的视频中，展示了如何将卷积核应用到图像上。接下来，将展示一些调整卷积操作的方法。

![](img/2d726c6cf38a18b87e60fdc97328f64e_9.png)

## 理解步幅 🚶‍♂️

![](img/2d726c6cf38a18b87e60fdc97328f64e_11.png)

步幅决定了卷积核在输入图像上每次滑动的距离。以下是关于步幅的详细说明。

![](img/2d726c6cf38a18b87e60fdc97328f64e_13.png)

当你将一个3x3的卷积滤波器应用到图像上时，首先计算滤波器与图像左上角3x3区域的元素级乘积。

![](img/2d726c6cf38a18b87e60fdc97328f64e_15.png)

然后，你向右移动一个像素，进行相同的计算。

![](img/2d726c6cf38a18b87e60fdc97328f64e_17.png)

接着，你将滤镜向下移动一个像素，并从最左侧重新开始。

![](img/2d726c6cf38a18b87e60fdc97328f64e_19.png)

这个过程持续进行，直到到达图像的右下角。这种每次向右或向下移动一个像素的方式，被称为使用**步长1**。

![](img/2d726c6cf38a18b87e60fdc97328f64e_21.png)

所以，你每次移动一个像素，步长为1。

![](img/2d726c6cf38a18b87e60fdc97328f64e_23.png)

向右移动的步长不必与向下移动的步长相同。例如，你的步长可以是向右2个像素，向下4个像素。

![](img/2d726c6cf38a18b87e60fdc97328f64e_25.png)

你的步幅越大，卷积核覆盖的图像区域就越少，这能为你的滤波器节省更多的计算空间。

![](img/2d726c6cf38a18b87e60fdc97328f64e_27.png)

但同时，更大的步幅会使计算速度更快。这绝对是一个需要在覆盖率和计算效率之间进行的权衡。

![](img/2d726c6cf38a18b87e60fdc97328f64e_29.png)

所以，步幅为2意味着每次你向右移动两步。

![](img/2d726c6cf38a18b87e60fdc97328f64e_31.png)

你的滤波器从左上角开始，但现在每次向右移动两个像素。

![](img/2d726c6cf38a18b87e60fdc97328f64e_33.png)

因为它在上下两个方向也移动，所以你在那里向下移动两个像素，因此只进行一次计算。

![](img/2d726c6cf38a18b87e60fdc97328f64e_35.png)

这更快。正如你所见，步幅决定了过滤器访问图像的多少部分。

![](img/2d726c6cf38a18b87e60fdc97328f64e_37.png)

在这个例子中，卷积的结果现在是一个更小的2x2矩阵。

![](img/2d726c6cf38a18b87e60fdc97328f64e_39.png)

因为你只访问了四个3x3区域中的滤波器，两个在上，两个在下。

![](img/2d726c6cf38a18b87e60fdc97328f64e_41.png)

## 理解填充 🖼️

![](img/2d726c6cf38a18b87e60fdc97328f64e_43.png)

填充是基本卷积的另一种调整。以下是关于填充的详细说明。

观察这个3x3的图像和2x2的滤波器。这里的像素值并不重要，所以已经把它们放在外面了。当你计算步幅为1的卷积时，在这种情况下，你将访问图片的四个部分。

分别是左上角、右上角、左下角和右下角。最后，中心像素被访问了四次，角落的像素只被访问一次，而边缘的其他像素被访问两次。这意味着中心保存的信息比边缘的信息获得更多的关注，这通常是正确的。

这是一个问题。假设边缘的特征是你关心的，例如，这可能是图像中一只完整的狗，或者可能是狗的一部分，比如它的鼻子。这是有用的信息，你想要确保你的滤波器对整个图像施加同等重要的强调。

![](img/2d726c6cf38a18b87e60fdc97328f64e_45.png)

为了解决这个问题，你可以给图像添加一个边框，这样所有的信息都出现在这个新图片的中心，就像被框起来的图片。

![](img/2d726c6cf38a18b87e60fdc97328f64e_47.png)

这叫做**填充**。填充的大小可以根据你的滤波器和图片的大小而变化。填充的值可以是任何实际值，但通常被设置为零，这叫做**零填充**。

![](img/2d726c6cf38a18b87e60fdc97328f64e_49.png)

所以，在你进行卷积计算之前，滤波器会扫描图像及其边框。

![](img/2d726c6cf38a18b87e60fdc97328f64e_51.png)

但这次，结果是图像中的每个原始像素都被访问了相同的次数。真正访问次数少的只是新添加的边框像素。这是因为现在图像中的每个原始像素都位于中心，被那个边框所包围。

## 总结 📝

![](img/2d726c6cf38a18b87e60fdc97328f64e_53.png)

本节课中我们一起学习了卷积神经网络中的两个核心调整参数。

![](img/2d726c6cf38a18b87e60fdc97328f64e_55.png)

*   **步幅**：告诉滤波器如何扫描图像，它影响输出特征图的大小以及原始像素被访问的频率。步幅越大，输出尺寸越小，计算越快，但可能丢失一些空间信息。
    *   公式：若输入尺寸为 $n \times n$，滤波器尺寸为 $f \times f$，步幅为 $s$，则输出尺寸约为 $\lfloor \frac{n - f}{s} + 1 \rfloor \times \lfloor \frac{n - f}{s} + 1 \rfloor$。

*   **填充**：就像在图像周围放置一个边框，以便给图像的边缘像素和中心像素相同的重要性。最常用的是零填充。
    *   公式：若填充大小为 $p$，则输出尺寸变为 $\lfloor \frac{n + 2p - f}{s} + 1 \rfloor \times \lfloor \frac{n + 2p - f}{s} + 1 \rfloor$。

![](img/2d726c6cf38a18b87e60fdc97328f64e_57.png)

通过合理设置步幅和填充，我们可以控制特征图的尺寸，保护边缘信息，并在模型的表达能力和计算成本之间取得平衡。
