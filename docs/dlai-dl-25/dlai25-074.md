## GauGAN的技术亮点 ⚙️

关于GauGAN的一个酷炫之处是，它使用了你在StyleGAN中看到的一些先进技术。它实际上使用了**自适应实例归一化**，以便将分割图作为输入并加以利用。

![](img/193eee9fb4f8d4c66e3975fe554b0333_43.png)

![](img/193eee9fb4f8d4c66e3975fe554b0333_43.png)

以下是其核心思想的一个简化表示：
```python
# 伪代码：自适应实例归一化 (AdaIN) 的核心思想
output = style_scale * (input - mean(input)) / std(input) + style_bias
```
它使用自适应实例归一化来为生成的图像添加风格信息。

![](img/193eee9fb4f8d4c66e3975fe554b0333_45.png)

![](img/193eee9fb4f8d4c66e3975fe554b0333_45.png)
