# Excel中级教程 - P66：使用XLOOKUP函数 🔍

![](img/797f4cc0ad75088c1560e826b4d80aeb_0.png)

在本节课中，我们将学习如何在Microsoft Excel中使用功能强大的XLOOKUP函数。XLOOKUP是VLOOKUP和HLOOKUP函数的现代替代品，它更灵活、更直观，能够解决许多传统查找函数的局限性。

## 概述

XLOOKUP函数的核心目的是根据已知信息查找并返回相关的其他信息。例如，你可以通过员工编号查找其姓名、部门或薪资。与VLOOKUP只能向右查找不同，XLOOKUP可以向任意方向查找数据，并且默认进行精确匹配，使用起来更加简便。

## XLOOKUP函数基础

上一节我们介绍了XLOOKUP的用途，本节中我们来看看它的基本语法和参数。

XLOOKUP函数的基本语法如下：
`=XLOOKUP(lookup_value, lookup_array, return_array, [if_not_found], [match_mode], [search_mode])`

以下是每个参数的含义：
*   **lookup_value**：要查找的值。
*   **lookup_array**：要搜索的单元格区域或数组。
*   **return_array**：要返回结果的单元格区域或数组。
*   **[if_not_found]**（可选）：如果未找到匹配项，则返回此文本。如果省略，则返回`#N/A`错误。
*   **[match_mode]**（可选）：指定匹配类型。`0`为精确匹配（默认），`-1`为精确匹配或下一个较小项，`1`为精确匹配或下一个较大项，`2`为通配符匹配。
*   **[search_mode]**（可选）：指定搜索模式。`1`为从第一项开始搜索（默认），`-1`为从最后一项开始搜索，`2`为对按升序排序的数组进行二分搜索，`-2`为对按降序排序的数组进行二分搜索。

## 实战演练：根据员工编号查找姓名

现在，让我们通过一个具体例子来实践。假设我们有一个员工列表，包含姓名、编号、部门和薪资。我们需要根据G2单元格中的员工编号，在A列中查找对应的员工姓名。

操作步骤如下：
1.  在目标单元格（例如H2）中输入公式：`=XLOOKUP(G2, C:C, A:A)`
2.  按下回车键，公式将返回与编号对应的员工姓名。
3.  如需处理未找到的情况，可以扩展公式：`=XLOOKUP(G2, C:C, A:A, “未找到”, 0)`
4.  使用填充柄向下拖动公式，即可为多个编号批量查找姓名。

## XLOOKUP的灵活应用

我们已经学会了基本的单向查找，XLOOKUP的强大之处在于其灵活性。它可以轻松实现多条件查找和逆向查找。

以下是XLOOKUP的一些高级应用场景：
*   **向左查找**：这是XLOOKUP相对于VLOOKUP的最大优势之一。例如，根据员工编号（C列）查找其部门（D列），公式为：`=XLOOKUP(G2, C:C, D:D)`
*   **根据姓名查找**：查找值不限于编号，也可以是姓名。例如，根据员工姓名（A列）查找其薪资（E列），公式为：`=XLOOKUP(G2, A:A, E:E)`
*   **处理未找到的值**：通过`[if_not_found]`参数，可以自定义查找失败时的返回结果，使表格更整洁。

## 与VLOOKUP的对比及注意事项

在结束之前，我们来总结一下XLOOKUP的主要优势和使用前提。

XLOOKUP相比VLOOKUP的主要优点包括：能够向左查找；语法更简洁直观；默认执行精确匹配，无需指定`FALSE`参数。

需要注意的是，XLOOKUP是较新的函数，目前主要适用于Microsoft 365、Excel 2021及更新版本的Excel。如果你的Excel版本中暂时没有XLOOKUP函数，可以先使用VLOOKUP函数完成工作。
