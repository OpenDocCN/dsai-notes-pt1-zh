# Excel高效技巧系列课程 - P28：数字格式详解 📊

![](img/016aab5dfb86cf7c4f484606195ab90f_0.png)

在本节课中，我们将学习Excel中各种数字格式的用法。掌握数字格式能让你更清晰地展示数据，无论是日期、时间、货币还是百分比。

## 概述

数字格式决定了单元格中数据的显示方式，而不改变其实际值。例如，日期在Excel内部以数字存储，但可以通过格式以“2023年10月27日”或“10/27/23”等形式显示。

## 如何访问数字格式

首先，选择需要更改格式的单元格或整列。例如，左键单击B列选中它。

接着，转到Excel主选项卡的功能区，在“数字”组中可以看到当前格式（如“常规”）和一个下拉列表。

## 常用数字格式解析

以下是Excel中几种核心的数字格式及其效果。

*   **常规格式**：Excel的默认格式，不应用任何特定格式。
*   **数字格式**：将数据以纯数字形式显示，可设置小数位数。例如，将日期格式改为“数字”会显示其背后的序列值。
*   **货币格式**：为数字添加货币符号（如$），并通常显示两位小数。公式示例：`=SUM(A1:A10)` 的结果可用此格式显示为 `$1,234.56`。
*   **会计专用格式**：与货币格式类似，但货币符号会左对齐，数字右对齐，便于列内数字垂直对齐阅读。
*   **短日期格式**：将日期显示为数字形式，例如 `2023/10/27`。
*   **长日期格式**：将日期显示为包含星期和月份全称的形式，例如 `2023年10月27日，星期五`。
*   **时间格式**：显示时间，可选择12小时制（带AM/PM）或24小时制。
*   **百分比格式**：将数字乘以100后显示，并添加百分号。例如，0.95显示为 `95%`。
*   **分数格式**：将小数显示为分数，例如0.5显示为 `1/2`。
*   **文本格式**：将数字作为文本处理，Excel不会对其执行计算。

## 使用“设置单元格格式”对话框

如果内置格式不能满足需求，可以使用更高级的选项。

上一节我们介绍了基础格式，本节中我们来看看如何通过“设置单元格格式”对话框进行精细调整。

有两种方法可以打开此对话框：
1.  点击“数字”组右下角的启动器按钮。
2.  在数字格式下拉列表中，滚动到底部并选择“更多数字格式”。

在对话框中，你可以：
*   在“分类”列表中选择类别（如日期、时间、特殊格式）。
*   在右侧选择具体的格式类型并预览示例。
*   例如，在“日期”类别中，可以选择“2023年3月14日”这种月份全拼的格式。
*   在“时间”类别中，可以选择“13:30”这种24小时制（军事时间）格式，并可选择是否显示秒数。
*   在“特殊”类别中，可以将数字快速格式化为邮政编码、电话号码或社会保险号码。

## 调整列宽以适配格式

更改格式后，显示内容可能过长或过短。你可以将鼠标指针移至列标题（如B和C列）之间的分隔线上，双击即可自动调整列宽至最合适的大小。

## 总结

本节课中我们一起学习了Excel数字格式的核心功能。我们了解了如何应用基础格式（如货币、日期），如何通过“设置单元格格式”对话框访问更多高级选项（如长日期、24小时制），以及如何快速调整列宽。正确使用数字格式能让你的电子表格更专业、更易读。
