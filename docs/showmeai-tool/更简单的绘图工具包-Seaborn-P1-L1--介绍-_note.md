## 什么是 Seaborn？

上一节我们介绍了本课程的主题，本节中我们来看看 Seaborn 的具体定义。

Seaborn 是一个基于 Matplotlib 构建的 Python 数据可视化库。如果你不熟悉 Matplotlib，它是一个用于创建静态、动画和交互式可视化的底层绘图库。Seaborn 在其基础上提供了更高级的接口和更美观的默认样式。

其核心优势在于，它用更简洁的代码就能生成统计图形。一个典型的 Matplotlib 绘图可能需要多行代码来设置样式，而 Seaborn 通常只需一行核心命令。

例如，绘制一个简单的散点图，对比代码如下：

```python
# Matplotlib 方式
import matplotlib.pyplot as plt
plt.scatter(x, y)
plt.title('My Plot')
plt.xlabel('X Axis')
plt.show()

# Seaborn 方式
import seaborn as sns
sns.scatterplot(x=x, y=y)
```

## 为什么选择 Seaborn？

了解了 Seaborn 的定义后，我们来看看它有哪些吸引人的特点。

以下是 Seaborn 的几个主要优点：

*   **美观的默认样式**：Seaborn 自带一套精心设计的配色方案和绘图样式，让图表看起来更专业、更现代。
*   **简化复杂图表**：对于统计图表（如分布图、箱线图、热力图等），Seaborn 提供了专门的、易于使用的函数。
*   **与 Pandas 无缝集成**：它能很好地处理 Pandas 的 DataFrame 数据结构，使得数据分析和可视化流程更加流畅。
*   **内置统计估计**：许多绘图函数能够自动计算并展示数据的统计信息，如趋势线、置信区间等。
