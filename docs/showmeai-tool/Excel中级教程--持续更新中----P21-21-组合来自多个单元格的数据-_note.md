# Excel中级教程 - P21：组合来自多个单元格的数据 📊

![](img/6689f1c6c070627a588a751f13ea0a13_0.png)

在本节课中，我们将学习如何在Excel中将多个单元格中的数据合并到一个单元格中。这个技巧对于创建客户档案、生成报告摘要或整理信息非常有用。

## 概述

我们将通过一个客户关系追踪表的例子，演示如何使用“&”符号（和号）将姓名、销售额等信息连接起来，并利用自动填充功能快速应用到整列数据。

---

## 数据合并基础

上一节我们介绍了数据的基本操作，本节中我们来看看如何将不同单元格的内容组合在一起。

假设我们有一个客户表格，其中名字（First Name）在A列，姓氏（Last Name）在B列，购买总额（Total Purchases）在C列。我们希望在G列创建一个“客户档案”，将这三项信息合并显示。

首先，在G2单元格输入标题“客户档案”，并利用“格式刷”工具使其与表格其他标题格式一致。

## 使用“&”符号连接内容

以下是使用“&”符号连接单元格内容的基本步骤：

1.  在目标单元格（例如G3）中输入等号（`=`）开始公式。
2.  点击或输入第一个要引用的单元格地址（例如A3）。
3.  输入连接符“`&`”。
4.  点击或输入第二个要引用的单元格地址（例如B3）。
5.  按回车键完成公式。

初始公式为：`=A3&B3`

执行后，G3单元格将显示“GinaPullen”，但名字和姓氏之间没有空格。

## 在连接中添加文本和空格

为了解决无空格的问题，我们需要在公式中手动添加空格作为文本字符串。

更新后的公式为：`=A3&" "&B3`

*   `" "` 代表一个空格字符。
*   这个公式将A3的内容、一个空格、B3的内容依次连接。

我们还可以添加更多信息和格式。例如，将购买总额也加入，并用冒号和美元符号进行格式化。

最终的组合公式如下：
`=A3&" "&B3&": $"&C3`

这个公式会将A3（名）、一个空格、B3（姓）、一个冒号加空格、一个美元符号以及C3（总额）连接起来，最终显示为“Gina Pullen: $40”。

## 使用自动填充应用公式

手动为每一行输入公式非常低效。我们可以利用Excel的“自动填充”功能快速复制公式。

1.  选中已输入公式的单元格（G3）。
2.  将鼠标指针移至该单元格右下角，直到指针变成黑色十字（即“自动填充柄”）。
3.  按住鼠标左键并向下拖动，覆盖需要应用公式的所有行。
4.  松开鼠标，公式将被复制到每一行，并且单元格引用会自动调整。

例如，拖动填充后，G4单元格的公式会自动变为 `=A4&" "&B4&": $"&C4`，从而正确组合第二行客户的数据。

**重要提示**：通过“&”连接生成的结果是文本字符串。即使其中包含数字（如`$40`），Excel也会将其视为文本，因此无法直接对这些结果进行数学运算（如求和）。

## 总结

![](img/6689f1c6c070627a588a751f13ea0a13_2.png)

本节课中我们一起学习了Excel中组合多个单元格数据的方法。核心操作是使用连接符“`&`”构建公式，例如 `=A1&" "&B1`，并配合“自动填充”功能高效地处理整列数据。这个技巧能帮助你快速整合信息，生成清晰易读的汇总内容。
