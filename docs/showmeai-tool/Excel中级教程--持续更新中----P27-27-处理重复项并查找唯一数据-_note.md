# Excel中级教程 - P27：处理重复项并查找唯一数据 🎯

![](img/409c19b730b648cee998141381746053_0.png)

在本节课中，我们将学习如何在Excel中识别并处理数据中的重复项，以及如何提取唯一值列表。掌握这些技巧能帮助你清理数据，确保分析的准确性。

## 概述：识别重复项

上一节我们介绍了数据的基本整理，本节中我们来看看如何处理重复数据。在示例中，我们有一个虚构的音乐CD商店库存表，其中包含一些重复的记录。

![](img/409c19b730b648cee998141381746053_0.png)

你可以看到，例如乐队“Alphaville”及其专辑“St attractor”出现了两次。电子表格中还存在其他重复项。虽然有时重复可能无关紧要，但本例中我们需要将其移除。

## 方法一：使用条件格式高亮重复项

以下是使用“条件格式”功能直观地标记出重复项的步骤。

1.  选中需要检查的数据范围。
2.  点击【开始】选项卡下的【条件格式】。
3.  选择【突出显示单元格规则】->【重复值】。
4.  在弹出的对话框中保持默认设置（“重复”和“浅红色填充深红色文本”），点击【确定】。

执行后，所有重复出现的单元格（如乐队名、专辑名）都会被高亮显示。此时，如果数据量小，你可以手动选中重复行并右键【删除】。

## 方法二：使用“删除重复项”功能

对于数据量大的情况，手动删除效率低下。我们可以使用“删除重复项”功能批量处理。

1.  点击数据区域内的任意单元格。
2.  转到【数据】选项卡，在【数据工具】组中点击【删除重复项】。
3.  在弹出的对话框中，选择要依据哪些列来判断重复。例如，同时勾选“乐队”和“专辑”列，则只有这两列信息完全相同的行才会被视作重复。
4.  点击【确定】，Excel会报告发现并删除了多少个重复值，保留了多个唯一值。

![](img/409c19b730b648cee998141381746053_2.png)

**核心概念**：选择不同的列组合，删除重复项的结果会不同。
*   若只依据“乐队”列删除，则会删除乐队名重复的所有行，只保留每个乐队的第一条记录。
*   若同时依据“乐队”和“专辑”列删除，则只会删除乐队和专辑都完全相同的行。

## 方法三：使用“高级筛选”提取唯一值列表

有时，我们不需要删除数据，只是想获得一个不重复的列表。例如，客户只需要所有乐队的名单。

以下是使用“高级筛选”提取唯一值列表的步骤。

1.  点击源数据列（如“乐队”列）中的任意单元格。
2.  转到【数据】选项卡，在【排序和筛选】组中点击【高级】。
3.  在弹出的对话框中：
    *   选择【将筛选结果复制到其他位置】。
    *   **列表区域**会自动填入选中的数据范围（如`$A$1:$A$62`）。
    *   **条件区域**留空。
    *   在【复制到】框中，点击右侧按钮，然后在工作表中选择一个空白单元格（如`$I$2`）作为粘贴起始位置。
    *   **务必勾选【选择不重复的记录】**。
4.  点击【确定】。

这样，一个仅包含不重复乐队名称的列表就会生成在指定位置。之后，你可以清除可能自动应用的格式，以便使用这个列表。

## 总结

本节课中我们一起学习了在Excel中处理重复数据的三种核心技巧：
1.  **高亮显示**：利用条件格式直观标记重复项，适合小范围数据检查和手动处理。
2.  **批量删除**：使用“删除重复项”功能，可依据指定列快速删除重复行。
3.  **提取列表**：通过“高级筛选”中的“选择不重复的记录”选项，生成唯一值列表以供他用。

合理运用这些方法，可以有效地管理和净化你的数据集。
