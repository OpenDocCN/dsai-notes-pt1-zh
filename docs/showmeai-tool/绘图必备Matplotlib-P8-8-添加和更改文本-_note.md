## 文本的重要性

添加和更改文本是绘图中非常重要的一部分。你需要为观众提供关于图表内容的信息。例如，你需要标记X轴以说明“50”这个数字代表什么。你需要标记Y轴以明确数据的度量单位。你还需要为图表添加一个有用的标题，以便人们理解数据的上下文以及你试图传达的内容。

## 实践：为图表添加文本

![](img/9fb8d9dd04796c9d352a78350c807c17_25.png)

让我们通过一个具体的例子来学习如何添加文本。我们将创建一个展示不同年龄平均胆固醇水平的条形图。

![](img/9fb8d9dd04796c9d352a78350c807c17_27.png)

首先，我们生成一个没有文本标签的基础图表。

![](img/9fb8d9dd04796c9d352a78350c807c17_29.png)

```python
# 假设已有数据准备
fig, ax = plt.subplots()
ax.bar(ages, avg_cholesterol)
```

![](img/9fb8d9dd04796c9d352a78350c807c17_31.png)

现在，我们的观众对X轴和Y轴代表什么毫无头绪。因此，我们需要添加坐标轴标签。

![](img/9fb8d9dd04796c9d352a78350c807c17_33.png)

### 添加坐标轴标签

![](img/9fb8d9dd04796c9d352a78350c807c17_35.png)

我们将使用坐标轴对象（`ax`）的 `set_xlabel` 和 `set_ylabel` 方法。

![](img/9fb8d9dd04796c9d352a78350c807c17_37.png)

```python
ax.set_xlabel('年龄')
ax.set_ylabel('平均胆固醇')
```

执行上述代码后，“年龄”这个词会出现在X轴下方，“平均胆固醇”会出现在Y轴左侧。

### 添加图表标题

![](img/9fb8d9dd04796c9d352a78350c807c17_39.png)

![](img/9fb8d9dd04796c9d352a78350c807c17_40.png)

接下来，如果我们想给这个图表一个标题，可以使用 `set_title` 方法。

![](img/9fb8d9dd04796c9d352a78350c807c17_42.png)

```python
ax.set_title('不同年龄的平均胆固醇水平')
```

![](img/9fb8d9dd04796c9d352a78350c807c17_44.png)

现在，“不同年龄的平均胆固醇水平”会显示为图表的标题。我们是在这个 `ax` 坐标轴对象上进行这些操作的，所以这是这个特定坐标轴的标题。

![](img/9fb8d9dd04796c9d352a78350c807c17_46.png)

## 核心要点总结

![](img/9fb8d9dd04796c9d352a78350c807c17_48.png)

本节课中我们一起学习了如何为Matplotlib图表添加和修改文本。以下是两个主要操作：
*   **添加坐标轴标签**：使用 `ax.set_xlabel()` 和 `ax.set_ylabel()` 方法。
*   **添加图表标题**：使用 `ax.set_title()` 方法。
