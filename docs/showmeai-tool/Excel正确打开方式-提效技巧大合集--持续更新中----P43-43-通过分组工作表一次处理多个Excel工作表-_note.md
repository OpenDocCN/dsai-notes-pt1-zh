# Excel高效技巧课程 - P43：43）通过分组工作表一次处理多个Excel工作表 📊

![](img/dbe3adaf002f1c6ab91c4ac3462410dc_0.png)

在本节课中，我们将学习如何在Excel中高效地处理包含多个工作表的工作簿。核心技巧包括**分组工作表以进行批量编辑**，以及**快速导航和管理大量工作表**。掌握这些方法能显著提升你的工作效率。

## 概述：为何需要分组工作表？ 🤔

当工作簿中包含多个结构相似的工作表时，逐个设置格式或输入相同数据非常耗时。通过分组工作表功能，你可以**一次性选中多个工作表**，随后进行的任何编辑操作（如输入数据、设置格式）都会同步应用到所有选中的工作表上。

## 如何创建并分组多个工作表

首先，我们从一个空白工作簿开始。默认情况下，工作簿底部只有一个工作表标签（如“Sheet1”）。

以下是添加并分组多个工作表的步骤：

1.  **添加新工作表**：点击现有工作表标签右侧的“+”号，即可添加新工作表。重复此操作，可以添加多个工作表（例如Sheet2、Sheet3等）。
2.  **分组相邻工作表**：点击第一个工作表标签（如Sheet1），按住 `Shift` 键，再点击最后一个工作表标签（如Sheet5）。此时，所有从Sheet1到Sheet5的标签背景将变为白色，表示它们已被**同时选中并分组**。
3.  **分组非相邻工作表**：若只想选择其中几个不连续的工作表，可以按住 `Ctrl` 键，然后依次点击需要的工作表标签（如Sheet1和Sheet3）。

**核心操作代码**：`Shift + 点击` 用于连续选择；`Ctrl + 点击` 用于非连续选择。

## 在分组状态下进行批量操作 🚀

上一节我们介绍了如何创建分组，本节中我们来看看分组后能执行哪些高效的批量操作。

一旦工作表被分组，你在**活动工作表**（组中当前显示的工作表）上执行的绝大部分操作，都会复制到组内所有其他工作表。

以下是你可以进行的操作示例：

*   **批量输入数据**：在A1单元格输入“Month”，在B1单元格输入“Task 1”，这些内容会出现在所有分组工作表的相同位置。
*   **使用填充柄**：在A列输入“January”后，可以使用单元格右下角的**自动填充手柄**向下拖动，快速生成月份序列。此序列也会同步到所有分组工作表。
*   **批量调整格式**：
    *   选中第一行，点击 **B**（加粗）按钮，所有工作表的第一行都会加粗。
    *   选中A到F列，在“开始”选项卡的“字体”组中，可以统一修改字体颜色或填充背景色。

完成所有编辑后，**务必记得取消分组**。只需单击任意一个未在分组内的工作表标签，或右键单击已分组的工作表标签并选择“取消组合工作表”即可。否则，后续操作会继续影响之前分组的所有工作表。

## 如何快速导航和管理大量工作表标签 🔍

当工作簿中的工作表数量非常多时，通过左右滚动箭头逐个查找会非常低效。

以下是更快捷的管理方法：

1.  **快速跳转到特定工作表**：在工作表导航箭头（左下角）上**单击右键**，会弹出一个包含所有工作表名称的完整列表。直接点击列表中的目标工作表名称，即可立即跳转。
2.  **重命名工作表**：双击工作表标签，标签名称会进入编辑状态，直接输入新名称后按 `Enter` 键即可。或者，右键单击标签，选择“重命名”。
3.  **调整工作表顺序**：直接点击并拖动某个工作表标签，将其移动到新的位置即可重新排序。
