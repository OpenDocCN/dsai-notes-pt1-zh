# Excel中级教程 - P19：使用TODAY函数追踪目标日期 📅

![](img/e92d5aca9a44d6cfbabd7746a6ec019f_0.png)

在本节课中，我们将学习如何使用Excel的TODAY函数来动态获取当前日期，并利用它来计算距离未来某个目标日期（如节假日或项目截止日）还有多少天。这是一个非常实用的技巧，可以用于创建自动更新的倒计时或日期提醒。

---

## 认识TODAY函数

TODAY函数是Excel中一个简单的日期函数，它不需要任何参数，其作用是返回当前的系统日期。每次打开或重新计算工作表时，这个日期都会自动更新。

其基本语法是一个简单的公式：
```
=TODAY()
```

上一节我们介绍了日期函数的基本概念，本节中我们来看看如何具体应用TODAY函数。

---

## 基础应用：插入当前日期

首先，我们学习如何在工作表中插入一个动态的当前日期。

1.  选择一个单元格，例如 `B3`。
2.  输入公式 `=TODAY()`。
3.  按下回车键，单元格中便会显示今天的日期。

这个功能非常有用。例如，在打印报表时，你可以在页眉或页脚插入此公式，以确保每次打印时都能自动印上当天的日期，便于记录和归档。

---

## 进阶应用：创建日期倒计时

理解了TODAY函数的基础用法后，我们可以进一步利用它来计算距离未来某个特定日期还有多少天。以下是创建倒计时列表的步骤。

### 第一步：建立数据源工作表

为了避免干扰主工作表中的其他数据，最佳实践是将固定的目标日期存放在一个单独的工作表中。

1.  点击工作表左下角的 **“+”** 号，创建一个新工作表（例如Sheet2）。
2.  在Sheet2的`A1`单元格输入公式 `=TODAY()`，作为动态的日期参照。
3.  在`B`列依次输入未来的目标日期，例如万圣节、感恩节等。

### 第二步：在主工作表计算倒计时

接下来，我们回到主工作表（Sheet1），利用公式计算距离每个节日的天数。

1.  在Sheet1的`E2`单元格（对应万圣节）输入等号 `=`。
2.  点击切换到Sheet2，用鼠标点击万圣节日期所在的单元格（例如`B2`）。此时公式栏会显示类似 `=Sheet2!B2` 的引用。
3.  在公式栏中，在此引用后输入减号 `-`，然后再次点击Sheet2中`TODAY()`函数所在的单元格（`A1`）。完整的公式应为：
    ```
    =Sheet2!B2 - Sheet2!$A$1
    ```
    （使用`$`符号锁定`A1`单元格，便于后续公式填充）
4.  按下回车键，单元格将显示计算结果，例如“26”，表示距离万圣节还有26天。

### 第三步：批量填充公式

对于其他节日，我们可以使用Excel的自动填充功能快速复制公式。

1.  选中已输入公式的单元格（`E2`）。
2.  将鼠标光标移至该单元格右下角，当光标变成黑色“+”号时，按住鼠标左键向下拖动，覆盖感恩节、圣诞节等对应的单元格区域。
3.  松开鼠标，公式即被复制，并自动计算出距离每个节日的天数。

---

## 应用场景与总结

本节课中我们一起学习了TODAY函数的核心用法。通过 `=TODAY()` 这个简单的公式，我们能够实现：

*   **动态日期标记**：确保文档日期始终最新。
*   **自动倒计时**：通过公式 **`=目标日期 - TODAY()`**，轻松计算距离未来事件的天数。

![](img/e92d5aca9a44d6cfbabd7746a6ec019f_2.png)

你可以将这个方法应用到各种场景中，例如追踪项目截止日期、提醒重要会议或生日，或者作为数据报告周期的自动提醒。将目标日期存放在独立的工作表中，能使你的主表格结构更清晰，数据管理更便捷。

希望本教程能帮助你更高效地使用Excel管理时间与日期。
