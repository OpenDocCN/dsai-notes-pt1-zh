# Excel中级教程 - P28：使用复选框 📋

![](img/2441a26a79ae27fdf3f186e170b77f3b_0.png)

在本节课中，我们将学习如何在Excel电子表格中添加和使用复选框。复选框是一种实用的表单控件，可用于标记任务完成状态或记录特定属性。我们将从最简单的添加方法开始，逐步介绍如何将复选框与单元格链接，并探讨其实际应用。

## 确保“开发工具”选项卡可见 🔧

在添加复选框之前，需要确保Excel的功能区中显示了“开发工具”选项卡。默认情况下，该选项卡是隐藏的。

以下是显示“开发工具”选项卡的步骤：
1.  在Excel功能区的任意位置右键单击。
2.  在弹出的菜单中选择“自定义功能区”。
3.  在右侧的“主选项卡”列表中，找到并勾选“开发工具”选项。
4.  点击“确定”按钮。

完成上述操作后，“开发工具”选项卡将出现在功能区中。

## 插入简单的复选框 ✅

我们将从一个简单的例子开始，学习如何插入不链接到单元格的复选框。

首先，切换到“开发工具”选项卡。在“控件”组中，点击“插入”按钮。在弹出的菜单中，选择“表单控件”区域下的复选框图标（注意：有两套控件，“ActiveX控件”功能更强大但较复杂，本例使用更简单的“表单控件”）。

点击复选框图标后，鼠标指针会变成十字形。在希望放置复选框的单元格（例如E2）内单击，即可插入一个默认带文本的复选框。

插入的复选框可能位置不精确。要移动它，需要**右键单击并拖动**复选框到目标位置，这与通常移动对象的方式不同。

若要修改复选框旁边的文本，可以右键单击复选框，选择“编辑文本”，然后删除原有内容并输入新文本（例如“已更新联系信息”）。

要为列表中的其他行快速添加复选框，可以选中已放置复选框的单元格（E2），然后拖动其右下角的填充柄向下填充。此操作会将复选框控件复制到下方的单元格中。

## 链接复选框到单元格 🔗

上一节我们介绍了如何插入简单的复选框，本节中我们来看看如何将复选框的状态（选中/未选中）链接到特定单元格，以便在公式中引用。

首先，按照上述方法在F2单元格插入一个新的复选框（例如用于标记“完成安全培训”）。

接下来，右键单击该复选框，选择“设置控件格式”。在弹出的对话框中：
*   在“值”区域，可以设置复选框的默认状态（未选中、已选中或混合）。
*   点击“单元格链接”框右侧的箭头按钮，然后选择希望链接的单元格（例如G2）。
*   还可以勾选“三维阴影”为复选框添加视觉效果。
*   设置完成后，点击“确定”。

现在，当勾选或取消勾选该复选框时，链接的单元格（G2）会相应地显示`TRUE`或`FALSE`。

**重要提示**：当使用填充柄复制已链接单元格的复选框时，所有复制的复选框默认会链接到同一个单元格（本例中是G2）。这意味着勾选其中任意一个，所有复选框都会联动。

要使每个复选框独立工作，必须为每个复选框单独设置其链接的单元格：
1.  右键单击第二个复选框（F3），选择“设置控件格式”。
2.  将其“单元格链接”修改为G3（或相应的目标单元格）。
3.  对后续的每个复选框重复此操作，分别链接到G4、G5等单元格。

虽然可以手动在设置框中输入单元格地址（如G4）来提高效率，但对于数据量很大的表格，此过程仍较为繁琐。

## 复选框的实际应用示例 📊

将复选框链接到单元格后，其产生的`TRUE`/`FALSE`逻辑值可以被其他Excel功能利用，实现自动化操作。

以下是两个典型的应用场景：

**1. 在公式中使用**
可以在其他单元格中使用`IF`函数，根据复选框的状态返回不同结果。
例如，在H2单元格输入公式：`=IF(G2=TRUE, “已完成”, “待完成”)`。当G2对应的复选框被勾选时，H2会显示“已完成”。

**2. 设置条件格式**
可以基于复选框的状态高亮显示整行数据，提供视觉提醒。
*   选中数据区域（如A2到F2）。
*   点击“开始”选项卡下的“条件格式” -> “新建规则”。
*   选择“使用公式确定要设置格式的单元格”。
*   输入公式：`=$G2=FALSE`（假设状态列在G列）。
*   点击“格式”按钮，设置当公式为真时的格式（如填充红色）。
*   点击“确定”后，所有“完成安全培训”复选框未勾选的行将被高亮为红色。

此外，为了界面整洁，可以将链接状态值的列（如G列）隐藏（右键点击列标选择“隐藏”），而不影响复选框和公式的正常工作。

## 课程总结 🎯

本节课中我们一起学习了Excel中复选框的使用。
*   我们首先确保了“开发工具”选项卡可见，这是使用表单控件的前提。
*   接着，我们学会了插入简单的复选框，并可以通过拖动填充柄快速复制。
*   然后，我们探索了更高级的用法：将复选框链接到单元格，使其能输出`TRUE`/`FALSE`值。同时，我们也了解到批量复制链接复选框时需要手动调整每个控件的链接指向。
*   最后，我们看到了复选框状态值在`IF`函数和条件格式中的实际应用，这大大增强了表格的交互性和自动化能力。

复选框是一个简单而强大的工具，非常适合用于创建清单、跟踪任务进度或构建交互式报表。
