# Excel中级教程 - P13：使用PROPER函数修正文本大小写 ✨

在本节课中，我们将学习如何使用Excel的PROPER函数，快速修正单元格中不规范的英文大小写格式。这个功能对于整理从他人处接收到的、大小写混乱的数据尤其有用。

## 问题场景与解决方案概述

假设你收到一个电子表格，其中的电影标题大小写格式混乱：有些全大写，有些全小写，还有一些大小写规则不一致。手动修正这些内容既费时又费力。

幸运的是，Excel提供了一个名为`PROPER`的函数，可以自动将文本转换为每个单词首字母大写、其余字母小写的标准格式。接下来，我们将一步步学习如何应用这个函数。

## 操作步骤详解

上一节我们介绍了`PROPER`函数能解决的问题，本节中我们来看看具体的操作流程。

![](img/9f08b408fe5d3dd60bbb1605f45afaa1_1.png)

### 第一步：插入新列

首先，在包含问题数据的列（例如A列）旁边插入一个空白列，用于存放修正后的结果。

1.  右键点击A列右侧的列字母（如B列）。
2.  在弹出的菜单中选择“插入”。

![](img/9f08b408fe5d3dd60bbb1605f45afaa1_1.png)

### 第二步：应用PROPER函数

在新插入列的第一个单元格（如B2）中输入公式，引用问题单元格。

1.  点击单元格B2。
2.  输入公式：`=PROPER(A2)`
3.  按下键盘上的`回车`键。

此时，B2单元格会显示A2单元格文本经`PROPER`函数处理后的结果，即每个单词的首字母变为大写。

### 第三步：批量填充公式

无需为每个单元格重复输入公式，我们可以使用填充柄快速将公式应用到整列。

1.  选中已输入公式的单元格B2。
2.  将鼠标指针移至该单元格右下角的小绿方块（填充柄）上，直至指针变为黑色加号。
3.  按住鼠标左键，向下拖动至需要填充的最后一个单元格，然后松开鼠标。

拖动后，B列的所有单元格都会应用相同的公式逻辑，自动修正A列对应行文本的大小写。

### 第四步：将公式结果转换为静态值

目前B列的数据依赖于A列的公式。如果我们直接删除原始的A列，B列的数据将会出错。因此，需要将B列的公式结果转换为静态的文本值。

以下是操作步骤：

1.  点击B列的列字母，选中整列。
2.  右键点击并选择“复制”。
3.  右键点击A列的列字母，选择“选择性粘贴”。
4.  在弹出的对话框中，将“粘贴”选项从“全部”切换为“值”。
5.  点击“确定”。

现在，A列中的数据已经是修正后的静态文本，不再包含公式。你可以通过点击A列单元格，在公式栏中查看确认。

### 第五步：清理辅助列

最后，删除之前用于辅助操作的B列即可。

1.  右键点击B列的列字母。
2.  选择“删除”。

至此，A列中的文本已全部被修正为规范的大小写格式，且不依赖于任何其他数据。

## 注意事项与总结

本节课中我们一起学习了使用`PROPER`函数高效修正文本大小写的方法。回顾一下核心流程：**插入新列 → 输入`=PROPER(原单元格)`公式 → 拖动填充 → 选择性粘贴为“值” → 删除辅助列**。

需要提醒的是，`PROPER`函数遵循严格的规则（每个单词首字母大写）。如果原文本中有特殊的缩写或固定写法（例如“Back to the Future II”中的“II”），该函数可能会进行“错误”的修正。在这种情况下，你可能需要手动进行微调。
