# Excel高效技巧课程 - P46：使用强大的AND函数 🧠

![](img/5d134be359adaaaf159264a337a5f4ba_0.png)

在本节课中，我们将学习如何使用Excel中的AND函数。这个函数用于判断多个条件是否同时为真，是进行复杂逻辑判断的得力工具。

## 概述

AND函数的核心作用是：当所有给定的条件都为真时，返回`TRUE`；如果其中任何一个条件为假，则返回`FALSE`。其基本语法如下：

**公式：**
`=AND(条件1, 条件2, 条件3, ...)`

接下来，我们将通过一个具体的案例来演示它的应用。

## 案例背景

假设我们有一张虚构公司的员工数据表。公司管理层决定，为**销售部门**中**工龄至少5年**且**绩效评估至少为6分**的员工发放奖金。

我们的目标是在Excel中自动判断哪些员工符合条件。

## 第一步：使用基础AND函数

首先，我们使用基础的AND函数来判断员工是否符合获奖条件。

以下是具体操作步骤：

1.  在目标单元格（例如F2）输入等号`=`和函数名`AND(`。
2.  输入第一个逻辑测试：判断工龄是否大于4年（即至少5年）。可以输入`B2>4`。
3.  输入逗号`,`分隔，然后输入第二个逻辑测试：判断绩效是否大于5（即至少6分）。可以输入`D2>5`。
4.  再次输入逗号`,`，输入第三个逻辑测试：判断部门是否为“销售”。由于“销售”是文本，需要加上引号，输入`E2="销售"`。
5.  输入右括号`)`闭合函数，然后按回车键。

完整的公式如下：
`=AND(B2>4, D2>5, E2="销售")`

按下回车后，单元格会根据判断结果显示`TRUE`或`FALSE`。对于第一位员工Gina Pullin，如果她的数据满足所有条件，结果将显示为`TRUE`。

**公式应用技巧：**
输入公式后，无需为每一行重复输入。只需选中已输入公式的单元格，拖动右下角的填充柄向下填充，或直接双击填充柄，公式便会自动复制到整列。

## 第二步：结合IF函数输出更直观的结果

上一节我们介绍了基础AND函数会返回`TRUE`或`FALSE`。本节中我们来看看如何结合IF函数，让结果显示为更直观的文字，例如“奖金”或空白。

我们可以将AND函数嵌套在IF函数中使用。IF函数的语法是：`=IF(条件, 条件为真时返回的值, 条件为假时返回的值)`。

以下是具体操作步骤：

1.  在目标单元格输入`=IF(`，开始IF函数。
2.  将整个AND函数作为IF函数的“条件”部分输入。即输入`AND(B2>4, D2>5, E2="销售")`。
3.  输入逗号`,`，然后输入条件为真时希望显示的内容，例如`"奖金"`（文本需加引号）。
4.  再次输入逗号`,`，输入条件为假时希望显示的内容。如果希望留空，可以输入`""`（两个英文双引号）。
5.  输入右括号`)`闭合IF函数，按回车键。

完整的嵌套公式如下：
`=IF(AND(B2>4, D2>5, E2="销售"), "奖金", "")`

应用此公式后，符合条件的员工旁会显示“奖金”，不符合条件的则显示为空白单元格，使结果一目了然。
