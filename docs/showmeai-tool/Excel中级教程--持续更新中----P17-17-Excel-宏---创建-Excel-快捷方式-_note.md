# Excel中级教程 - P17：Excel宏 - 创建快捷方式 🚀

![](img/2bdad9b27f9903642b3c2b937f9c8ab5_0.png)

在本节课中，我们将学习如何使用Excel宏来创建快捷方式，以自动化重复性任务，从而提升工作效率。我们将从一个简单的例子开始：创建按钮，实现不同工作表之间的快速跳转。

## 概述

宏是Excel中用于自动执行一系列操作的功能。通过录制宏，我们可以将一系列手动步骤（如切换工作表、设置单元格格式）保存下来，并通过按钮或快捷键一键执行。本节课将指导你完成创建宏、将其绑定到按钮以及保存启用宏的工作簿的完整流程。

## 1. 准备工作与录制宏

上一节我们介绍了宏的基本概念。本节中，我们来看看如何开始录制第一个宏。

首先，确保你的Excel功能区显示了“开发工具”选项卡。如果未显示，请在功能区任意位置右键单击，选择“自定义功能区”，然后在主选项卡列表中勾选“开发工具”。

我们将创建一个用于从“员工”工作表跳转到“项目”工作表的宏。

1.  切换到“开发工具”选项卡。
2.  点击“录制宏”按钮。
3.  在弹出的对话框中，为宏命名（例如“按钮到项目标签”）。
4.  为宏设置一个键盘快捷键（例如 `Ctrl+Shift+A`）。请注意避免与常用快捷键（如 `Ctrl+C`）冲突。
5.  将宏的保存位置设置为“当前工作簿”。
6.  可选项：为宏添加描述。
7.  点击“确定”开始录制。

**核心操作**：开始录制后，只需点击目标工作表（如“项目”）的标签，然后点击“停止录制”按钮。宏录制的是具体的操作（点击），而非鼠标移动或时间间隔。

## 2. 创建并绑定表单控件按钮

录制好宏之后，我们需要一个方便的方式来触发它。以下是创建按钮并将其与宏关联的步骤。

1.  在“开发工具”选项卡中，点击“插入”，然后选择“表单控件”下的“按钮”图标。
2.  鼠标指针会变成十字形，在希望放置按钮的位置单击并拖动以绘制按钮。
3.  松开鼠标后，会弹出“指定宏”对话框。
4.  从列表中选择你刚刚录制的宏（如“按钮到项目标签”），点击“确定”。
5.  要修改按钮文字，可以右键单击按钮选择“编辑文字”，或直接单击按钮上的文字进行修改。
6.  单击按钮外的区域退出编辑，然后单击按钮即可测试宏是否正常工作。

**编辑按钮**：如需调整按钮位置或大小，需先**右键单击**按钮将其选中，然后才能进行拖动或缩放操作。

## 3. 创建交互式格式设置宏

上一节我们创建了用于导航的宏。本节中，我们来看看如何创建一个更智能的宏，它能根据当前选中的单元格来应用格式。

这个宏将把选中的单元格或区域标记为“已审核”（例如填充蓝色背景），而无需预先指定具体单元格。

1.  首先，**不要**选择任何特定单元格，只需点击工作表任意空白处。
2.  点击“开发工具”选项卡中的“录制宏”。
3.  将宏命名为“标记为已审核”，并设置快捷键（如 `Ctrl+Shift+C`）。
4.  点击“确定”开始录制。
5.  转到“开始”选项卡，选择一种填充颜色（如蓝色）。
6.  返回“开发工具”选项卡，点击“停止录制”。
7.  按照上一节的方法，创建一个按钮并将宏“标记为已审核”分配给它。

**工作原理**：由于录制宏时没有预先选择特定单元格，此宏会将操作（填充蓝色）应用于**执行宏时当前选中的任何单元格或区域**。你可以选中一个名字、一行或一个区域，然后点击按钮，选中的内容就会变为蓝色。

## 4. 保存与安全设置

在包含宏的工作簿中完成工作后，保存步骤至关重要。以下是必须注意的事项。

包含宏的工作簿必须保存为“Excel启用宏的工作簿（*.xlsm）”格式，否则宏将无法被保存。

1.  点击“文件” > “另存为”。
2.  在“保存类型”下拉列表中，选择“Excel启用宏的工作簿（*.xlsm）”。
3.  为文件命名并点击“保存”。

![](img/2bdad9b27f9903642b3c2b937f9c8ab5_2.png)

**安全警告**：首次打开包含宏的 `.xlsm` 文件时，Excel通常会显示安全警告，提示“宏已被禁用”。要使用文件中的宏功能，必须点击“启用内容”按钮。

![](img/2bdad9b27f9903642b3c2b937f9c8ab5_3.png)

## 总结

本节课中我们一起学习了Excel宏的基础应用。我们掌握了如何通过“录制宏”功能将一系列操作自动化，如何创建表单控件按钮来方便地触发宏，以及如何创建能根据选中区域动态应用格式的智能宏。最后，我们明确了保存启用宏的工作簿（`.xlsm`格式）以及打开时启用内容的重要性。宏是强大的效率工具，从简单的导航到复杂的格式设置，都能帮助你节省大量重复操作的时间。
