在本节课中，我们将通过一个具体的示例，了解 Jupyter Notebook 的实际应用场景和强大功能。我们将看到它如何被用于数据科学项目，以及它如何促进代码、文档和结果的可视化与共享。

![](img/bebfc5b1a8153c6a7ba708b3cb0c3183_4.png)

上一节我们介绍了 Jupyter Notebook 的基本概念和界面。本节中，我们来看看它在真实世界中的具体应用。

![](img/bebfc5b1a8153c6a7ba708b3cb0c3183_7.png)

我将展示一个来自 GitHub 的代码库，其中包含一个有趣的 Jupyter Notebook 画廊。如果你对 Jupyter Notebook 的用途感到好奇，答案是它们可以用于你可能想用数据和 Python 做的任何事情。这个画廊由许多人汇编而成，汇总了许多不同的 Notebook。

![](img/bebfc5b1a8153c6a7ba708b3cb0c3183_8.png)

![](img/bebfc5b1a8153c6a7ba708b3cb0c3183_10.png)

以下是该画廊中包含的一些主要类别示例：
*   通用 Python 编程和科学计算。
*   统计学和概率。

![](img/bebfc5b1a8153c6a7ba708b3cb0c3183_12.png)

例如，在“统计学和概率”部分，有一个关于“在野外数据集中标记的人脸子集上进行面部识别”的 Notebook。通过一个在线查看器链接，我们可以直接查看这个 Notebook。

![](img/bebfc5b1a8153c6a7ba708b3cb0c3183_13.png)

![](img/bebfc5b1a8153c6a7ba708b3cb0c3183_15.png)

打开后，你会看到使用这个在线查看器，我们能够：
*   确切地看到作者使用的代码。
*   确切地看到他们执行了哪些操作。
*   查看结果输出的图像。
*   查看他们基于数据生成的图表。

![](img/bebfc5b1a8153c6a7ba708b3cb0c3183_17.png)

作者还可以使用 Markdown 文本将整个分析过程划分为逻辑清晰的多个部分。这就是 Jupyter Notebook 在科学界和数据科学家中越来越受欢迎的原因。

![](img/bebfc5b1a8153c6a7ba708b3cb0c3183_18.png)

![](img/bebfc5b1a8153c6a7ba708b3cb0c3183_19.png)

一般来说，这是因为你能够在一个 Notebook 中完成以下工作：
1.  进行一些实验和建模。
2.  对你的数据进行任何你想要的转换和操作。
3.  训练一个能够进行预测的模型。

![](img/bebfc5b1a8153c6a7ba708b3cb0c3183_20.png)

![](img/bebfc5b1a8153c6a7ba708b3cb0c3183_21.png)

同时，你还可以：
*   撰写关于项目背景和分析思路的说明文字。
*   轻松地将整个 Notebook 分享给其他人。

![](img/bebfc5b1a8153c6a7ba708b3cb0c3183_23.png)

这是一种非常好的方式，它能让你的工作对他人更加可见和透明。你可以与任何人分享这个 Notebook，即使他们不具备深厚的技术背景。你不需要为了分享一些分析而专门去创建一个网络应用程序或完整的网站。

![](img/bebfc5b1a8153c6a7ba708b3cb0c3183_25.png)

因此，Jupyter Notebook 在许多不同的场合都非常方便。虽然我不确定你想用它们具体做什么，但我相信它一定非常有趣且有用。

![](img/bebfc5b1a8153c6a7ba708b3cb0c3183_26.png)

![](img/bebfc5b1a8153c6a7ba708b3cb0c3183_27.png)

本节课中，我们一起学习了 Jupyter Notebook 在实际项目中的应用。我们看到了它如何整合代码、可视化结果和文本说明，成为一个强大的交互式文档，并极大地促进了工作的可重复性和协作共享。
