# 7：准备最终报告 📄

在本节课中，我们将学习如何准备一份结构清晰、重点突出的数据科学最终报告。我们将以MATLAB Live Script为基础，介绍如何将代码与结果分离，创建易于导航的文档，并导出为合适的格式。

---

![](img/47649ed3ca09f9b2beb6ea0c53e48790_1.png)

![](img/47649ed3ca09f9b2beb6ea0c53e48790_2.png)

可执行的Live Script非常适合与需要交互、编辑和运行代码的同事共享协作，但它并非总是呈现最终结果的理想格式。本数据科学报告的要求明确指出，需要以**非可执行格式**呈现最终结果。

在这种情况下，将代码与结果交织在一起，会使包含多个关键结果的报告难以阅读。

![](img/47649ed3ca09f9b2beb6ea0c53e48790_4.png)

因此，你需要将重要结果放在报告主体部分优先展示，而将代码放在附录中供参考。

![](img/47649ed3ca09f9b2beb6ea0c53e48790_6.png)

本节视频将以Course3项目为例，展示如何使用Live Script准备数据科学报告。你将学习如何将代码示例放入附录、在必要章节间创建链接，以及将图形以图片形式插入。

### 创建目录与章节结构

![](img/47649ed3ca09f9b2beb6ea0c53e48790_8.png)

上一节我们明确了报告的结构原则，本节我们来看看如何具体搭建报告框架。

![](img/47649ed3ca09f9b2beb6ea0c53e48790_10.png)

首先从添加目录和必要章节开始。

在空白脚本中添加目录后，目录将随着章节标题的添加而自动更新。在本例中，我们先添加顶级章节标题，并立即填写一些细节。

![](img/47649ed3ca09f9b2beb6ea0c53e48790_12.png)

以下是各独立章节可能包含的要求：
*   **书面细节描述**：对每个部分进行文字说明。
*   **需放入附录的参考代码**：将执行的代码放在附录部分。

![](img/47649ed3ca09f9b2beb6ea0c53e48790_14.png)

### 填充报告内容与创建链接

![](img/47649ed3ca09f9b2beb6ea0c53e48790_16.png)

在搭建好框架后，接下来需要填充具体内容并建立内部链接。

![](img/47649ed3ca09f9b2beb6ea0c53e48790_18.png)

Course3项目的模板包含了对数据的简要描述。你需要为自己的当前项目撰写类似的内容。

![](img/47649ed3ca09f9b2beb6ea0c53e48790_20.png)

请注意，模板还包含了对用于**数据导入、重构和初始特征工程**的脚本的引用。

为了防止任何可能的脚本执行错误，这段代码需要作为**非可运行的代码示例**添加到附录中。

![](img/47649ed3ca09f9b2beb6ea0c53e48790_22.png)

![](img/47649ed3ca09f9b2beb6ea0c53e48790_24.png)

接下来，为了确保报告易于他人浏览，你需要在正文的相应部分添加指向附录代码的**超链接**。

![](img/47649ed3ca09f9b2beb6ea0c53e48790_26.png)

具体操作步骤如下：
1.  高亮你想要设置为链接的文本。
2.  点击“超链接”按钮。
3.  选择“链接到文档中的位置”。
4.  然后滚动到附录并点击相应的标题。

请确保为报告的每个章节添加所有必需的描述和超链接。

![](img/47649ed3ca09f9b2beb6ea0c53e48790_28.png)

![](img/47649ed3ca09f9b2beb6ea0c53e48790_29.png)

### 插入图形与模型细节

![](img/47649ed3ca09f9b2beb6ea0c53e48790_31.png)

处理完文本和代码后，我们来看看如何将分析结果中的图形有效地整合到报告中。

![](img/47649ed3ca09f9b2beb6ea0c53e48790_33.png)

![](img/47649ed3ca09f9b2beb6ea0c53e48790_35.png)

那么，如何插入图形呢？

在报告中包含图形且无需代码运行的最简单方法是：在Live Editor中右键单击图形，然后选择“**复制输出**”。

![](img/47649ed3ca09f9b2beb6ea0c53e48790_37.png)

![](img/47649ed3ca09f9b2beb6ea0c53e48790_39.png)

然后将图片粘贴到报告的相应章节。

如果你使用 **Classification Learner App** 进行预测建模，可以通过将鼠标悬停在图形右上角并选择“**复制为图像**”来将图形复制到报告中。

![](img/47649ed3ca09f9b2beb6ea0c53e48790_41.png)

不要忘记同时包含App中显示的**模型详细信息**。

![](img/47649ed3ca09f9b2beb6ea0c53e48790_43.png)

此外，如果你使用了该App，则需要导出模型训练函数，并将代码放入附录。

![](img/47649ed3ca09f9b2beb6ea0c53e48790_45.png)

### 最终检查与导出

![](img/47649ed3ca09f9b2beb6ea0c53e48790_46.png)

在完成所有内容的填充和格式调整后，最后一步是进行最终检查并导出报告。

![](img/47649ed3ca09f9b2beb6ea0c53e48790_48.png)

在你仔细检查报告已包含所有必需内容后，使用“**保存 > 导出为 PDF**”将其导出为所需格式。

![](img/47649ed3ca09f9b2beb6ea0c53e48790_50.png)

![](img/47649ed3ca09f9b2beb6ea0c53e48790_51.png)

在提交前审阅PDF文件是一个好习惯。

祝你的报告顺利成功！

![](img/47649ed3ca09f9b2beb6ea0c53e48790_53.png)

本节课中我们一起学习了如何准备一份专业的数据科学报告。关键步骤包括：使用目录构建清晰结构，将核心结果置于正文、详细代码置于附录，通过超链接增强可读性，以图片形式插入图形，并最终导出为PDF格式进行提交审阅。
