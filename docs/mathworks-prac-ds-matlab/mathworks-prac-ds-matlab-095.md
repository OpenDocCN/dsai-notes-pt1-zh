同时，你也学会了如何解释均方根误差和R平方等关键指标。一个好的模型应同时具备较低的RMSE和接近1的R平方值。

核心评估指标公式如下：
*   **均方根误差**：`RMSE = sqrt(mean((y_true - y_pred).^2))`
*   **R平方**：`R² = 1 - (SS_res / SS_tot)`
