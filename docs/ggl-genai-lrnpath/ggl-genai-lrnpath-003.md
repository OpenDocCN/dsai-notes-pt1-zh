#  003：负责任AI简介 🧭

![](img/48772ae48200c89829a9a61577895924_0.png)

在本节课中，我们将要学习什么是负责任的人工智能，以及为什么它在AI开发和应用中至关重要。我们将探讨负责任AI的核心原则、实践意义，并了解谷歌如何将责任融入其AI产品的设计与部署中。

![](img/48772ae48200c89829a9a61577895924_0.png)

人工智能正被广泛讨论。但负责任地使用AI意味着什么？这正是我要讲解的内容。我是Manny，谷歌的一名安全工程师。我将教你理解谷歌为何制定AI原则，识别组织内部对负责任AI实践的需求，认识到负责任AI会影响项目所有阶段的所有决策，并认识到组织可以设计符合其自身业务需求和价值观的AI工具。

听起来不错，让我们开始吧。你可能没有意识到，但我们许多人已经每天都在与人工智能互动，从交通和天气预测到推荐你可能喜欢观看的下一部电视剧。

随着AI变得越来越普遍，许多非AI赋能的技术开始显得不足。就像现在无法上网的手机一样。AI系统正在使计算机能够以十年前难以想象的方式观察、理解和与世界互动。并且这些系统正以惊人的速度发展。

然而，我们必须记住，尽管取得了这些显著进步，AI并非完美无缺。开发负责任的AI需要理解可能的问题、局限或意外后果。技术是社会现状的反映，因此，如果没有良好的实践，AI可能会复制现有的问题或偏见，并将其放大。

事情在这里变得棘手，因为并不存在一个关于负责任AI的普适定义，也没有一个简单的清单或公式来定义应如何实施负责任的AI实践。相反，各组织正在制定反映其使命和价值观的自身AI原则。

幸运的是，虽然这些原则对每个组织都是独特的，但如果你寻找共同主题，会发现一套贯穿透明度、公平性、问责制和隐私的一致性理念。让我们深入了解谷歌如何看待这些问题。

谷歌对负责任AI的根植于一项承诺：致力于构建**为所有人服务、安全可靠、尊重隐私并由科学卓越性驱动**的AI。我们制定了自身的AI原则、实践、治理流程和工具，它们共同体现了我们的价值观，并构成了我们负责任AI的方法。

我们已经将“责任设计”的理念融入到我们的产品，更重要的是，融入到我们的组织中。像许多公司一样，我们使用AI原则作为指导负责任决策的框架。

在负责任AI的应用中，我们每个人都扮演着角色。无论你参与AI流程的哪个阶段，从设计到部署或应用，你所做的决策都会产生影响。这就是为什么拥有一个定义明确且可重复的负责任AI使用流程如此重要。

关于人工智能存在一个常见的误解，即机器扮演着核心决策角色。实际上，是**人**设计和构建这些机器，并决定如何使用它们。让我解释一下。

人们参与AI开发的每个方面。他们收集或创建用于训练模型的数据。他们控制AI的部署以及其在特定环境中的应用方式。本质上，人类的决策贯穿于我们的技术产品之中。每当一个人做出决策时，他实际上是在根据自己的价值观做出选择。

无论是决定使用生成式AI而非其他方法来解决问题，还是在机器学习生命周期的任何环节，那个人都引入了自己的一套价值观。这意味着每个决策点都需要考虑和评估，以确保从概念到部署和维护的整个过程中，选择都是负责任地做出的。

因为这些技术有潜力影响社会的许多领域，更不用说人们的日常生活，所以带着伦理意识来开发这些技术非常重要。

负责任AI并不意味着只关注明显有争议的用例。如果没有负责任的AI实践，即使是看似无害的AI用例，或那些出于良好意图的用例，仍可能引发伦理问题、意外后果，或者无法达到其应有的益处。

伦理和责任之所以重要，不仅因为它们代表了正确的事情，还因为它们可以指导AI设计，使其对人们的生活更有益。

那么，这与谷歌有何关系？我们认识到，将责任融入任何AI部署中，能打造更好的模型，并与我们的客户以及客户的客户建立信任。如果在任何环节破坏了这种信任，我们将面临AI部署停滞、失败，或最坏情况下对利益相关者造成伤害的风险。

总而言之，这符合我们在谷歌的信念：**负责任AI = 成功的AI**。我们通过一系列评估和审查来做出关于AI的产品和业务决策。这些流程在我们的产品领域和地域范围内，为我们的方法注入了严谨性和一致性。

这些评估和审查首先确保任何项目都符合我们的AI原则。虽然AI原则有助于让团队立足于共同的承诺，但并非每个人都会同意关于如何负责任地设计产品的每一个决定。这就是为什么建立人们可以信任的稳健流程非常重要，这样即使他们不同意最终决定，也会信任驱动该决策的过程。

以上我们主要从理论上讨论了指导原则对AI的重要性。但在实践中它们是什么样子的呢？让我们深入探讨。

2018年6月，我们宣布了七项AI原则来指导我们的工作。这些是积极管理我们研究和产品开发并影响我们业务决策的具体标准。

以下是每一项的概述：

1.  **AI应有益于社会**。任何项目都应考虑广泛的社会和经济因素，并且只在我们认为总体可能的收益大大超过可预见的风险和弊端时才会进行。
2.  **AI应避免制造或强化不公平偏见**。我们力求避免对人们造成不公正的影响，特别是与种族、民族、性别、国籍、收入、性取向、能力以及政治或宗教信仰等敏感特征相关的影响。
3.  **AI应基于安全目的构建和测试**。我们将继续发展和应用强有力的安全实践，以避免产生伤害风险的意外结果。
4.  **AI应对人负责**。我们将设计能提供适当反馈、相关解释和申诉机会的AI系统。
5.  **AI应融入隐私设计原则**。我们将提供通知和同意的机会，鼓励采用具有隐私保护措施的架构，并对数据使用提供适当的透明度和控制。
6.  **AI应坚持高标准的科学卓越性**。我们将与一系列利益相关者合作，在此领域促进深思熟虑的领导力，借鉴科学严谨和多学科的方法。我们将负责任地分享AI知识，通过发布教育材料、最佳实践和研究，使更多人能够开发有用的AI应用。
7.  **AI的应用应符合这些原则**。许多技术有多种用途，因此我们将努力限制潜在有害或滥用的应用。

以上就是我们的七项原则。但除了这七项原则，还有一些特定的AI应用是我们不会追求的。我们不会在以下四个应用领域设计或部署AI：

*   造成或可能造成整体伤害的技术。
*   主要目的或实施方式是造成或直接促进人身伤害的武器或其他技术。
*   收集或使用信息进行违反国际公认准则的监控的技术。
*   目的违背国际法和人权广泛接受原则的技术。

确立原则是一个起点，而非终点。不变的事实是，我们的AI原则很少直接给出关于如何构建产品的答案。它们不能也不应让我们回避艰难的对话。它们是一个基础，确立了我们的立场、我们构建什么以及我们为何构建，并且它们是我们企业AI产品成功的核心。

本节课中，我们一起学习了负责任AI的重要性及其核心原则。我们了解到，AI并非自主决策，而是人类价值观和决策的体现，因此必须在开发和应用的全过程中贯彻责任意识。谷歌的七项AI原则及其禁止领域，为我们提供了一个将伦理考量融入技术实践的具体框架。记住，**负责任地开发和应用AI，是确保技术真正造福社会、取得成功的关键**。
