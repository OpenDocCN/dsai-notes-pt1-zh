# 001：谷歌量子纠错导论 🧠

![](img/732434d6ae15447f8fa0bf4043430b14_0.png)

![](img/732434d6ae15447f8fa0bf4043430b14_2.png)

在本节课中，我们将开启谷歌量子人工智能的量子纠错实践之旅。课程将从量子计算机硬件概览开始，逐步深入到量子电路理论，并最终构建起量子纠错的核心概念与实用工具链。

![](img/732434d6ae15447f8fa0bf4043430b14_0.png)

欢迎来到谷歌的量子纠错导论课程。我是马斯克·福勒，在这个领域已经工作了超过20年。我很荣幸能为大家讲解这门课程的内容。

在开始之前，我想感谢几位为此课程做出贡献的人。戴夫、亚伯和奥古斯特都帮助构思并推动了这门课程的成型。同时，制作团队的杰西卡、伊娃和埃斯也帮助打磨了最终的成果。

为了概述我们在这门课程中希望达成的目标，我们将从量子计算机硬件的概览开始。目前存在多种多样的硬件平台，它们都非常新奇且有趣。我们将从这里起步，确保大家对人们正在尝试构建的设备有一个扎实的基础了解。

在掌握了硬件基础之后，我们将进入量子电路理论部分。这部分内容会进展得比较快，但我们会从最基础的概念开始讲解。因此，即使你之前没有接触过这个领域，我希望在几次内容重复学习后，你也能跟上课程后续的进度。

有了这些理论基础，我们将进一步深入到量子纠错本身。我们将逐步构建起能够实际检测错误、并对这些信息进行经典处理的基本电路。我们将层层递进，讲解你可能尚未听过的概念，例如稳定子码。我们还会介绍能够处理大规模量子电路并快速返回结果的模拟器。

我们将讨论ST，这是我们用于量子纠错电路的最先进模拟器。最后，课程将一直延伸到Crumble，这个工具帮助我们管理在受量子纠错保护的数据上进行计算的复杂性。

那么，我们这就开始吧。
