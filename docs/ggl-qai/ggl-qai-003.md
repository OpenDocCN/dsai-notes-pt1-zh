在本节课中，我们将学习量子计算的核心基础：量子态和量子电路。我们将暂时抛开所有硬件细节，专注于理解量子比特（qubits）的表示方式、其基本性质以及如何通过量子门（gates）来操作它们。这些概念是理解后续量子纠错技术的基石。

## 量子态：量子计算机的“0”与“1”

在经典计算机中，我们处理的是比特（bits）和比特串。我们可以对它们进行加法、乘法、逻辑运算等操作。在量子计算中，我们同样需要关注操作的规则，但存储的内容更为复杂。

一个由量子比特组成的系统，其状态不是简单的0或1字符串。对于一个由n个量子比特组成的系统，其状态由2^n个复数（称为振幅）来描述。每个振幅对应一个可能的经典比特串状态。

**公式表示**：一个三量子比特系统的状态可以表示为：
`|ψ⟩ = α|000⟩ + β|001⟩ + γ|010⟩ + δ|011⟩ + ε|100⟩ + ζ|101⟩ + η|110⟩ + θ|111⟩`
其中，α, β, γ...θ是复数振幅。

这意味着，要在一个经典计算机中表示一个量子计算机的状态，我们需要一个长度为2^n的数组。这引出了一个深刻的物理问题：一小部分硬件如何能存储指数级大量的信息？虽然其物理本质尚不完全清楚，但实验已验证了这种可能性。

量子态有一个关键约束：每个振幅模的平方（|α|^2, |β|^2...）代表观测到对应经典状态的概率，且所有概率之和必须为1。在此约束下，振幅可以是任意的复数。

## 量子门：操作量子态

接下来，我们看看如何通过量子门来操作这些振幅和概率。

### 比特翻转门（X门）

X门是最简单的量子门之一。它的作用类似于经典的非门（NOT gate）：
*   作用于 |0⟩ 态，产生 |1⟩ 态。
*   作用于 |1⟩ 态，产生 |0⟩ 态。

更有趣的是它对叠加态的作用。考虑一个三量子比特态 `α|000⟩ + β|111⟩`。对其应用不同的X门（下标表示作用在哪个量子比特上，约定最右为0号，向左递增）：
*   **X₂**（作用于最左量子比特）：结果为 `α|100⟩ + β|011⟩`。
*   **X₁**（作用于中间量子比特）：结果为 `α|010⟩ + β|101⟩`。
*   **X₀**（作用于最右量子比特）：结果为 `α|001⟩ + β|110⟩`。

这个简单的例子展示了量子计算威力的来源：**通过一个基本操作，就能同时操纵一个指数级大叠加态中的所有分量**。

### 相位翻转门（Z门）

由于量子振幅是复数，我们可以改变其相位（符号）。Z门的作用是：
*   作用于 |0⟩ 态：不变。
*   作用于 |1⟩ 态：振幅乘以 -1（符号翻转）。

再次以态 `α|000⟩ + β|111⟩` 为例，对其应用任何Z门（Z₂, Z₁ 或 Z₀），结果都是 `α|000⟩ - β|111⟩`。因为只有所有量子比特都为1的分量才会被翻转符号。

## 量子电路：门序列的图示

当我们有多个门要按顺序作用于一个量子态时，通常用量子电路图来表示。电路图从左到右代表时间流逝，每条水平线代表一个量子比特。

让我们通过一个例子来追踪电路的作用。初始态为 `α|000⟩ + β|111⟩`。
1.  遇到第一个X门（作用于量子比特2）：态变为 `α|100⟩ + β|011⟩`。
2.  遇到Z门（作用于量子比特1）：态变为 `α|100⟩ - β|011⟩`（因为中间位为1的分量 `|011⟩` 符号翻转）。
3.  遇到最后一个X门（作用于量子比特0）：态变为 `α|101⟩ - β|010⟩`。

对于更长的复杂电路，只需按从左到右的顺序依次应用每个门即可。

## 更多基础量子门

除了X和Z门，还有几个非常重要的基础门。

### 哈达玛门（H门）

H门用于创建叠加态：
*   `H|0⟩ = (|0⟩ + |1⟩)/√2`。这个态有50%概率测得0或1，称为 **|+⟩** 态。
*   `H|1⟩ = (|0⟩ - |1⟩)/√2`。这个态同样有50%概率测得0或1，但相位不同，称为 **|-⟩** 态。

**关键点**：|+⟩ 和 |-⟩ 是物理上可区分的态。如果你对 |+⟩ 态再作用一个H门，会回到 |0⟩；对 |-⟩ 态作用H门，则会回到 |1⟩。

### 受控非门（CNOT门）

要进行有意义的计算，必须让量子比特之间发生相互作用。CNOT门是最常用的双量子比特门。
*   **控制位**（带黑点）：如果它为 |1⟩，则对目标位执行翻转（X门操作）。
*   **目标位**（带⊕符号）：根据控制位状态决定是否被翻转。

其真值表清晰地展示了这一点（假设控制位为左量子比特，目标位为右量子比特）：
*   |00⟩ → |00⟩
*   |01⟩ → |01⟩
*   |10⟩ → |11⟩
*   |11⟩ → |10⟩

### 受控Z门（CZ门）

CZ门与CNOT门类似，但作用是相位翻转。
*   当**两个**量子比特都为 |1⟩ 时，整个态的符号被翻转（乘以-1）。
*   其他情况下，态不变。

由于作用是对称的（两个量子比特角色等价），其电路符号像一个哑铃（两端各有一个点），而不区分明显的控制和目标。

## 电路等价性示例

让我们通过一个练习来巩固理解。分析以下电路对于四个不同输入 |00⟩, |01⟩, |10⟩, |11⟩ 的输出。

**电路**：对最右量子比特（q0）作用H门 → 对两量子比特作用CZ门 → 再次对q0作用H门。

以下是计算输入 |10⟩ 的步骤：
1.  初始：|10⟩
2.  对q0作用H：`|1⟩ ⊗ (|0⟩+|1⟩)/√2 = (|10⟩ + |11⟩)/√2`
3.  作用CZ门（翻转|11⟩的符号）：`(|10⟩ - |11⟩)/√2`
4.  再次对q0作用H：展开并合并同类项后，得到最终态 **|11⟩**。

对所有输入进行计算后，你会发现这个电路（H - CZ - H）的输出结果，与直接应用一个**CNOT门**（以左量子比特为控制，右量子比特为目标）的效果完全一样。

**电路恒等式**：`H(q0); CZ(q1, q0); H(q0) ≡ CNOT(q1, q0)`
这是一个重要的电路恒等式，可以在简化或优化量子电路时使用。

## 量子测量

测量是量子计算中至关重要的一步，它会不可逆地改变量子态。

回到之前的例子，经过三个门操作后我们得到了态 `α|101⟩ - β|010⟩`。现在，如果我们测量中间那个量子比特（q1），会发生什么？

测量会以一定的概率“坍缩”整个量子态：
*   以概率 |α|^2 测得q1为 |1⟩，此时整个计算机状态坍缩为与测量结果一致的 `|101⟩`。
*   以概率 |β|^2 测得q1为 |0⟩，此时整个计算机状态坍缩为 `|010⟩`。

**核心要点**：对任何一个量子比特的测量，都会导致整个计算机的态坍缩到**与观测结果相容**的那些可能性上，其他所有分量都会消失。

## 总结

本节课我们一起学习了量子计算的基础构建模块：
1.  **量子态**：由复数振幅描述，能编码指数级信息，但受概率归一化约束。
2.  **量子门**：包括单量子比特门（如X, Z, H门）和双量子比特门（如CNOT, CZ门），用于操作量子态。
3.  **量子电路**：用从左到右的图示表示门序列的操作顺序。
4.  **测量**：使量子态坍缩到与测量结果一致的基础态，是获取计算结果的方式。
