## 基础环境设置 🖥️

第一步是在你的计算机上设置并准备好用于不同步骤的工具。可以将其视为在你的实际计算机上为数据科学和机器学习项目创建一个工作空间。

为此，我们将使用 **Anaconda**。Anaconda 就像是数据科学和机器学习工具的“五金商店”。一切始于基础：你的计算机。我们将安装 Anaconda。

一旦你安装了 Anaconda，获取我们所需的其他机器学习工具（如 Jupyter Notebooks、PyTorch、NumPy、Matplotlib、Pandas、Scikit-learn、CatBoost、XGBoost 和 TensorFlow）将变得非常轻松。其中一些工具名称较长，但有了 Anaconda，获取这些我们需要应用到之前所创建框架流程中的工具将轻而易举。

## 核心工具及其在流程中的角色 🔗

接下来，我们将对其中大部分工具进行详细讲解。市面上有很多机器学习工具，但我们的目标是确保你掌握一些可以立即开始使用的实用工具，并具备动手实践的知识。

如果将这些工具对应到我们的流程中，会是怎样的呢？

以下是主要工具及其对应流程阶段的介绍：

*   **编写代码与工作展示**：我们将使用 **Jupyter Notebooks** 来编写 Python 代码和交流我们的工作。这也是本课程所有项目所使用的文件格式。坚持学习，你将在课程结束时完成几个包含真实机器学习项目的 Notebook，用于向他人展示你的成果。

*   **数据分析**：在问题定义、实验、数据分析和机器学习的整个建模环节（图中已用颜色标出），可以将其视为数据科学的核心。我们之前见过类似的图表，但现在是将这些工具映射到我们之前设计的框架上。
    对于数据分析，你将使用诸如 **Pandas**、**Matplotlib** 和 **NumPy** 等工具。这对应流程中的第 2、3、4 步。

*   **构建机器学习模型**：你将使用诸如 **TensorFlow**、**PyTorch**、**Scikit-learn**、**XGBoost** 和 **CatBoost** 等工具来构建机器学习模型。

需要记住的重要一点是，你最重要的角色**不是**熟记每个单一库的所有函数，而是**知道针对何种问题使用何种工具**。这就是为什么第一步是定义我们的问题。这样你就能知道，可以从你的“工具袋”（即这些工具）中，为这些步骤选用这些工具，为那些步骤选用那些工具。
