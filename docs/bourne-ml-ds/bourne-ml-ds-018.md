# 18：评估类型 📊

![](img/430d7693555816a9a9bb62b759fc00a4_0.png)

在本节课中，我们将学习机器学习项目流程的第三步：评估。我们将探讨什么是评估指标，为什么它在项目开始时就需要明确，以及不同类型的机器学习问题如何对应不同的评估指标。

---

## 概述

在完成了问题定义和数据准备之后，第三步是确定项目的评估方式。评估的核心在于回答一个问题：**什么定义了我们的成功？** 一个明确的评估指标能确保项目团队朝着共同的目标努力。

## 什么是评估指标？

评估指标是衡量机器学习算法预测未来表现好坏的标尺。它量化了模型的性能。

## 如何选择评估指标？

选择评估指标取决于你要解决的问题类型。以下是不同问题类型常用的评估指标：

**以下是三种常见问题类型及其对应的评估指标：**

1.  **分类问题**：预测某事物属于哪个类别（例如，判断患者是否患有心脏病）。
    *   **准确率**：模型预测正确的比例。
    *   **精确率**：在所有被模型预测为“正类”的样本中，真正是“正类”的比例。
    *   **召回率**：在所有真正的“正类”样本中，被模型正确预测出来的比例。

2.  **回归问题**：预测一个数值（例如，预测汽车的售价）。
    *   **平均绝对误差**：预测值与真实值之间绝对差异的平均值。公式为：`MAE = (1/n) * Σ|y_i - ŷ_i|`
    *   **均方误差**：预测值与真实值之间差异的平方的平均值。公式为：`MSE = (1/n) * Σ(y_i - ŷ_i)^2`

3.  **推荐问题**：从海量选项中推荐少量项目（例如，为顾客推荐10款产品）。
    *   **Precision@K**：在模型推荐的前K个结果中，相关结果所占的比例。当K=10时，即衡量前10个推荐的质量。

## 评估指标的重要性与灵活性

在项目开始时坐下来思考评估指标，能确保团队中的每个人都为相似的目标而努力。

> 但重要的是要记住，这些指标也不必一成不变。随着你对数据的深入了解，评估指标可能会在项目进行过程中发生变化。

## 实践案例

为了更具体地理解，我们来看一个实践案例。我们曾有一个项目，目标是利用汽车保险索赔的文本内容来预测事故责任方。

与我们合作的保险公司要求，模型必须达到至少**95%的准确率**，这个项目才值得继续推进。

这意味着，我构建的模型在阅读一份索赔报告后，预测事故责任方的准确率必须达到95%，即每20次预测中最多只能错1次。

## 总结与思考

本节课我们一起学习了机器学习中的评估步骤。我们了解到，评估指标是衡量模型成功的标准，并且针对分类、回归和推荐等不同问题，需要选用不同的评估指标（如准确率、均方误差、Precision@K等）。明确评估指标有助于团队聚焦目标，同时它也应随着项目的深入而保持灵活性。
