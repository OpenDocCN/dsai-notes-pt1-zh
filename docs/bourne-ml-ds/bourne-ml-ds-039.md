## 概述

Pandas是一个用于数据分析和操作的Python库。在机器学习和数据科学工作流中，原始数据通常需要经过一系列处理才能用于算法。Pandas正是帮助我们完成这项工作的核心工具。

上一节我们介绍了Jupyter Notebook和Anaconda等环境设置工具，本节中我们来看看如何利用Pandas进行数据分析。

## 为什么选择Pandas？

Pandas之所以成为数据科学领域的首选工具，主要有以下几个原因：
*   **简单易用**：其API设计直观，便于学习和操作数据。
*   **生态集成**：它能与NumPy、Matplotlib以及众多机器学习库无缝协作。
*   **数据准备**：它擅长将原始数据（`raw data`）转换为适合机器学习算法输入的格式。

## 本节内容预览

鉴于Pandas功能非常庞大，我们将聚焦于最常用、最核心的部分。以下是本系列视频将要涵盖的主要内容：

我们将首先了解Pandas的两种主要数据结构。

以下是Pandas的两种核心数据结构：
1.  **Series**：一维带标签的数组。
2.  **DataFrame**：二维的、表格型的数据结构，是数据分析中最常用的对象。

接着，我们会学习如何读写数据。

以下是数据导入导出的常用方法：
*   使用 `pd.read_csv()` 导入CSV文件。
*   使用 `df.to_csv()` 将数据导出为CSV文件。

然后，我们将探索如何从统计角度描述数据。

以下是描述数据的常用函数：
*   `df.describe()`：生成描述性统计摘要。
*   `df.mean()`：计算平均值。
*   `df.info()`：查看数据框的信息概览。

Pandas的强大之处在于其灵活的数据查看与选择方式。

以下是查看和选择数据的方法示例：
*   `df.head()`：查看数据前几行。
*   `df[‘column_name’]`：选择单列。
*   `df.loc[]` 和 `df.iloc[]`：基于标签或整数位置进行选择。

最后，我们将学习如何对数据进行转换和操作，为机器学习做好准备。

以下是数据操作的核心技能：
*   处理缺失值（如使用 `df.fillna()`）。
*   修改、添加或删除列。
*   对数据进行分组和聚合（使用 `df.groupby()`）。
