## 工具框架与机器学习问题

上一节我们介绍了机器学习问题的解决框架。现在，我们将该框架与现有的数据科学和机器学习工具进行匹配。

我们将从基础开始。由于机器学习领域非常广泛，可用的工具众多，因此我们需要一种工具管理方案。

为此，你可以使用 Anaconda、Miniconda 和 Conda。我知道你可能在想 Daniel 到底在说什么，但请相信我，一旦你理解了这些，未来设置机器学习项目（例如使用你在这里看到的工具）将会变得轻而易举。

你可以将 Anaconda 和 Miniconda 视为数据科学家的硬件商店和工作台，而 Conda 则是你的个人助手。

---

## Anaconda 与 Miniconda：软件发行版

Anaconda 和 Miniconda 是软件发行版。就像你从应用商店下载应用程序一样，它们包含了其他人编写的代码，你可以在自己的计算机或智能手机上使用。

这两者也是如此，它们包含了其他人编写的代码，我们可以利用这些代码在自己的计算机上进行机器学习项目。

由他人编写的代码集合通常被称为**包**或**工具**。例如，对于我们讨论过的心脏病分类问题，我们可能希望使用多个 Python 包，每个包都可以被视为项目不同阶段的不同工具。

在项目初期，我们可能使用 `pandas` 和 `matplotlib` 包进行数据分析和探索。然后，为了构建机器学习模型，我们可能使用 `scikit-learn` 包。我们可能在 `Jupyter` 笔记本中使用所有这些包，而 `Jupyter` 笔记本来自 `Jupyter` 包。

---

## Anaconda：功能全面但占用空间大

将 Anaconda 下载到你的计算机大约需要 3 GB 的空间，它会安装所有主要的数据科学和机器学习包。然而，请记住，我们说 Anaconda 就像数据科学工具的硬件商店，这意味着它几乎包含了你所能想到的每一个数据科学工具。

根据我的经验，其中许多工具并不常用。由于我们专注于成为实践者，我们将使用最有用的工具。这就是 Miniconda 的用武之地。

---

## Miniconda：轻量级的工作台

Miniconda 就像数据科学家的工作台，它从最低要求开始，例如 Python 和其他几个工具。因此，它在你的计算机上占用的空间大约减少了 10 倍，大约为 200 MB。

为了节省空间并快速开始，我们将使用 Miniconda。但我提到 Anaconda 是因为我希望你在有人提到它或在网上看到它时能够了解它。

接下来，我们将看看 Conda，它就像是随 Miniconda 一起提供的助手。一旦你下载了 Miniconda，它将包含一些工具，然后你可以使用 Conda 来帮助你管理这些工具。
