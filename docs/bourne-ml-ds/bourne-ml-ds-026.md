在本节课中，我们将明确后续的学习方向。根据您是否已掌握 Python 编程语言，课程将分为两条路径。无论选择哪条路径，最终都将掌握相同的数据科学与机器学习核心知识与技能。

## 两条路径的由来

上一节我们介绍了课程的整体框架，本节中我们来看看一个关键的分叉点。课程进行至此，我们需要根据学员的先验知识来规划后续的学习路线。

![](img/6dc803779781dd29f909f0a17d91a26a_0.png)

之所以出现两条路径，是为了确保课程能包容所有背景的学员。为了继续学习数据科学与机器学习，掌握一门编程语言是必要条件，而 **Python** 是该领域的通用语言。

## 路径一：已掌握 Python

如果您已经熟悉 Python 编程，可以直接进入下一讲的学习。从下一讲开始，课程将默认您具备使用 Python 进行编程的能力。

## 路径二：需要学习 Python

如果您是编程新手，或者熟悉其他语言但不了解 Python，则需要先学习 Python 基础。

以下是给选择此路径学员的具体步骤：
1.  前往课程大纲的底部，找到 Python 入门章节。
2.  完成该 Python 章节的所有学习内容。
3.  完成后，返回本讲，即可继续后续的课程学习。

请放心，Python 学习并不像想象中那么困难，您将学到足以胜任相关职位的技能。

## 为什么必须学习 Python？

您可能会问，为什么一定要用 Python？用 Excel 处理数据和制作图表不行吗？

尽管 Excel 功能强大，但在某些方面，像 **Python** 这样的编程语言对于数据科学和数据分析更具优势。

与 Excel 主要通过可视化界面拖拽单元格进行操作不同，Python 具有以下显著优点：
*   **强大的数据处理能力**：能够处理几乎任何类型的数据。
*   **自动化**：可以自动化重复性任务，提高效率。
*   **高效处理大数据**：当数据量变得非常庞大时，Python 的处理速度更快。
*   **高级分析与机器学习**：提供 Excel 不具备的高级统计和机器学习功能。
*   **高级数据可视化**：能够创建比 Excel 标准图表更复杂、更精美的可视化效果。
*   **跨平台与可移植性**：Python 可以在任何操作系统上运行，便于部署和分享数据科学项目。

因此，掌握 Python 是必不可少的。
