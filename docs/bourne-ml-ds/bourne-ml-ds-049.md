# 49：NumPy 简介 🧮

![](img/a651f60fdac4b2ee5aa56398ccd8db26_0.png)

在本节课中，我们将要学习 **NumPy**，它是 Python 中进行数据科学、机器学习和数值计算的基础库。NumPy 是 “Numerical Python” 的缩写，其核心功能是将数据转换为一系列数字，以便机器学习算法能够从中找出规律。

## 概述

上一节我们介绍了 Pandas 用于处理表格数据。本节中，我们来看看 **NumPy**，它构成了几乎所有科学计算库的基石。NumPy 的核心优势在于其**速度**，因为它底层使用了优化的 C 代码，并通过**向量化**（或称**广播**）操作来避免低效的循环计算，从而高效处理大规模数值数据。

## 为什么使用 NumPy？

虽然 Python 本身可以进行数值计算，但在处理大规模数据时，NumPy 的速度要快得多。这主要得益于其底层的 C 语言优化。此外，NumPy 还是其他重要库（如 Pandas）的基础。

以下是 NumPy 的主要优势：
*   **速度快**：底层由 C 语言实现，计算效率高。
*   **向量化操作**：通过广播机制，无需编写循环即可对整个数组进行计算。
*   **功能强大**：提供了多维数组对象和大量数学函数。
*   **生态基础**：是 Pandas、SciPy、Scikit-learn 等库的依赖。
