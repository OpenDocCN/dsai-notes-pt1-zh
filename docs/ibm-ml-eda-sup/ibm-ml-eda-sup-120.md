### 探索 Gamma 参数

我们将创建一个包含不同 `gamma` 值的列表，并观察其对模型复杂度的影响。`gamma` 是径向基函数（RBF）核特有的参数。

![](img/9023075bb37309c656e1bbb703ef6d08_22.png)

以下是我们要测试的 `gamma` 值列表：

![](img/9023075bb37309c656e1bbb703ef6d08_24.png)

*   `gamma = 0.5`
*   `gamma = 1`
*   `gamma = 2`
*   `gamma = 10`

在运行代码前，请思考：哪个 `gamma` 值会导致模型更复杂（即正则化更弱）？哪个值会导致模型更简单（即正则化更强）？

运行代码后，我们观察到以下规律：
*   **`gamma = 0.5`**：决策边界非常平滑，模型复杂度低，意味着**高正则化**。
*   **`gamma = 1`**：决策边界出现一些弯曲，模型复杂度增加。
*   **`gamma = 2`**：决策边界弯曲更明显，模型复杂度更高，**正则化减弱**。
*   **`gamma = 10`**：决策边界变得非常曲折、波动，模型极其复杂，接近过拟合状态。

![](img/9023075bb37309c656e1bbb703ef6d08_27.png)

**结论**：`gamma` 值越大，模型复杂度越高，正则化效果越弱。

运行代码后，我们观察到以下规律：

## 第二部分：处理大规模数据集的策略 ⚡

![](img/9023075bb37309c656e1bbb703ef6d08_51.png)

![](img/9023075bb37309c656e1bbb703ef6d08_52.png)

上一节我们通过可视化理解了超参数的作用，本节中我们来看看当面对大规模数据集时，如何选择高效的SVM训练策略。

![](img/9023075bb37309c656e1bbb703ef6d08_54.png)

我们讨论了三种情况下的选择：
1.  **特征多或数据量大**：使用标准的线性SVC。
2.  **数据量更大，且需要非线性边界**：使用核函数SVC（如RBF核）。
3.  **数据量非常大**：使用**核近似（Kernel Approximation）** 配合线性分类器。

为了对比性能，我们将对一个数据集进行计时实验，比较以下两种方法的训练速度：
*   **方法A**：直接使用 `SVC(kernel=‘rbf’)`。
*   **方法B**：使用 `Nystroem` 进行核近似，将数据映射到高维空间，然后使用随机梯度下降（`SGDClassifier`）线性分类器。

![](img/9023075bb37309c656e1bbb703ef6d08_56.png)

首先，我们准备数据，使用所有特征（而不仅仅是之前可视化用的两个）。

### 方法B：核近似 + 线性分类器

接下来，我们使用 `Nystroem` 进行核近似变换，然后用 `SGDClassifier` 进行快速线性分类。

```python
from sklearn.kernel_approximation import Nystroem
from sklearn.linear_model import SGDClassifier

nystroem = Nystroem(kernel=‘rbf‘)
sgd = SGDClassifier()

![](img/9023075bb37309c656e1bbb703ef6d08_73.png)

# 先将数据变换到高维空间
X_transformed = nystroem.fit_transform(X)
# 计时训练线性分类器
%timeit sgd.fit(X_transformed, y)
```
**结果**：训练耗时约 **102 毫秒**，比标准SVC快了约**10倍**。

> **核心概念解释**：`Nystroem` 变换通过一个子样本（landmark points）来近似完整的核矩阵，将原始数据映射到一个新的高维特征空间（例如，默认映射到100维）。在这个新空间里，数据可能变得线性可分，从而可以使用高效的线性分类器。

## 总结 📝

本节课中我们一起学习了：
1.  **超参数调优**：通过可视化实验，我们明确了 `gamma` 和 `C` 参数如何控制SVM模型的复杂度与正则化强度。`gamma`/`C` 值越大，模型越复杂，正则化越弱。
2.  **核近似技术**：面对大规模数据集时，直接使用核SVC可能非常缓慢。我们学习了使用 `Nystroem` 等核近似方法将数据映射到高维，再结合 `SGDClassifier` 等高效线性分类器，可以极大提升训练速度，是处理大数据问题的实用策略。
