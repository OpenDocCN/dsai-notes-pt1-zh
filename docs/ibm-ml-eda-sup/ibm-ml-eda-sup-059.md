# 059：线性回归演示（第三部分）📊

![](img/a83072abd1a6c80f344495935c2e5f2a_3.png)

在本节课中，我们将学习如何评估线性回归模型的预测结果，并理解数据变换（特别是Box-Cox变换）对模型性能的影响。我们将重点关注如何将变换后的预测值还原到原始尺度，并比较变换前后的模型性能。

![](img/a83072abd1a6c80f344495935c2e5f2a_5.png)

![](img/a83072abd1a6c80f344495935c2e5f2a_7.png)

![](img/a83072abd1a6c80f344495935c2e5f2a_9.png)

上一节我们创建了线性回归模型并进行了预测。本节中，我们来看看如何解释这些预测结果，并评估模型的性能。

![](img/a83072abd1a6c80f344495935c2e5f2a_11.png)

我们首先需要思考：预测完成了吗？我们实际预测了什么？如何解释这些预测？

![](img/a83072abd1a6c80f344495935c2e5f2a_13.png)

![](img/a83072abd1a6c80f344495935c2e5f2a_14.png)

现在，快速移除图表中的点状标记，以便查看实际的预测值。

![](img/a83072abd1a6c80f344495935c2e5f2a_16.png)

让我们查看原始的 `y_test` 数据。

我们发现预测值和原始测试值可能不在同一个尺度上。

![](img/a83072abd1a6c80f344495935c2e5f2a_18.png)

它们看起来相差甚远，我们需要思考为什么会出现这种情况。

这是因为我们使用了Box-Cox变换对变量进行了转换。

为了将预测值对齐，并使其能够预测未来的中位数值，我们必须进行逆变换，将它们还原到与原始数据相同的尺度。

![](img/a83072abd1a6c80f344495935c2e5f2a_20.png)

![](img/a83072abd1a6c80f344495935c2e5f2a_22.png)

为了做到这一点，我们可以从 `scipy.special` 库中导入 `inv_boxcox` 函数。

![](img/a83072abd1a6c80f344495935c2e5f2a_24.png)

这只是原函数的逆运算。例如，如果原函数是 `f(x) = sqrt(x)`，那么它的逆函数就是 `x^2`。在处理Box-Cox变换时，我们可以用类似的方式思考，只需使用逆函数将其还原为 `x` 的原始形式。

![](img/a83072abd1a6c80f344495935c2e5f2a_26.png)

![](img/a83072abd1a6c80f344495935c2e5f2a_27.png)

回顾之前对整个数据集进行的Box-Cox变换，我们使用了 `bc_result = boxcox(...)`。调用该函数并传入整个 `y` 值后，我们得到了两个输出：Box-Cox变换后的数组和lambda值。为了进行逆变换，我们只需要传入想要还原的数组（即经过Box-Cox变换的Y变量版本）以及原始Box-Cox变换中使用的lambda值（这是我们得到的第二个结果）。

![](img/a83072abd1a6c80f344495935c2e5f2a_29.png)

如果我们将其与前10个实际值进行比较，会发现我们已经成功还原到了原始的10个值。

这向我们展示了我们的尺度确实不同：这是Box-Cox变换后的数据，然后我们通过逆变换将其还原回原始数据。

![](img/a83072abd1a6c80f344495935c2e5f2a_31.png)

![](img/a83072abd1a6c80f344495935c2e5f2a_32.png)

![](img/a83072abd1a6c80f344495935c2e5f2a_33.png)

现在的问题是，我们如何应用适当的逆变换，使我们的预测值 `yp_bc` 与 `y_test` 处于同一尺度，以便计算R²分数？

![](img/a83072abd1a6c80f344495935c2e5f2a_35.png)

我们只需要对预测值使用逆Box-Cox变换，并传入我们在上一节中定义的 `lambda2`（即 `boxcox` 函数的第二个输出）。

![](img/a83072abd1a6c80f344495935c2e5f2a_36.png)

![](img/a83072abd1a6c80f344495935c2e5f2a_38.png)

一旦得到逆变换后的值，我们就可以计算 `y_pred_train`（即经过Box-Cox逆变换的新变量）与原始 `y_test` 之间的R²分数。

![](img/a83072abd1a6c80f344495935c2e5f2a_40.png)

我们看到R²分数约为0.88。

![](img/a83072abd1a6c80f344495935c2e5f2a_42.png)

![](img/a83072abd1a6c80f344495935c2e5f2a_43.png)

现在，我们想确定Box-Cox变换是提高了还是降低了我们的R²分数。

![](img/a83072abd1a6c80f344495935c2e5f2a_45.png)

因此，我们将初始化一个新的线性回归模型。我们将使用 `X_train_s`（即经过标准缩放器处理的 `X_train`）和 `Y_train` 来拟合这个模型。请注意，这里的Y变量没有经过Box-Cox变换。

![](img/a83072abd1a6c80f344495935c2e5f2a_47.png)

![](img/a83072abd1a6c80f344495935c2e5f2a_48.png)

现在，模型已经拟合了这组新的X和Y变量。我们将使用这个新的LR对象进行预测。

![](img/a83072abd1a6c80f344495935c2e5f2a_50.png)

![](img/a83072abd1a6c80f344495935c2e5f2a_52.png)

在经过了标准缩放器变换的 `X_test_standard` 上进行预测，并查看在不使用Box-Cox变换处理 `Y_train` 的情况下，我们的R²分数是多少。

![](img/a83072abd1a6c80f344495935c2e5f2a_53.png)

我们看到得到的R²分数是0.85。这表明，通过对结果变量进行Box-Cox变换使其更接近正态分布，确实提高了我们的模型分数。

![](img/a83072abd1a6c80f344495935c2e5f2a_55.png)

![](img/a83072abd1a6c80f344495935c2e5f2a_56.png)

本节课中，我们一起学习了如何将Box-Cox变换后的预测值还原到原始尺度，并比较了数据变换前后线性回归模型的性能。我们发现，对目标变量进行适当的变换可以改善模型的预测能力。本节的演示到此结束，期待在下一讲中与您相见。
