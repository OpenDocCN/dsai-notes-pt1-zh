# 026：变量转换 📊

![](img/486ced6354effff02d14225023247f6d_1.png)

在本节课中，我们将要学习变量转换的概念及其在机器学习，特别是线性回归模型中的应用。变量转换是数据预处理和特征工程中的一个重要步骤，它可以帮助我们处理非正态分布的数据，揭示数据中潜在的线性关系，从而提升模型的性能。

## 概述

![](img/486ced6354effff02d14225023247f6d_3.png)

![](img/486ced6354effff02d14225023247f6d_4.png)

线性回归的数学理论假设模型的残差呈正态分布。然而，实际数据中的特征和结果变量常常呈现正偏态或负偏态分布。数据转换可以帮助算法找到更优的解决方案，最终使残差更接近正态分布。

## 常见的转换方法

![](img/486ced6354effff02d14225023247f6d_14.png)

以下是几种常见的数据转换方式。

*   **对数转换**：使用 `numpy` 库中的 `log` 或 `log1p` 函数。`log1p` 是 `log(x+1)`，用于处理数据中可能存在的零值，因为零的对数无定义。
    ```python
    import numpy as np
    transformed_data = np.log(data)  # 或 np.log1p(data)
    ```
*   **Box-Cox转换**：这是一种更复杂的方法，能够自动找到将偏态数据（如右偏数据）转换为正态分布的最佳方式。

## 多项式特征

类似地，我们可以通过添加多项式特征来估计数据中的高阶关系。这是工具箱中的另一个技巧。

![](img/486ced6354effff02d14225023247f6d_33.png)

因此，我们不仅可以纳入预算特征，还可以将预算的平方 `budget²` 作为一个特征，为模型增加更多灵活性，并使用这些多项式特征拟合一个线性模型。同样，即使我们包含了新特征，最终得到的仍然是一个线性模型。

![](img/486ced6354effff02d14225023247f6d_34.png)

![](img/486ced6354effff02d14225023247f6d_35.png)

![](img/486ced6354effff02d14225023247f6d_33.png)
![](img/486ced6354effff02d14225023247f6d_34.png)
![](img/486ced6354effff02d14225023247f6d_35.png)

这背后的思想是，在这一点上，模型只有一个拐点。因此，它不是一条直线，而是会弯曲。这时你会用到类似 `x²` 这样的项，将次数提高到2。

![](img/486ced6354effff02d14225023247f6d_37.png)

![](img/486ced6354effff02d14225023247f6d_38.png)

![](img/486ced6354effff02d14225023247f6d_37.png)
![](img/486ced6354effff02d14225023247f6d_38.png)

我们甚至可以将其扩展到三次、四次或任何更高次的多项式。

![](img/486ced6354effff02d14225023247f6d_40.png)

我们改变了特征，但保持了一个基础的线性模型。只是现在的特征被转换成了 `x + x² + x³` 等形式。

![](img/486ced6354effff02d14225023247f6d_42.png)

![](img/486ced6354effff02d14225023247f6d_42.png)
![](img/486ced6354effff02d14225023247f6d_43.png)

![](img/486ced6354effff02d14225023247f6d_43.png)

我们在图表中看到的将不仅仅是收益递减。现在我们有两个拐点。曲线先上升，然后下降。这意味着在某个时间点之后，不仅存在收益递减，而且在某个阈值之后，更高的预算可能导致票房收入减少。

## 使用Scikit-learn实现多项式转换

![](img/486ced6354effff02d14225023247f6d_46.png)

接下来，我们将结合代码深入探讨。以下是一个如何使用Scikit-learn创建多项式转换的示例。

![](img/486ced6354effff02d14225023247f6d_48.png)

第一步是导入 `PolynomialFeatures` 函数。

![](img/486ced6354effff02d14225023247f6d_50.png)

```python
from sklearn.preprocessing import PolynomialFeatures
```

![](img/486ced6354effff02d14225023247f6d_52.png)

然后，我们将创建 `PolynomialFeatures` 类的一个实例，并将其命名为 `poly_feat`。

```python
poly_feat = PolynomialFeatures(degree=2) # degree指定多项式的最高次数
```

`poly_feat` 现在是一个转换器对象。这意味着它可以拟合一组数据（例如我们稍后将看到的X数据），然后转换它。或者，它可以使用从拟合某个数据集中学到的东西来转换另一个相同大小的数据集。

这里我们看到执行 `poly_feat.fit(X)`，将转换器拟合到我们的X数据。

![](img/486ced6354effff02d14225023247f6d_54.png)

下一步是，现在我们有了已拟合数据的 `poly_feat` 转换器对象，我们可以用它来将数据从其原始形式（即我们之前看到的X）转换为 `X²` 等形式。

```python
X_poly = poly_feat.transform(X)
```

现在 `X_poly` 就包含了 `X` 和 `X²` 等特征。

## 总结

![](img/486ced6354effff02d14225023247f6d_60.png)

![](img/486ced6354effff02d14225023247f6d_61.png)

本节课中我们一起学习了变量转换。我们了解到，为了满足线性回归对残差正态分布的假设，常常需要对非正态的原始数据进行转换。我们介绍了对数转换和Box-Cox转换来处理偏态数据，使其更接近正态分布。更重要的是，我们学习了如何通过对数转换和创建多项式特征（如 `X²`, `X³`）来揭示和建模数据中潜在的非线性关系，同时保持使用线性模型进行拟合的框架。最后，我们通过Scikit-learn的 `PolynomialFeatures` 类演示了如何在实际代码中实现多项式特征转换。变量转换是特征工程中强大的工具，能显著提升模型的表达能力和预测性能。
