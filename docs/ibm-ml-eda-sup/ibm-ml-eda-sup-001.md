# 001：课程介绍与概述 🎬

在本节课中，我们将了解这门机器学习课程的整体框架、学习目标以及所需的基础知识。课程将从人工智能与机器学习的基础介绍开始，逐步深入到数据清洗、探索分析及特征工程等核心实践环节。

---

大家好，我是 Miguel，我将担任本课程的讲师。

我们将从人工智能与机器学习的基础介绍开始。

第一课将帮助我们认识到干净数据的价值。

接下来，我们将学习一些工具和技术，以帮助我们充分利用数据。

在本课程中，我们将通过实践，获得从不同来源获取数据、进行探索性数据分析和特征工程的经验。

强烈建议你具备一定的统计学和 Python 编程基础。

但如果你没有相关背景，也没有关系。你可能会发现部分内容有时具有挑战性。

请坚持下去，如有需要可以重新观看视频课程，并在讨论区与同学和讲师积极互动。

我们共同努力，互相帮助。

现在，本课程的一个主要目标是，你将获得使用相关工具和技术的经验，从而能够驾驭你真正感兴趣的数据集。

你将练习如何获取数据、探索数据、清洗数据并基于数据提出假设。

你将能把数据整理到良好的状态，以备用于训练机器学习模型。

因此，在学习本课程的过程中，请思考如何在工作或个人项目中应用这些工具和技术。

那么，我们课程中见。

---

本节课中我们一起学习了课程的整体目标、内容结构以及对学习者的基础要求。我们明确了从数据认识到预处理，再到为机器学习模型准备数据的完整学习路径，并强调了实践与社区互动的重要性。
