# 049：机器学习课程介绍（第1课）📚

在本节课中，我们将要学习机器学习的基础知识，特别是线性回归模型的构建方法。线性回归是理解更复杂模型的重要基础，适用于任何连续型目标变量，例如损失、收入、用户数量等。这些目标通常可以用“多少？”来提问。

## 机器学习定义与类型 🤖

上一节我们介绍了课程的整体目标，本节中我们来看看机器学习的基本定义和主要类型。

机器学习主要分为三种类型：监督学习、无监督学习和半监督学习。此外，机器学习的两个主要目标是**预测**和**解释**。

## 构建优秀线性回归模型的工具 🛠️

了解了基本概念后，本节我们将深入探讨构建优秀监督学习线性回归模型所需的核心工具。

以下是构建模型过程中需要掌握的关键步骤和方法：

*   **数据分割**：为了避免模型**过拟合**，需要将数据集分割为训练集和测试集。
*   **交叉验证**：使用交叉验证技术来更稳健地评估模型性能。
*   **多项式与正则化**：应用多项式回归和正则化方法（如岭回归、Lasso回归）来构建更高级的**正则化回归**模型。

## 总结 📝

本节课中我们一起学习了机器学习课程的介绍，明确了线性回归作为建模基础的重要性。我们了解了机器学习的三种主要类型（监督、无监督、半监督）和两个核心目标（预测与解释）。最后，我们概述了构建优秀线性回归模型将要用到的关键工具，包括数据分割、交叉验证以及多项式与正则化技术。
