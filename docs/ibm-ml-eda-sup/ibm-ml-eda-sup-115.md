## 从二维到三维的映射过程 🗺️

## 径向基函数与核技巧 ✨

现在，回到我们的原始二维空间。每个相似度度量都是由**径向基函数**决定的。

这个函数帮助我们创建在高维空间中彼此靠近的簇。一旦我们映射到那个高维空间，这些簇就有望变得线性可分。

![](img/6653686c09bb3652305a9ffdd279598e_19.png)

![](img/6653686c09bb3652305a9ffdd279598e_14.png)

![](img/6653686c09bb3652305a9ffdd279598e_20.png)

![](img/6653686c09bb3652305a9ffdd279598e_21.png)

![](img/6653686c09bb3652305a9ffdd279598e_22.png)

这里需要注意几点：
1.  在实际应用中，我们不会像这里演示的这样只为三个随机点做映射，而是会为数据集中的**每一个点**都进行映射。
2.  虽然我们不深入数学细节，但支持向量机中**核技巧**的特殊之处在于，我们可以在高维空间中进行计算（维度等于数据集中所有样本的数量），而无需真正将数据显式地映射到高维空间，从而避免了巨大的计算开销。

## 使用Scikit-learn实现代码 💻

理解了原理之后，我们来看看如何用Scikit-learn库实际实现高斯核支持向量机。

以下是实现步骤：

首先，像之前一样，导入包含我们分类方法的类。这里我们不再使用线性支持向量机，而是导入`SVC`类，它允许我们指定不同的核函数。

接下来，创建类的一个实例。这里我们创建了一个名为`rbf_svc`的实例，它代表使用径向基函数核的SVC。

在初始化类时，我们需要传入几个超参数：

![](img/6653686c09bb3652305a9ffdd279598e_37.png)

```python
from sklearn.svm import SVC

# 创建RBF核SVM分类器实例
rbf_svc = SVC(kernel=‘rbf‘， gamma=0.1， C=1.0)
```

![](img/6653686c09bb3652305a9ffdd279598e_39.png)

![](img/6653686c09bb3652305a9ffdd279598e_40.png)

以下是关键参数说明：
*   **`kernel`**：指定核函数类型。除了`‘rbf‘`，还可以选择`‘poly‘`（多项式核）或`‘linear‘`（线性核）等。具体可查阅官方文档。
*   **`gamma`**：控制高斯分布的“影响范围”。**`gamma`值越高，正则化程度越低**。
*   **`C`**：惩罚系数。与`gamma`类似，**`C`值越低，正则化程度越高**。

简单来说，`gamma`和`C`都是控制模型复杂度的正则化项。较低的值意味着更强的正则化（模型更简单），较高的值意味着更弱的正则化（模型更复杂，可能过拟合）。
