## 从现有特征创建新特征 🧩

![](img/b27423f2623e70674e059337c1b6dec8_9.png)

上一节我们讨论了寻找决策边界的目标。本节中我们来看看第一种方法：从现有特征创建新特征。

这是一种我们熟悉的方法，例如在之前的课程中创建多项式特征。

![](img/b27423f2623e70674e059337c1b6dec8_11.png)

![](img/b27423f2623e70674e059337c1b6dec8_12.png)

以下是一个例子：我们最初有两个特征（例如“预算”和“评分”），通过组合它们可以创建三个额外的新特征。

![](img/b27423f2623e70674e059337c1b6dec8_13.png)

```python
# 假设原始特征为 x1 (预算) 和 x2 (评分)
# 创建的多项式特征可能包括：
新特征1 = x1**2  # 预算的平方
新特征2 = x2**2  # 评分的平方
新特征3 = x1 * x2 # 预算与评分的乘积
```

![](img/b27423f2623e70674e059337c1b6dec8_15.png)

![](img/b27423f2623e70674e059337c1b6dec8_16.png)

因此，我们不再仅仅拥有“预算”和“评分”，而是拥有了这三个额外的特征。这三个新特征在我们的决策空间中代表了三个新的维度。

![](img/b27423f2623e70674e059337c1b6dec8_17.png)

### 相似性函数的形式 📐

![](img/b27423f2623e70674e059337c1b6dec8_34.png)

这些相似性函数具体是什么样子呢？我们将为每一部参考电影创建一个**高斯函数**，用以衡量其他电影的特征与该电影的相似程度。

![](img/b27423f2623e70674e059337c1b6dec8_35.png)

![](img/b27423f2623e70674e059337c1b6dec8_37.png)

其核心公式如下：

**相似度 = exp(-γ * ||x - l||²)**

![](img/b27423f2623e70674e059337c1b6dec8_39.png)

![](img/b27423f2623e70674e059337c1b6dec8_40.png)

其中：
*   `x` 是待评估数据点的特征向量。
*   `l` 是选定的参考点（如《低俗小说》的特征向量）。
*   `||x - l||²` 表示 `x` 与 `l` 之间的欧几里得距离的平方。
*   `γ` 是一个参数，控制相似度随距离衰减的速度。
*   `exp` 是指数函数，确保输出值在0到1之间。

![](img/b27423f2623e70674e059337c1b6dec8_42.png)

![](img/b27423f2623e70674e059337c1b6dec8_43.png)

对于《低俗小说》，我们计算每个数据点特征与《低俗小说》特征之间的距离，并通过高斯函数得到相似度值。

![](img/b27423f2623e70674e059337c1b6dec8_45.png)

同样地，我们可以为《黑天鹅》创建另一个相似性函数，为所有其他电影计算与《黑天鹅》的相似度度量。

![](img/b27423f2623e70674e059337c1b6dec8_46.png)

最后，我们对《变形金刚》进行相同的操作，创建第三个特征函数来计算每部电影与《变形金刚》的相似度。

### 利用新函数映射到高维空间 🗺️

我们如何利用这些新函数，将最初的数据映射到高维空间呢？

![](img/b27423f2623e70674e059337c1b6dec8_53.png)

对于一个给定的数据点，由其评分`x1`和预算`x2`构成的特征向量，我们可以使用刚刚讨论的三个径向基函数（即那些高斯函数）计算出三个数值。

```python
# 对于输入特征 [x1, x2]
A1 = similarity_to_pulp_fiction([x1, x2])  # 与《低俗小说》的相似度
A2 = similarity_to_black_swan([x1, x2])    # 与《黑天鹅》的相似度
A3 = similarity_to_transformers([x1, x2])  # 与《变形金刚》的相似度
```

![](img/b27423f2623e70674e059337c1b6dec8_55.png)

![](img/b27423f2623e70674e059337c1b6dec8_56.png)

![](img/b27423f2623e70674e059337c1b6dec8_57.png)

这三个数值（例如 A1=0.9， A2=0.92， A3=0.3）分别代表了该数据点与数据集中那三部特定电影的相似度。我们可以利用这三个数值，将原本仅由IMDB评分和预算构成的二维空间，映射到一个新的三维空间，这个空间的三个维度就是A1、A2和A3。

![](img/b27423f2623e70674e059337c1b6dec8_58.png)

![](img/b27423f2623e70674e059337c1b6dec8_59.png)

对于另一个不同的数据点，其A1、A2和A3的值也会不同，这些值同样由它们与各部参考电影的相似度定义。

![](img/b27423f2623e70674e059337c1b6dec8_61.png)

这种方法能确保相似的电影在映射到高维空间后彼此更接近，从而更容易找到一个线性的超平面将它们分隔开来。

![](img/b27423f2623e70674e059337c1b6dec8_62.png)
