关于生成式代码，目前存在许多恐惧、不确定性和疑虑。同时，也有大量炒作声称LLM将使开发者失业。

我个人不同意这种观点。我想明确指出，LLM可以成为开发者的得力助手，让你变得更高效。在本课程中，我逐步介绍了多种使用LLM的方法，这些方法超越了简单的代码生成，能够帮助你成为一名更优秀的软件工程师。

我希望这些内容能激发你思考许多应用场景，让你了解如何成为一名更出色的开发者。😊
