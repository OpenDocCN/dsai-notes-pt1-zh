以下是创建技能时需要牢记的几个关键步骤：

1.  **从基础开始**：创建技能时，从基本的 Markdown 格式指令开始。
2.  **逐步扩展**：随后遵循**渐进式披露**的原则进行扩展。
3.  **监控与迭代**：在真实场景中监控你的智能体如何使用技能，并根据观察结果进行迭代优化。
4.  **提供清晰描述**：确保技能描述包含足够的细节，以便你的智能体知道何时应该使用该技能。
