## 执行系统命令与处理日志

除了基本的输入输出，我们还探索了在Python中执行系统命令的方法。

我们使用 `subprocess` 模块来运行外部命令，其基本调用方式如下：
```python
import subprocess
result = subprocess.run(["ls", "-l"])
```
