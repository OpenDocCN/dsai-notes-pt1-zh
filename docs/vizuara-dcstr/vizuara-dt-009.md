## 问题概述

我们面临一个经典的回归问题。数据如下：
*   **X轴**：药物剂量
*   **Y轴**：药物有效性百分比
*   **数据**：我们收集了大约10到15名患者的数据，记录了在不同药物剂量下的有效性百分比。
*   **目标**：基于这些训练数据，构建一个回归树模型。该模型能够根据新患者的药物剂量，预测其药物有效性百分比。

最终构建完成的优化决策树如下图所示，它包含根节点、分支和叶节点。我们的任务就是一步步构建出这棵树。

## 确定根节点

上一节我们介绍了回归树的应用和最终目标，本节中我们来看看如何开始构建的第一步：确定回归树的根节点。

根节点是决策树的起点，它决定了数据的第一次分割。在我们的例子中，根节点显示为“剂量 < 14.5？”。为什么是14.5而不是其他值？这就是我们今天要解决的问题。

以下是确定根节点的具体步骤：

### 步骤一：计算前两个剂量值的平均值

首先，我们需要从数据中找出第一个分割点。一个初始的方法是查看数据的前两个剂量值（在图中用橙色圆圈标出），并计算它们的平均值。

设第一个点的剂量为 `x1`，第二个点为 `x2`。计算公式如下：
`平均值 = (x1 + x2) / 2`
在本例中，计算得到平均值为 **3**。

我们将以 **3** 作为候选根节点进行测试。

### 步骤二：以平均值作为分割点构建临时树

接下来，我们以“剂量 < 3？”作为根节点，构建一个临时的决策树。
*   如果剂量小于3（位于图中橙色虚线的左侧），则将该区域所有点的药物有效性平均值作为预测值。左侧只有一个点，其值为0，因此预测值为 **0**。
*   如果剂量大于等于3（位于橙色虚线右侧），则将该区域所有点的药物有效性平均值作为预测值。计算右侧所有点的平均值，得到 **38.8**。

因此，这个临时树规则是：
*   若剂量 < 3，预测有效性 = 0
*   若剂量 >= 3，预测有效性 = 38.8

### 步骤三：定性评估临时树

现在，我们直观地评估这个以“剂量=3”为根节点的树是否优秀。
*   **左侧**：只有一个数据点，预测值（0）与其真实值（0）完全吻合，表现完美。
*   **右侧**：包含了大部分数据点。预测值是一个固定的38.8。从图中可以明显看出，许多点的真实值远高于或低于38.8，导致预测误差很大。

由于大部分数据点都集中在右侧，且预测误差显著，我们可以**定性**地判断：选择“剂量=3”作为根节点并不是一个好的选择。

### 步骤四：量化评估——残差平方和

仅凭定性判断不够精确。在分类树中，我们使用基尼不纯度等指标；在回归树中，我们使用**残差平方和**来量化模型的误差。

**残差平方和** 的计算方法是：对于每一个数据点，计算其**真实值**与模型**预测值**之差的平方，然后将所有点的平方差求和。

公式表示为：
`SSR = Σ(真实值_i - 预测值_i)²`

这个指标越小，说明模型的预测整体上越接近真实值，模型越好。

让我们应用这个指标来量化评估我们的临时树：
*   对于左侧唯一的点：真实值=0，预测值=0，残差平方为 `(0-0)² = 0`。
*   对于右侧每一个点：真实值各不相同，预测值均为38.8。每个点都会产生一个 `(真实值 - 38.8)²` 的平方差。

将所有点的残差平方相加，就得到了这个以“剂量=3”为根节点的树的SSR。这个SSR值会很大，这从数学上证实了我们之前的定性判断——这不是一个优秀的根节点选择。

## 总结

本节课中我们一起学习了构建回归树的第一步：确定根节点。
1.  我们从一个简单的启发式方法开始：取前两个数据点特征值的平均值作为候选分割点。
2.  我们根据这个分割点构建了一个临时的决策树，并进行了定性分析。
3.  我们引入了**残差平方和**这个关键指标，用于量化回归树的预测误差。

![](img/13f3bd2734ae72d660fcdeb16d9a1299_2.png)

我们发现，第一个候选根节点（剂量=3）效果不佳。那么，如何找到最优的根节点呢？下一节课我们将介绍，通过系统地尝试所有可能的分割点，并选择那个能使**残差平方和最小**的分割点，来确定最佳的根节点。
